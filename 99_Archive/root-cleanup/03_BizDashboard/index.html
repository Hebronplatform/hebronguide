📖 용어 설명
시장가 (Market Rate)
미국/캐나다 프리랜서 플랫폼(Upwork, Clutch 등) 기준 실거래 가격 범위입니다.
권장 판매가
시장가 중간값 × 선택한 포지션 배율로 자동 계산되는 내 추천 판매 가격입니다.
원가 (Cost)
프로젝트에 드는 실비용 (도메인·호스팅·툴 구독·외주비 등)입니다. 직접 수정 가능합니다.
순수익 (Net Profit)
판매가 − 원가. 내 손에 실제로 남는 금액입니다.
순수익률 (%)
순수익 ÷ 판매가 × 100. 높을수록 효율적인 서비스입니다. 35% 이상이 목표입니다.
시간당 수익 ($/hr)
순수익 ÷ 작업시간. 내 시간 1시간이 얼마를 버는지 나타냅니다.
MRR (Monthly Recurring Revenue)
매월 안정적으로 들어오는 반복 수익. 유지보수·호스팅 대행 등 구독형 서비스에서 발생합니다.
ARR (Annual Recurring Revenue)
MRR × 12. 연간 반복 수익 총액입니다. 사업 안정성의 핵심 지표입니다.
포지션 티어 / 가격 전략
시장가 대비 내 가격 위치입니다. ×1.0=시장가 수준, ×1.3=프리미엄 30% 가산.
등급 A / B / C
A = 순수익률 45%↑ (우수) / B = 35~44% (양호) / C = 34%↓ (재검토 필요).
WMS / CRM
WMS = 웹사이트 관리 시스템, CRM = 고객 관계 관리 시스템. 회원·예약·데이터를 관리하는 백엔드 시스템입니다.
