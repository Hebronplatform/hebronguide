# HebronGuide 도시 생성 시스템 (City Generation System)
**Claude Code 자동 생성 프레임워크 — 공식 설계 문서**  
제정: 2026-05-05

---

## 시스템 선언

> **HebronGuide는 단순한 도시 가이드가 아니다.**  
> 미국에서 살아가는 한인을 위한 **"생활 내비게이션 시스템(Life Navigation System)"**이다.

---

## 트리거 명령

```
"Generate HebronGuide for [City Name]"
```

이 명령을 입력하면 Claude Code가 아래 5계층 구조로 완전한 도시 가이드를 자동 생성한다.

---

## 5계층 정보 아키텍처

```
┌─────────────────────────────────────────────────────┐
│  Layer A: 미국 사회 시스템 (전국 공통)                │
│  U.S. Social System — Federal/State/County/City 구조  │
├─────────────────────────────────────────────────────┤
│  Layer B: 도시 시스템 (도시별 생성)                   │
│  City System — 개요·산업·주거·교통·치안·교육·생활비   │
├─────────────────────────────────────────────────────┤
│  Layer C: 한인 커뮤니티 시스템 (도시별 생성)           │
│  Korean Community — 한인구·밀집지·교회·마켓·단체       │
├─────────────────────────────────────────────────────┤
│  Layer D: 타민족 커뮤니티 시스템 (도시별 생성)          │
│  Other Ethnic — 중국·히스패닉·인도·필리핀·베트남 등   │
├─────────────────────────────────────────────────────┤
│  Layer E: 한인을 위한 지혜로운 생활 전략 (도시별)       │
│  Wise Living — 생존 가이드·30일 플랜·맞춤 전략         │
└─────────────────────────────────────────────────────┘
```

---

## Layer A: 미국 사회 시스템 (전국 공통 — 모든 도시 동일)

### A-1. 행정 구조
- Federal → State → County → City 4단계 구조
- 각 단계의 역할과 한인에게 중요한 접점

### A-2. 의료 시스템
- 보험 종류 (Employer, Marketplace, Medicaid/Apple Health, Medicare)
- 1차·2차·응급 의료 체계
- 한국어 통역 요청 방법
- 무보험자 무료 클리닉 (Free Clinic)

### A-3. 교육 시스템
- K-12 공립학교 (무료, 학군제)
- Community College → University 경로
- ESL 프로그램
- 도서관 무료 교육 자원

### A-4. 법·치안 시스템
- 경찰 종류 (시경·군경·주경·FBI)
- 응급: 911 / 비응급: 311
- 법원 시스템
- 이민자 권리 기본 사항

### A-5. 교통·DMV
- 운전면허 취득 과정
- DMV 예약 방법
- REAL ID 요건
- 주요 대중교통 시스템

### A-6. 세금 시스템
- Federal Income Tax (IRS)
- State Income Tax (주별 상이)
- Sales Tax (주별 상이)
- Property Tax
- VITA 무료 세금신고 (연 1-4월)
- 한인 회계사 활용 팁

### A-7. 주거 시스템
- 렌트 계약 (Lease, Month-to-Month)
- 보증금·렌트 법률 (Tenant Rights)
- HOA 이해
- 주택 구매 기초 (Mortgage, Property Tax)
- 신용 없는 이민자를 위한 렌트 전략

### A-8. 취업 시스템
- Work Authorization (비자별 취업 자격)
- 고용 계약서 이해
- Payroll·W-2·1099
- 직장 내 차별 신고 (EEOC)
- LinkedIn·WorkSource 활용

### A-9. 이민 시스템
- 비자 종류 (F/J/H/O/L/EB/DV)
- 영주권 경로
- 시민권 요건 (N-400)
- USCIS 공식 절차
- 이민 변호사 활용 시기

## Layer B: 도시 시스템 (도시별 자동 생성)

### B-1. 도시 개요
- 인구·면적·설립 연도
- 도시의 성격 (Tech Hub, Finance Center, Trade Port 등)

### B-2. 인구 구성
- 인종별 구성 (최신 Census 기준)
- 이민자 비율
- 한인 비율·규모

### B-3. 주요 산업·고용주
- Top 5 고용 산업
- Top 10 대형 고용주
- 한인이 취업하기 유리한 분야

### B-4. 공항·국제 연결성
- 주요 공항 (코드·시설)
- 한국 직항 여부
- 국제 연결성 점수

### B-5. 교통·통근
- 대중교통 (Metro/Bus/Rail)
- 통근 구조 (차량 필수 여부)
- 주요 교통 앱

### B-6. 주거 시장
- 평균 렌트 (Studio/1BR/2BR/3BR)
- 한인 밀집 거주지 Top 3
- 좋은 학군 지역
- 가성비 지역

### B-7. 안전·치안 지도
- 안전 지역
- 주의 지역
- 야간 안전 팁

### B-8. 교육·학교
- 공립학군 평가 (Niche 기준)
- 주요 대학·커뮤니티 칼리지
- 한국어 학교

### B-9. 생활비 지수
- 시애틀/전국 대비 생활비
- 주요 항목별 비용

### B-10. 기후·환경
- 사계절 특징
- 재해 위험 (지진/허리케인/홍수)
- 공기질 지수

### B-11. 도시 장단점
- 한인에게 유리한 점 Top 5
- 한인에게 불리한 점 Top 5

## Layer C: 한인 커뮤니티 시스템 (도시별 생성)

### C-1. 한인 인구 규모
- 재외동포청 기준 (U.S. Census보다 정확)
- 성장 트렌드

### C-2. 한인 밀집 지역
- Top 3 한인 거주지 (주소·특징·렌트)
- Koreatown 존재 여부

### C-3. 한인 교회 (Tier 시스템 적용)
- ⭐ Tier 1: 가정교회 사역 (IHM 인증)
- ✅ Tier 2: SBC·보수 장로교
- ✅ Tier 3: 기타 정통 교단
- 🚫 이단 주의 목록

### C-4. 한인 마켓·식당
- 한인 마트 (H-Mart, 코마트 등)
- 검증된 한식당 Top 5
- 한인 카페 Top 3

### C-5. 한인 기관·단체
- 한인회·총영사관·한인상공회의소
- 한인 서비스 센터 (KCSC 유사 기관)
- 한인 미디어 (신문·방송·유튜브)

### C-6. 한인 비즈니스 생태계
- 주요 한인 업종
- 한인 상권 지역
- 한인 창업 지원 자원

### C-7. Korea Connectivity Score (한국 연결성 점수)
| 항목 | 점수 | 내용 |
|---|---|---|
| 직항 노선 | /30 | 인천 직항 여부·소요시간 |
| 한국 기업 지사 | /20 | 삼성·LG·현대 등 |
| 한국 유학생 수 | /15 | 주요 대학 한인 유학생 |
| 한국 관광객 | /15 | 연간 방한 한국인 |
| 한-도시 산업 교류 | /20 | 무역·투자·협력 |
| **총점** | **/100** | |

## Layer E: 한인을 위한 지혜로운 생활 전략 (도시별 생성)

### E-1. 미국 사회 시스템 활용 전략
- 한인이 잘 모르는 무료 공공자원 Top 10
- 세금 절약 전략
- 의료비 절약 전략

### E-2. 타민족 커뮤니티 협력 전략
- 어느 민족 마켓에서 무엇을 사야 하나
- 저렴한 타민족 맛집 Top 5
- 타민족 커뮤니티와의 네트워킹 방법

### E-3. 한인 커뮤니티 장단점 분석
- 장점: 언어·문화·신뢰 기반 네트워크
- 주의점: 정보 과잉·확인되지 않은 소문
- 교회를 통한 자연스러운 커뮤니티 진입

### E-4. 뉴커머 생존 가이드 (Newcomer Survival Guide)
- 도착 첫 주 반드시 할 것
- 첫 달 안에 처리할 것
- 3개월 안에 안정화할 것

### E-5. 도시별 한국인 맞춤 전략
- "이 도시에서 한국인에게 유리한 것"
- "이 도시에서 한국인이 흔히 겪는 실수"
- "이 도시에서 성공한 한국인의 공통점"

### E-6. 정착 30일 플랜
```
Week 1: 임시거주·SIM·은행·SSN 신청
Week 2: 운전면허·건강보험·자녀 학교 등록
Week 3: 장기 주거·한인 커뮤니티 연결·교회
Week 4: 취업 준비·신용카드 발급·생활 안정
```

### E-7. 맞춤 전략 (대상별)
- 👨‍👩‍👧 가족 동반 이민자 전략
- 🎓 유학생 전략
- 💼 직장인·주재원 전략
- 👴 시니어·은퇴자 전략
- 🚀 창업자·자영업자 전략

## 기술 구현 요건

### 1. 시애틀 코드베이스 기반 복제
```
hebronguide/src/app/components/HebronGuide.tsx 구조 유지
도시별 데이터만 교체:
  - CITY_META (도시명·색상·이미지)
  - TOP5_RESTAURANTS (검증 맛집)
  - TOP5_SETTLE (정착 기관)
  - TOP5_EXPLORE (관광 명소)
  - 각 Screen의 데이터 배열
```

### 2. 새로운 5계층 탭 구조 (확장)
```
현재 탭: 홈·정착·교회·맛집·관광·도움·취업·교육·생활비
확장 탭:
  + 미국 시스템 (Layer A)
  + 도시 정보 (Layer B)  
  + 타민족 커뮤니티 (Layer D)
  + 지혜로운 생활 (Layer E)
```

### 3. 다국어 (KO/EN 필수, ES·ZH 선택)

### 4. SEO 최적화
- 도시별 meta description
- 구조화 데이터 (Schema.org LocalBusiness)
- sitemap.xml 자동 업데이트

### 5. 모듈형 구조
```
각 도시 데이터: /src/data/{city}-data.ts
공통 컴포넌트: 변경 없음
도시별 이미지: Unsplash 무료 사진 URL
```

```
"Generate HebronGuide for Dallas" 입력
         ↓
Step 1: 리서치 (Agent 병렬 실행)
  - 달라스 도시 데이터 수집
  - 한인 커뮤니티 데이터
  - 타민족 커뮤니티 데이터
  - 와이즈 리빙 전략 수립
         ↓
Step 2: 데이터 구조화
  - 5계층 JSON 데이터 생성
  - 검증 (주소·전화·웹사이트)
         ↓
Step 3: 코드 생성
  - HebronGuide.tsx 도시별 데이터 삽입
  - 새 탭 구조 반영
  - 이미지 URL 교체
         ↓
Step 4: SEO 최적화
  - sitemap.xml 업데이트
  - meta 태그 생성
         ↓
Step 5: 배포
  - git push → Vercel 자동 배포
  - hebronguide.com/{city}/ 라이브
```
