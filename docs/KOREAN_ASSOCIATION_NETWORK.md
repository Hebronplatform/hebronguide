# 한인회 디지털 네트워크 전략

**Korean Association Network — 2026 공식 문서**
작성: 2026-07-02 | 최종 수정: 2026-07-02

> 계기: 페더럴웨이 한인회 홈페이지 제작 (2026-07). 이를 플래그십으로 전국 한인회 네트워크로 확장.
> 정체성: 이 전략은 [`REVENUE_STRATEGY.md`](REVENUE_STRATEGY.md) "두 팔 분리"의 **상업 팔** 플래그십 제품이자, HebronGuide **사명 팔**을 먹여살리는 데이터·신뢰 파이프라인이다.

---

## 핵심 명제

> **"전국 한인회 디지털 네트워크의 허브 = HebronGuide"**
> 한인회 사이트 하나하나를 파는 게 아니라, 서로 연결된 네트워크를 만들고 그 중심에 HebronGuide를 둔다.

단순 제작(일회성 현금) ❌ → 네트워크(자생 사이클) ⭕

---

## 왜 한인회가 이상적 쐐기(wedge)인가

| 이유 | 설명 |
|---|---|
| 홈페이지 부재·방치 | 대부분 한인회 사이트가 없거나 처참함 → 명백한 needs |
| 신뢰 앵커 | 교회처럼 지역 공신력 보유. 한인회가 쓰면 지역 업소·행사가 따라옴 |
| 반복 가능한 제품 | 한 번 템플릿화 → 도시마다 복제 판매 (시애틀=마스터 템플릿 원칙과 동일) |
| HebronGuide 도시와 1:1 매칭 | 이미 71개 도시 구조 존재 → 한인회 = 각 도시의 로컬 노드 |

---

## 네트워크 플라이휠

```
한인회 사이트 제작
   ↓
지역 업소·행사·검증 정보가 한인회 사이트에 모임
   ↓
그 데이터가 HebronGuide 도시 페이지로 흘러감 (상호 연동)
   ↓
HebronGuide가 풍부해짐 → 더 많은 한인회가 합류 원함
   ↓
(반복) — 한인회끼리도 연결된 전국 네트워크
```

---

## 비용 구조 — 광고 자금 모델 (한인회는 예산이 적다)

한인회는 대부분 비영리·자원봉사 → 큰 제작비 지불 불가.
**해법: 제작비를 한인회가 아니라 지역 업소 광고가 낸다.** (HebronGuide 광고 시스템 재사용)

```
한인회 사이트 = 무료/저가 제작
   → 사이트에 지역 업소 광고 게재 (community_items type=citynews 등)
   → 광고 수익으로 유지비 충당 + HebronGuide 수익 분배
   → 한인회(공짜 사이트) + 업소(노출) + HebronGuide(수익) 3자 이득
```

→ 정본 "섬기면서 수익" 원칙 부합. 광고는 큐레이션과 시각적으로 분리(REVENUE_STRATEGY 광고 지면 4단계 원칙).

## 단계별 로드맵

```
Phase 0 (지금) — 페더럴웨이 플래그십
  · 완벽하게 제작 → 스크린샷·후기 확보 → 영업 레퍼런스화

Phase 1 (1~3개월) — 제품화
  · 페더럴웨이 = 한인회 마스터 템플릿
  · 고정 패키지화 (제작 + 월 유지 + 광고 연동)

Phase 2 (3~9개월) — 지역 확산
  · 채널: HebronGuide 도시망 + 한인 미디어 + 교회 큐레이터 네트워크
  · 목표: 시애틀 메트로 → 주요 도시 한인회 5~10곳

Phase 3 (9개월+) — 네트워크 레이어
  · 전 한인회 사이트 상호 연결 + HebronGuide 데이터 통합
  · "전국 한인회 네트워크 = HebronGuide" 포지션 확립
```

## 정직한 리스크 & 가드레일

| 리스크 | 가드레일 |
|---|---|
| 한인회 내부 정치 (회장 선거·파벌) | 중립 서비스 제공자로만 남기. 정치 개입 ❌ |
| 자원봉사 운영 (느림·저예산) | 광고 자금 모델 필수. 제작비 직접 청구 최소화 |
| 유지보수 부담 → 죽은 사이트 양산 | 템플릿·자동화 필수. 커스텀 제작 스케일 금지 |
| 사명 희석 | 한인회 = 세속 시민 레이어. 히든 미션 자연 공존, "교회 앱" 인상 회피 |
| 교회·LLC 자금 혼용 | 제작·광고 수익은 Hebron Platform LLC로. 교회 재정과 완전 분리 |

북미 주요 한인회 100곳+ — 유한하지만 **B2B2C 쐐기로는 최상급**.
페더럴웨이가 첫 도미노. 서비스 팔(빠른 현금) + 네트워크(HebronGuide 데이터·신뢰 강화) 동시 달성.

## 임원·사무총장 만족 = 추천 확산의 열쇠 (2026-07-02 추가)

> 추천으로 확산되려면 첫 사이트(FWKAA)가 쇼케이스여야 하고, 임원들이 만족해야 추천이 돈다.
> 만족의 진짜 핵심은 **기능이 아니라 인정(recognition)과 위상**이다.

### 임원이 만족하는 7요소
| # | 요소 | 이유 |
|---|---|---|
| 1 | 임원진 소개(사진·직함·인사말) | 임원은 자기 이름·얼굴 노출을 가장 좋아함. 한인회 정치의 핵심=인정 |
| 2 | 격(格)·위신 | "우리 단체가 격 있어 보인다" (네이비+금+명조) |
| 3 | 연혁·활동 사진 | 단체가 살아있고 정당함을 증명 |
| 4 | 상급기관과 나란히 | 총영사관·민주평통·미주총연 + HebronGuide = 위상 상승 |
| 5 | 실용 기능 | 행사·공지·회원가입·후원·자료실 |
| 6 | 시니어 친화 | 큰 글씨·명확한 대비 (회원 다수 시니어) |
| 7 | 관리 부담 없음 | 우리가 유지관리 (방치 두려움 해소) |

### 3단계 제공 모델 (상황별)
| 상황 | 제안 | HebronGuide 연계 |
|---|---|---|
| 홈페이지 있음 | 강제 재제작 ❌ → 연계만(상호 링크·회원혜택·도시 배너), 낡았으면 업그레이드 제안 | 상호 링크 + 도시 페이지 |
| 없음 | 신규 제작(A/B 템플릿) | 풀 연계(SSO·업소·소식) |
| 낡음 | 업그레이드(콘텐츠 이전→모던 템플릿) | 풀 연계 |

### 추천 플라이휠
```
FWKAA 쇼케이스(만족) → 한인회장 간 추천(촘촘한 네트워크) → 다음 한인회
→ 각 사이트가 HebronGuide 도시 노드로 연결 → 전국 네트워크(허브=HebronGuide)
```
→ 그래서 **1호(FWKAA)를 반드시 탁월하게** 만들어야 한다.
