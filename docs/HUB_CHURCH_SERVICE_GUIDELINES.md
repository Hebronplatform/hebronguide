> *"황금율을 모든 것에 적용한다"* — 마7:12

좋은 날도, 피곤한 날도 — 같은 품질입니다.

## 1. 새가족 연결 프로토콜

### 자동 알림 수신 시

```
[HebronGuide 알림]
새 이민자: 김○○ 가족 (4인)
도착 예정: 2026-06-15 (인천 → 달라스)
연락처: 카카오 / 이메일
필요: 공항 픽업, 단기 숙박, 교회 연결 희망

→ 담당 허브교회: ○○한인교회
→ 72시간 내 연락 필수
```

### 72시간 이내 — 첫 연락 (필수)

```
전화 or 카카오 메시지 템플릿:

"안녕하세요, ○○님 가족.
저는 [달라스 ○○교회] 담당 [이름]입니다.
HebronGuide를 통해 연락드립니다.

[도착 날짜]에 오신다고 해서,
도착 전에 미리 인사 드리고 싶었습니다.
도움이 필요하신 것 있으시면 편하게 연락 주세요.
공항 픽업이나 초기 정착에 도움 드릴 수 있습니다."

⚠️ 금지: 교회 등록 권유, 헌금 언급, 종교적 압박
✅ 목적: 안심·환영·신뢰 제공
```

### 첫 주일 환영 체크리스트

```
□ 이름을 미리 알고 있다
□ 주차 안내원에게 이름 공유 (미리 대기)
□ 담당자가 입구에서 직접 기다림
□ 이름을 불러드림: "○○ 가족, 어서오세요!"
□ 예배 후 함께 식사 (또는 간단한 다과)
□ 주변 교인 1-2명 자연스럽게 소개
□ 다음 약속 잡기 (커피, 저녁 등)
```

### 3개월 정착 동반 계획

```
1개월차: 실생활 정착 지원
  □ 운전면허 함께 준비 (정보 공유)
  □ 학교 등록 도움 (자녀 있는 경우)
  □ 은행·보험 안내
  □ 한인 커뮤니티 소개

2개월차: 공동체 연결
  □ 목장/소그룹 자연스럽게 소개
  □ 비슷한 상황의 교인 연결
  □ 취업·사업 관련 교인 네트워크 소개

3개월차: 자립 확인
  □ 정착 체크리스트 완료 여부 확인
  □ 공동체 내 자리 잡았는지 확인
  □ 필요 시 추가 연결
  □ HebronGuide 피드백 수집
```

## 2. 서비스별 품질 기준

### 🚗 헤브론 라이드

```
드라이버 자격:
  ✅ 운전면허 유효 (최근 3년 이상)
  ✅ 차량 상태 양호 (4인 이상 탑승 가능)
  ✅ 한국어 가능
  ✅ 범죄 이력 없음

서비스 기준:
  □ 약속 시간 10분 전 도착
  □ 차량 내부 청결
  □ 짐 적재 도움
  □ 도착지 안내 정보 제공 (맛집, 마트 등)
  □ 연락처 제공 (정착 중 추가 질문 환영)

요금: 투명하게 사전 고지
금지: 요금 추가 요구, 종교 강요
```

### 🏠 헤브론 스테이

```
호스트 자격:
  ✅ 안전한 거주 공간 (독립된 방 or 독립 공간)
  ✅ 기본 생활 시설 (화장실, 주방 접근)
  ✅ 중독 물질 없는 환경
  ✅ 반려동물 사전 고지

서비스 기준:
  □ 도착 전 청소 완료
  □ 기본 생필품 준비 (수건, 세제 등)
  □ 집 안내 (열쇠, 주차, 쓰레기 분리수거 등)
  □ 인근 마트, 대중교통 안내
  □ 긴급 연락처 제공

금지: 숙박비 외 추가 요금, 개인 공간 침범
```

### 📚 헤브론 튜터

```
튜터 자격:
  ✅ 해당 과목 전문성 확인 (성적표/자격증)
  ✅ 학생 지도 경험 최소 6개월
  ✅ 범죄 이력 없음
  ✅ 학생/학부모 소통 능력

서비스 기준:
  □ 첫 수업 전 학부모 면담 (30분)
  □ 수업 계획서 사전 공유
  □ 결석 시 24시간 전 통보
  □ 월 1회 학습 진도 보고
  □ 학생 개인 정보 철저 보호

금지: 수업 중 개인 신상 요구, 추가 비용 요구
```

### 🧭 헤브론 관광

```
가이드 자격:
  ✅ 해당 도시 거주 2년 이상
  ✅ 주요 관광지·맛집·한인 커뮤니티 숙지
  ✅ 한국어 유창, 기본 영어 가능
  ✅ 운전 가능 (차량 투어 시)

서비스 기준:
  □ 투어 전날 일정 재확인 연락
  □ 픽업 장소 10분 전 도착
  □ 비상 연락망 공유
  □ 음식 알레르기·이동 제한 사전 파악
  □ 투어 후 피드백 수집

투어 종료 후:
  자연스럽게 "혹시 ○○ 지역에 좋은 교회 찾으세요?"
  → 강요 없이, 물어보면 안내
금지: 투어 중 종교 강요, 교회 등록 유도
```

### 💍 헤브론 매칭

```
운영 기준:
  □ 프로필 등록 후 담당자 확인 인터뷰 (15분)
  □ 상대방 정보 사전 공유 후 양측 동의하에 소개
  □ 첫 만남: 공개 장소, 2시간 이내
  □ 만남 후 양측 피드백 수집 (24시간 내)
  □ 불편함 신고 즉시 처리

프라이버시:
  □ 연락처는 양측 동의 시만 공유
  □ 프로필 정보 제3자 공유 금지
  □ 탈퇴 요청 즉시 처리

금지: 만남 강요, 개인 정보 유출, 수수료 외 비용 요구
```

## 3. 브랜드 언어 가이드

### 표면에서 쓰는 말 (보편적)

```
✅ "믿을 수 있는 한인 드라이버"
✅ "따뜻한 한인 가정에서 시작하는 정착"
✅ "이 도시를 깊이 아는 한인이 안내합니다"
✅ "진지하고 따뜻한 만남"
✅ "낯선 도시, 혼자 시작하지 않아도 됩니다"
✅ "사람이 사람을 섬깁니다. 목마른 이에게 생수를 권하듯."
```

### 절대 쓰지 않는 말

```
❌ "교회 검증 드라이버" → "믿을 수 있는 드라이버"
❌ "목사님 보증 매칭" → "신중한 연결"
❌ "구원받으세요" → (절대 사용 금지)
❌ "교회 나오세요" → 물어볼 때만 안내
❌ "하나님이 보내셨습니다" → 개인 신앙으로만
```

### 자연스러운 교회 연결 언어

```
물어볼 때: 
  "혹시 ○○에서 한인 교회 찾으시나요?
   제가 아는 좋은 교회 몇 군데 소개해 드릴 수 있어요."

관심 표현 시:
  "저도 사실 ○○교회 다니는데, 한번 같이 가보실래요?
   부담 없이 구경만 오셔도 돼요."

관심 없을 시:
  즉시 화제 전환, 절대 재권유 없음
```

## 4. 자동화 시스템 로드맵

### Phase 1 — 지금 당장 (수동 + 템플릿)

```
도구: 이메일·카카오 + 구글 스프레드시트 + HebronGuide 앱

자동화된 것:
  ✅ HebronGuide 앱 → 파트너 안내
  ✅ 이메일 템플릿 준비 (이 문서 기반)
  ✅ 체크리스트 (localStorage로 앱에서)

수동으로 하는 것:
  → HebronGuide → 허브교회 이메일 알림 (수동 전송)
  → 파트너 현황 구글 스프레드시트 관리
  → 월별 수익 정산 (이메일로)
```

### Phase 2 — 3-6개월 (반자동)

```
도구: Supabase + Resend(이메일) + Vercel Functions

새 이민자 등록 시:
  HebronGuide 앱 → Supabase DB → 
  자동 이메일 → 해당 도시 허브교회

허브교회 → 72시간 내 미응답 시:
  자동 리마인더 이메일

파트너 수입:
  서비스 완료 → Supabase → 
  월말 자동 정산 리포트 이메일
```

### Phase 3 — 6-12개월 (완전 자동화)

```
도구: Supabase + 파트너 대시보드 + 앱 내 예약

파트너 대시보드:
  □ 새가족 알림 실시간
  □ 서비스 예약 관리
  □ 수입 현황 실시간
  □ 피드백 수집 자동화

사용자 앱:
  □ 서비스 예약 (라이드·스테이·튜터 등)
  □ 가이드 프로필 확인
  □ 리뷰 작성
  □ 실시간 알림

품질 모니터링:
  □ 평점 4.0 미만 시 자동 알림
  □ 72시간 무응답 시 자동 경고
  □ 분기별 파트너 성과 리포트
```

## 5. 품질 모니터링 기준

### 허브교회 평가 기준 (분기별)

```
자동 측정:
  □ 72시간 내 응답률 (목표: 95%+)
  □ 새가족 3개월 정착률 (목표: 80%+)
  □ 사용자 만족도 평점 (목표: 4.5/5.0+)

수동 평가:
  □ HebronGuide 브랜드 지침 준수 여부
  □ 금지 행동 위반 여부

```

  ✗ 개인 정보 유출
  ✗ 이단·사이비 연결
  ✗ 서비스 비용 외 금전 요구

경고 후 3개월 이내 미개선 시 취소:
