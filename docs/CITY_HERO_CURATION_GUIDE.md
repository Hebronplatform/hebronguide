## 📐 3막 슬라이드 공식 표준 (모든 도시 필수)

### 🛩️ 슬라이드 1 — "첫눈에 반하다" (드론·공중 전경)

```
정착자의 심정: "이 도시가 내 새 보금자리가 될 곳이구나"
```

| 기준 | 내용 |
|---|---|
| **시각** | 드론 또는 헬기·전망대 등 공중에서 내려다보는 시야 |
| **구도** | 도시 전체 또는 넓은 구역이 한눈에 들어오는 와이드샷 |
| **색감** | 밝고 선명한 낮 (가능하면 파란 하늘 포함) |
| **감정** | 기대·설렘·경외·"내 앞에 새 땅이 펼쳐진다" |
| **금지** | 실내 사진, 사람 얼굴 클로즈업, 어두운 실루엣 |
| **키워드** | aerial, drone, skyline from above, bird's eye, city from air |

**모범 사례:**
- Seattle: Josh Hild 드론 — 레이크 + 다운타운 한눈에
- Toronto: Patrick Tomasso DJI 드론 — CN타워 + 겨울 새벽
- Houston: Nils Huenerfuerst 드론 — 다운타운 스카이라인 정직
- Manhattan: Florian Wehde 헬기 — 맨해튼 섬 전체

### 🏛️ 슬라이드 2 — "이 도시를 알아간다" (랜드마크)

```
정착자의 심정: "여기는 [도시명]이다, 내가 살 곳이다"
```

| 기준 | 내용 |
|---|---|
| **내용** | 그 도시의 가장 상징적인 건물·자연·장소 |
| **시각** | 지상 또는 가까운 앵글 (S1과 대비) |
| **색감** | 선명하고 특색 있는 도시 아이덴티티 색감 |
| **감정** | 인식·확인·"아, 내가 [도시명]에 왔구나" |
| **금지** | 다른 도시와 혼동될 수 있는 일반적 건물 |
| **예시** | Space Needle (시애틀), Golden Gate (SF), CN Tower (토론토), Big Ben (런던) |

### 🤝 슬라이드 3 — "환영받는다" (사람들·환대)

```
정착자의 심정: "나를 기다리는 사람들이 있다, 혼자가 아니다"
```

| 기준 | 내용 |
|---|---|
| **내용** | 거리의 사람들, 마켓, 커뮤니티 공간, 한인 상권 |
| **시각** | 사람들이 살아있는 활기찬 장면 |
| **색감** | 따뜻하고 생기 있는 색감 |
| **감정** | 소속감·안도·환대·"나도 이 커뮤니티의 일부가 될 수 있다" |
| **금지** | 포즈 잡은 개인 초상 (자연스러운 군중 OK), 어둡고 외로운 장면 |
| **예시** | Pike Place Market 군중 (시애틀), 広場 사람들, 한인 마켓 풍경, Shibuya 교차로 |

## 🎨 사진 품질 기술 기준

```css
/* 모든 도시 공통 CSS 필터 (명문화) */
filter: brightness(1.42) saturate(1.68) contrast(1.08);
/* 이유:
   brightness(1.42) — 밝고 생기있게
   saturate(1.68)   — 색감 선명하고 생동감 있게
   contrast(1.08)   — 선명하고 또렷하게 (0.9x 절대 금지 — 흐릿해짐)
*/
```

| 항목 | 기준 | 이유 |
|---|---|---|
| 해상도 | `w=1200&q=95` | q=90 이하 허용 안 함 |
| 오버레이 | `rgba(0,0,0,0.42)` 최대 | 사진이 살아야 함 |
| 컬러 | 채도 높고 선명한 낮 사진 우선 | 밤 사진은 S2 랜드마크에만 |
| 비율 | 16:9 기준, objectFit: cover | |

## ❌ 절대 금지 (비안산 원칙에서 확장)

```
1. 도시 불일치 — 다른 도시 사진을 해당 도시로 사용 금지
2. 흐릿·무채색 — 회색·안개·흐린 날 위주 사진 금지 (특수 경우 제외)
3. contrast 감소 — CSS contrast(0.95 이하) 절대 금지
4. AI 생성 이미지 — Unsplash 공식 사진만 (검증 필수)
5. 사람 클로즈업 — 개인 초상권 문제 있는 사진 금지
6. 텍스트 삽입 — 사진에 텍스트/로고 포함된 것 금지
7. 저화질 — q=90 이하 금지
```

## 🔍 큐레이션 절차 (신규 도시 추가 시 필수)

```
1. Unsplash 검색: "[도시명] aerial drone" → S1 후보 3장 선정
2. Unsplash 검색: "[도시명] landmark" → S2 후보 2장 선정  
3. Unsplash 검색: "[도시명] street market people" → S3 후보 2장 선정
4. 각 사진 ID 브라우저에서 직접 확인 (200 OK 필수)
5. 위치 태그 확인 (도시 불일치 방지)
6. CSS 필터 적용: brightness(1.42) saturate(1.68) contrast(1.08)
7. alt 텍스트: "[도시명] [내용] [슬라이드 목적]" 형식
```
