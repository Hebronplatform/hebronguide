## 핵심 철학

> **"섬기면서 수익을 내라"**  
> 모든 수익 활동은 사용자·업소·교회 모두에게 가치를 더해야 한다.

## 수익 루트 5가지

### 🥇 1번 — 웹사이트 제작 서비스 (즉시 시작 가능)

**개념**  
HebronGuide 등재 업소 중 공식 웹사이트가 없는 한인 업소에 제작 서비스 제안.  
업소는 HebronGuide에 등재 + 자체 웹사이트 확보. HebronGuide는 제작 수익 확보.

**서비스 패키지**

| 패키지 | 포함 내용 | 가격 |
|--------|----------|------|
| Starter | 1페이지 랜딩 (메뉴·위치·전화·SNS 링크) | $800 |
| Standard | 3-5페이지 (소개·메뉴·갤러리·예약 폼) | $1,500 |
| Premium | 온라인 주문·예약 시스템·카카오 채널 연동 | $3,000 |
| 유지관리 | 월간 업데이트·호스팅·도메인 관리 | $80-150/월 |

**영업 스크립트**  
> *"HebronGuide에 무료로 등재해드립니다. 웹사이트까지 만들어드리면  
> 구글 검색에도 뜨고, 한인 이민자 신규 고객이 훨씬 많이 찾아옵니다."*

**수익 시뮬레이션**
```
시애틀 한인 업소 중 웹사이트 없는 곳: 수백 개
월 5개 수주 × $1,500 평균 = $7,500/월
유지관리 누적 50곳 × $100 = $5,000/월 (2년 후)
연간 예상: $80,000-150,000
```

**영업 담당**  
- 교회 집사·사역자 (담임목사 서명 권한 필수 — 아래 정책 참조)
- 수수료: 계약금의 20% → 교회 계좌 입금 후 교회 내 분배

### 🥈 2번 — 프리미엄 비즈니스 등재

**무료 vs 프리미엄 차이**

| 항목 | 무료 | 프리미엄 ($80-200/월) |
|------|------|----------------------|
| 기본 정보 | ✅ | ✅ |
| TOP 10 상단 고정 | ❌ | ✅ |
| 사진 갤러리 (5장) | ❌ | ✅ |
| ✅ "검증됨" 배지 | ❌ | ✅ |
| 공유 시 강조 노출 | ❌ | ✅ |
| 월간 클릭수 리포트 | ❌ | ✅ |
| 한·영 소개 문구 | ❌ | ✅ |

**수익 시뮬레이션**
```
시애틀 30곳 × $120/월 = $3,600/월
20개 도시 × $3,600 = $72,000/월 (확장 후)
```

### 🥉 3번 — 제휴·소개 수수료 (가장 자연스럽고 고수익)

사용자가 이미 필요로 하는 서비스를 연결 → 수수료 발생

| 카테고리 | 파트너 대상 | 수수료 |
|----------|------------|--------|
| 🏠 부동산 | 한인 부동산 에이전트 | 거래금액 0.3-0.5% |
| 🏦 은행 계좌 개설 | Chase, WA Federal, 신한 | 계좌당 $50-150 |
| 🚗 자동차 보험 | Farmers, State Farm, AAA | 가입당 $100-300 |
| ⚖️ 이민 변호사 | Lynnwood 한인 로펌 | 의뢰당 $300-500 |
| 🏥 건강보험 | Premera, Apple Health 에이전트 | 가입당 $200-400 |
| 📱 휴대폰 개통 | T-Mobile, Mint Mobile | 개통당 $30-80 |
| ✈️ 항공권 | Korean Air, Asiana 제휴 | 발권당 $20-50 |

**구현 방식**  
앱 내 "전문가 연결" 버튼 → 파트너 연결 → 성사 시 수수료  
사용자 입장: 무료 상담 연결 (혜택으로 인식)

**연간 시뮬레이션**
```
부동산 월 2건 × $2,400 = $4,800/월
보험 월 10건 × $200 = $2,000/월
변호사 월 10건 × $400 = $4,000/월
합계: $10,800/월 → 연간 $130,000
```

### 4번 — 사용자 구독 프리미엄

**프리미엄 구독 $4.99/월**
- 즐겨찾기 무제한 + 기기 동기화
- "나만의 정착 플랜" 맞춤 추천
- 이민 서류 체크리스트 PDF 다운로드
- 월간 시애틀 한인 뉴스레터
- 파트너 업소 할인 쿠폰 (5-10%)

**원칙**: 전도 전략상 기본 기능은 영구 무료 유지.  
구독은 정착을 진지하게 준비하는 분들 대상 추가 가치 제공.

## 수익 로드맵

```
Phase 1 (지금 ~ 6개월): 신뢰 먼저, 수익 없음
  활동: 콘텐츠 완성, 사용자 확보, 브랜드 구축
  이유: 신뢰 없이는 아무것도 안 팔림

Phase 2 (6 ~ 12개월): 소프트 수익 시작
  월 목표: $3,000-5,000
  ├── 웹사이트 제작 월 2-3건 = $3,000
  └── 프리미엄 등재 초기 10곳 = $1,000

Phase 3 (12 ~ 24개월): 시스템화
  월 목표: $10,000-15,000
  ├── 웹사이트 제작+유지관리 = $5,000
  ├── 프리미엄 등재 30곳 = $3,600
  └── 제휴 수수료 = $5,000

Phase 4 (2년+): 다도시 스케일
  월 목표: $50,000+
  ├── 5개 도시 × Phase 3 수익 = $50,000
  └── B2B 기업 파트너십 추가
```

## ✝️ 교회 사역자 영업 권한 정책 (Pastor Authorization Policy)

> **이 정책은 선택이 아닌 필수다. 담임목사의 서명 없는 영업 활동은 HebronGuide 대표 자격이 없다.**

### 요건

| 항목 | 내용 |
|------|------|
| 담임목사 서명 | 영업 권한 부여 공문에 친필 서명 |
| 유효 기간 | 서명일로부터 **1년** |
| 연간 갱신 | 매년 갱신 — 갱신 없으면 자동 권한 소멸 |
| 갱신 시기 | 만료 30일 전 갱신 요청 |
| 수수료 지급 | 원칙적으로 **교회 계좌로** 입금 |

### 공문 양식

```
[HebronGuide 사역자 영업 권한 부여서]

교회명:           ___________________________
담임목사:         ___________________________
권한 부여 대상자: ___________________________
직분:             ___________________________

권한 내용:
  □ 업소 등재 제안
  □ 웹사이트 제작 영업
  □ 파트너십 협의
  □ 기타: ___________

유효 기간: ____년 __월 __일 ~ ____년 __월 __일

담임목사 서명: ________________  날짜: __________
교회 직인: (인)
```

### 연간 갱신 일정

```
1월:  HebronGuide → 각 교회 담임목사 갱신 안내 발송
2월:  교회 → 갱신 서명 공문 제출
3월:  HebronGuide → 확인 후 다음 1년 권한 활성화
미제출: 자동 소멸 → 갱신 후 재활성화
```

## ✝️ 가장 중요한 원칙 — 교회와 사업의 구조적 분리

> **"하나님과 재물을 겸하여 섬길 수 없느니라" (마태복음 6:24)**  
> **"내 아버지의 집으로 장사하는 집을 만들지 말라" (요한 2:16)**

### 왜 분리가 필요한가

교회와 사업이 구조적으로 뒤섞일 경우 발생하는 문제:

```
① 한국 교회 정서 — 목사 + 돈 = 즉각적 불신
② 성경적 문제 — 성전(교회)에 상업이 들어오는 구조
③ 법적 리스크 — 비영리(교회) + 영리(사업) 혼재
④ 신뢰 붕괴 — 하나가 흔들리면 셋 모두 무너짐
   (교회 신뢰 = HebronGuide 신뢰 = 목사 신뢰)
⑤ 선교 효과 감소 — "교회 앱"으로 보이면 비기독교인 진입 차단
```

### 올바른 구조 — 세 영역의 분리

```
✝️ 교회 (순수 사역 영역)
   Global Mission Church of Greater Seattle
   ├── 예배 · 목장 · 삶공부 · 선교
   ├── HebronGuide를 영적으로 지지·추천
   └── HebronGuide 재정에 관여하지 않음
   
         ↕  파트너십만
         (후원 · 홍보 · 네트워크 · 큐레이터 임명)
   
🌐 HebronGuide (독립 법인)
   별도 LLC 또는 501(c)(3) 비영리
   ├── 독립 이사회 운영 (목사는 이사회 멤버)
   ├── 교회와 재정 완전 분리
   ├── 크리스천 가치로 운영하되 교회는 아님
   └── 수익 일부 → 선교기금으로 자발적 기부
   
         ↕  기부 · 후원
   
🌍 선교 · 사회사업 기금
   교회 선교비 · 이민자 지원 · 타 도시 사역
```

### 성경적 근거

**사도행전 6:1-7** — 사도들이 재정·행정을 집사에게 분리한 사건

> *"우리가 하나님의 말씀을 제쳐 놓고 공궤를 일삼는 것이 마땅하지 아니하니"*

→ 영적 지도자는 영적 일에 집중  
→ 실제 운영·재정은 분리된 구조에서  
→ HebronGuide = 현대의 "집사 역할" 분리 구조

### 역할 구분표

| | 교회 | HebronGuide |
|---|---|---|
| **정체성** | 하나님의 집 | 크리스천 가치 독립 플랫폼 |
| **목적** | 예배·제자훈련 | 이민자 섬김·정보 제공 |
| **수익** | 헌금만 | 서비스 수익 |
| **목사 역할** | 담임목사 (본업) | 이사회 멤버 (자문) |
| **재정 관계** | 완전 분리 | 독립 운영 |
| **연결 방식** | 영적 파트너·추천자 | 독립 운영체 |

### 분리 후 창업자(목사) 보상 구조

```
교회 사례비:
  → 기존 그대로 (교회 당회 결정)
  → HebronGuide와 완전 무관

HebronGuide 이사회 자문료:
  → 이사회가 결정 (연간 소정 금액)
  → 외부에 투명하게 공시
  → "목사가 교회 이용해서 돈 번다" 비판 원천 차단

장기 (플랫폼 수익화 이후):
  → HebronGuide CEO/대표 급여 별도 책정
  → 이사회 승인·공시
  → 시장 기준: 월 $4,000-8,000 (성장 단계별)
```

### 분리가 히든 미션에 더 효과적인 이유

```
"교회 앱"으로 보이면:
  → 비기독교인: "나는 교회 안 다니는데" → 이탈

"독립 생활정보 앱"으로 보이면:
  → 누구나 부담 없이 사용
  → 자연스럽게 교회 탭 발견
  → "아, 교회 정보도 있네" → 관심 생기면 스스로 클릭
  → 히든 미션이 숨겨져 있을 때 더 효과적으로 작동

치킬레이(Chick-fil-A) 원칙:
  교회가 아니지만 크리스천 가치로 운영
  → 주일 휴무, 직원 존중, 품질 정직
  → 오히려 비기독교인도 신뢰하고 좋아함
  
HebronGuide = 한인 이민자를 위한 Chick-fil-A
```

### 큐레이터 보상 — 최종 구조

큐레이터는 교회 사역자이지만 보상은 HebronGuide 독립 법인에서 지급:

| 단계 | 보상 방식 | 금액 |
|---|---|---|
| Phase 1 (0-6개월) | 비현금 (타이틀·도구·실비) | 월 $50 실비 |
| Phase 2 (수익 발생 후) | 유치 등재 25% + 제작 소개 15% + 활동비 | 월 $100-300+ |
| Phase 3 (정착 후) | 월 정기 사례 + 성과 보너스 | 월 $300-600 |

**지급 경로**: Hebron Platform LLC → 큐레이터 담당 교회 계좌 → 담임목사 분배 결정

## 큐레이터 체계 — 담임목사 추천 원칙

> **담임목사의 추천은 단순 영업 정책이 아닌 목양적 인도다.**

```
작동 흐름:

담임목사: "HebronGuide 써보세요" (신뢰와 권위로 추천)
    ↓
교인이 앱 설치 (강한 신뢰 기반 진입)
    ↓
큐레이터(교인)가 SNS 운영 (지속적 지역 노출)
    ↓
앱 안에서 교회 탭 자연스럽게 발견
    ↓
새가족 모임 참석 → 목장 연결 → 공동체 편입

→ 이것이 디지털 선교의 완성된 파이프라인
```

**큐레이터 임명 원칙 요약**

| 항목 | 내용 |
|---|---|

| 급여 형태 | HebronGuide 독립 법인에서 사례 지급 |

| 타이틀 | HebronGuide 공식 디지털 사역자 |
