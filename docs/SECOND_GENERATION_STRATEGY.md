# 2세 한인 환영 전략 — HebronGuide의 답

**작성일: 2026-05-10 | 출처: CBS 뉴스 "번영 끝 정체된 한인교회…다음세대가 핵심" (2026-05-06)**

> *"네 자녀에게 부지런히 가르치며 ... 강론할 것이며"* — 신명기 6:7
>
> 1세대가 받은 복음의 빛이 2세대에게 계승되도록 — HebronGuide의 첫 관문이 답이 된다.

---

## 🎯 위기 진단 — CBS 보도 핵심 데이터

```
미국 내 한인 인구    : ~200만 (꾸준히 증가)
한인교회 성장        : 정체 (성도 수 늘지 않음)

한국계 미국인 기독교 비율:
  1세 (한국 출생) : 63%
  2세 (미국 출생) : 47%  ← 16%p 하락 ★

3대 변화 요인 (목회 현장 진단):
  ① 코로나 이후 온라인 예배 → "가나안 교인" 증가
  ② 1세 = 교회가 생활 도우미 / 2세 = 일·개인 우선
  ③ 한국어 중심 교회 → 언어·문화 격차로 2세 이탈
```

**현장 대응 흐름:**
- 관계 중심 목회 (테니스 클럽·집 초대 등 일상 만남)
- EM (영어 목회) 부서 확대
- 한국어 재교육 (3세에게 정체성 가르치기)
- K-Culture 접점 (K-pop으로 중고등부 접근)

## 🌿 HebronGuide의 답 — 4기둥과 정확히 매칭

폴 김 목사 진단 (2026-05-10):
> "2세를 어떻게 교회가 포용할 것인지, HebronGuide가 그 역할을 할 수 있는
> 대안을 제시하고 진행해야 겠다."

[`project_hebron_real_soul.md`]의 **4기둥**이 우연이 아니라 **2세 위기의 정확한 처방**이다:

| 4기둥 | 2세 위기 매칭 |
|---|---|
| **① 보여주는 전도** | 직접 전도 ❌ → 2세는 강요 거부, 일상 환대로만 다가옴 (마 5:16) |
| **② 준비된 교회** | EM·세대간 사역·관계 중심 교회 우선 매칭 |
| **③ 헤브론 제단** | 일상의 거룩함 = 교회 안 ≠ 신앙 (가나안 교인 포용) |
| **④ 연대 생태계** | 한 교회 ❌ → 2세 친화 EM 교회들의 도시별 연대 |

**이는 바뀌어야 할 것이 아니라, 이미 옳게 설계된 것이 명시되고 강화되어야 함.**

## 🛠️ 적용 단계 — 마 10:16 정신 (라벨 ❌, 경험으로 ✅)

> **2026-05-10 폴 김 목사 정정**:
> *"가나안 환영 배지... 이런 것은 뱀처럼 지혜롭고 비둘기처럼 순결 정신에
> 어긋나지 않을까? 이렇게 대놓고 이야기하면 무안하잖아."*

이 전략은 **공개 라벨·배지·카테고리화 없이** 표현·구조·경험에서 정신이 자연스럽게 흘러가도록 설계한다. 라벨은 양쪽 모두 무안하게 만들고 4기둥 ① 보여주는 전도를 훼손한다.

📖 정본: [`feedback_no_explicit_labeling.md`] (메모리)

### Phase 1 — 즉시 적용 (1-2주)

#### 1. 허브교회 SOP 자가 진단 — 내부 검토 항목 보강

[`HUB_CHURCH_APPLICATION_SOP.md`] 신청 양식에 EM·2세 검토 항목 추가하되 **반드시 내부 운영팀 검토용으로만 사용** (앱 외부 노출 ❌):

```
[자가 진단 — 내부 검토 전용]
[ ] EM (English Ministry) 부서 운영 여부
[ ] EM 부서 자율성 보장 정도 (자체 예산·결정권)
[ ] 2세·1.5세 청년 사역 자원 투입 정도
[ ] 출석률 압박 없는 환대 분위기 여부
```

위 답변은 **운영팀 매칭 결정 시 참고**. 새 이민자 2세 자녀를 둔 1세 부모가
거주지 인근 교회를 찾을 때 적합한 교회를 우선 매칭하는 데 활용. 앱에서는 라벨 없이 자연스럽게 노출.

#### 2. 교회 카드 표현 개선 — 배지 ❌, 자연스러운 desc ✅

❌ 회피: `🇺🇸 EM` / `🌿 가나안 환영` chip — 라벨링 ❌

✅ 권장: desc 본문에서 자연스러운 환대 문구. 예시:

```
"거실에서 시작되는 환대 공동체"           — 부담 없는 분위기 자동 전달
"한국어·영어 함께 드리는 예배"             — EM 운영 자연 표현
"누구든 처음 방문 환영"                   — 카테고리 ❌
"새 이민자 정착 동행"                     — 2세도 자연스럽게 포함
```

각 교회의 desc는 **그 교회의 실제 사역 표현**으로 작성. 라벨 ❌, 묘사 ✅.

#### 3. 영어 모드 첫 화면 — 일반 환대 메시지 (카테고리 ❌)

❌ 회피: "2세를 위한 메시지" / "가나안 교인 환영"

✅ 권장: 누구든 자연스럽게 받아들일 수 있는 일반 환대:

```
KO: "도시를 알고, 사람을 찾다 — 부담 없는 첫 관문"
EN: "Know your city. Find your people. — A welcoming starting point."
```

→ 영어 모드 사용자가 자연스럽게 자기에게 적용. 라벨 없이 도달 가능.

---

### Phase 2 — 장기 (1-3개월)

#### 4. 영어 우선 모드 — 구조적 환영

언어 토글로 영어를 선택하면 **모든 콘텐츠가 영어 우선**으로 표시. 이는 EM 친화·2세 친화의 **자동 결과**이지, 별도 카테고리 ❌.

```
구현 (이미 부분 적용):
- 한국어·영어·스페인어 토글
- 영어 모드: 헤드라인·desc·태그 영문 우선
- 한국어 fallback (필요 시 보조)
```

#### 5. K-Culture 접점 — 자연스러운 일상 콘텐츠

도시별 K-pop·BBQ·미트업 정보를 **이미 있는 맛집·이벤트 탭에 통합** (별도 "2세 트랙" ❌). 누구든 관심 있으면 발견.

#### 6. 첫 관문 메시지 — 자연 표현 (직접 라벨 ❌)

홈 화면 메시지로 누구든 자연스럽게 받아들이는 환대 표현:

```
✅ "당신의 속도로 둘러보세요" — 강요 없음, 카테고리 없음
✅ "도착부터 정착까지 함께"   — 누구든 적용
❌ "교회 안 가도 괜찮아요"     — 직접 라벨, 무안
❌ "가나안 교인 환영"          — 카테고리화, 무안
```

#### 7. 도시별 IHM 가정교회 디렉토리 — 이미 적용됨

별도 "2세·EM 교회" 서브탭 ❌. 대신 IHM 가정교회 본질이 이미 2세 친화:
- 거실 중심 (한국어 권위 ❌)
- 식사·관계 우선 (강요 ❌)
- 가족적 환대 (출석률 압박 ❌)

→ IHM 가정교회 자체가 라벨 없이 2세 친화 구조.

#### 8. 분기별 자가 진단 — 내부 운영용

허브교회 분기 점검에 다음 항목 추가 (운영팀 내부 검토만):

```
□ 우리 교회 2세 청년 정착 사례 (수치·스토리)
□ EM 부서 자율성 보장 여부
□ 출석률 압박 없는 환대 분위기 유지 여부
□ 가족적 거실 모임 활성도
```

## 🛡️ Hard Rules — 절대 금지

```
❌ 2세에게 한국어 강요
❌ 1세 가치관(권위·집단주의) 강요
❌ K-pop 등 문화 접점 우상화 (수단이지 목적 ❌)
❌ EM = 1세 통제 받는 부속 부서 묘사 (자율성 ✅)
❌ "교회 안 나오면 신앙 없다" 프레임 (4기둥 ③ 헤브론 제단 위반)
❌ 2세를 "잃어버린 양"으로 일방적 묘사 (그들의 자유와 선택 존중)
```
