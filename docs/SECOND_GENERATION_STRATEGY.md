# 2세 한인 환영 전략 — HebronGuide의 답

**작성일: 2026-05-10 | 출처: CBS 뉴스 "번영 끝 정체된 한인교회…다음세대가 핵심" (2026-05-06)**

> *"네 자녀에게 부지런히 가르치며 ... 강론할 것이며"* — 신명기 6:7
>
> 1세대가 받은 복음의 빛이 2세대에게 계승되도록 — HebronGuide의 첫 관문이 답이 된다.

---

## 🎯 위기 진단 — CBS 보도 핵심 데이터

```
미국 내 한인 인구    : ~200만 (꾸준히 증가)
한인교회 성장        : 정체 (성도 수 늘지 않음)

한국계 미국인 기독교 비율:
  1세 (한국 출생) : 63%
  2세 (미국 출생) : 47%  ← 16%p 하락 ★

3대 변화 요인 (목회 현장 진단):
  ① 코로나 이후 온라인 예배 → "가나안 교인" 증가
  ② 1세 = 교회가 생활 도우미 / 2세 = 일·개인 우선
  ③ 한국어 중심 교회 → 언어·문화 격차로 2세 이탈
```

**현장 대응 흐름:**
- 관계 중심 목회 (테니스 클럽·집 초대 등 일상 만남)
- EM (영어 목회) 부서 확대
- 한국어 재교육 (3세에게 정체성 가르치기)
- K-Culture 접점 (K-pop으로 중고등부 접근)

## 🌿 HebronGuide의 답 — 4기둥과 정확히 매칭

폴 김 목사 진단 (2026-05-10):
> "2세를 어떻게 교회가 포용할 것인지, HebronGuide가 그 역할을 할 수 있는
> 대안을 제시하고 진행해야 겠다."

[`project_hebron_real_soul.md`]의 **4기둥**이 우연이 아니라 **2세 위기의 정확한 처방**이다:

| 4기둥 | 2세 위기 매칭 |
|---|---|
| **① 보여주는 전도** | 직접 전도 ❌ → 2세는 강요 거부, 일상 환대로만 다가옴 (마 5:16) |
| **② 준비된 교회** | EM·세대간 사역·관계 중심 교회 우선 매칭 |
| **③ 헤브론 제단** | 일상의 거룩함 = 교회 안 ≠ 신앙 (가나안 교인 포용) |
| **④ 연대 생태계** | 한 교회 ❌ → 2세 친화 EM 교회들의 도시별 연대 |

**이는 바뀌어야 할 것이 아니라, 이미 옳게 설계된 것이 명시되고 강화되어야 함.**

## 🛠️ 적용 단계 — Phase 1 (즉시) → Phase 2 (장기)

### Phase 1 — 즉시 적용 (1-2주)

#### 1. 허브교회 SOP 자가 진단에 EM 항목 추가

[`HUB_CHURCH_APPLICATION_SOP.md`] 신청 양식 4번 환대 동행에 추가:

```
[ ] EM (English Ministry) 부서를 운영합니다.
[ ] EM 부서가 1세 부서와 별도 자율성을 가집니다.
[ ] 2세·1.5세 청년 사역에 의도적 자원 투입 중입니다.
[ ] 가나안 교인을 환영합니다 (출석률 압박 ❌).
```

신청 도착 시 위 4항목 모두 ✅인 교회는 **"2세 친화" 메타** 부여 → 앱에서 별도 배지 표시.

#### 2. 교회 카드에 "EM 운영" 표시 (`HebronGuide.tsx`)

[`getCityChurches()`] 데이터 모델에 새 필드:

```ts
type ChurchEntry = {
  emoji, tier, name, desc, tags,
  hasEM?: boolean;          // 영어 목회 부서 운영
  ganaanFriendly?: boolean; // 가나안 교인 환영
};
```

표시 UI: 카드 우측 상단에 `🇺🇸 EM` 또는 `🌿 가나안 환영` 배지 (작은 chip).

#### 3. 영어 모드 첫 화면 메시지 강화

현재 hero에 "Find Your Place" 정도만 있음. 2세 친화 메시지 추가:

```
KO: "1세 부모님이 받은 빛, 2세인 나도 자연스럽게 — 부담 없는 첫 관문"
EN: "Your parents' light, your way. — A no-pressure starting point for
    Korean Americans navigating life, faith, and city."
```

위치: 시애틀 도시 페이지 hero 부근 또는 "환영" 섹션.

### Phase 2 — 장기 (1-3개월)

#### 4. 2세·1.5세 전용 콘텐츠 트랙

```
도시별 가이드 영문 우선 모드
- 영어 우선 + 한국어 폴백 (현재는 한국어 우선)
- 1.5세·2세 페르소나 ("Jaehyun, 28, born in Seattle")
- 일·정착·관계·신앙 4축 콘텐츠
```

#### 5. K-Culture 접점 섹션

```
도시별 "K-Culture 만남"
- K-pop 댄스 모임·치킨 미트업·BBQ 게더링
- 1세대 한식 음식점 → 2세도 자연스럽게 (이미 맛집 탭에 있음)
- 분기별 K-Culture 이벤트 (도시 한인 미디어 + 허브교회 공동)
```

#### 6. "가나안 교인" 환영 메시지

```
홈 화면 또는 헬프 섹션에:
"교회를 잠시 떠나 있어도 괜찮습니다.
HebronGuide는 첫 관문입니다 — 강요가 없는 곳입니다."

→ 4기둥 ① 보여주는 전도 + 마 25:35 환대
```

#### 7. EM 친화 교회 디렉토리 (도시별)

```
교회 탭 → "2세·EM 교회" 서브탭 추가
- EM 운영 + 2세 친화 + 가나안 환영 ✅ 교회만 노출
- 영어 우선 표시 + 한국어 보조
- 새 이민자 2세 자녀를 둔 1세 부모도 검색 가능
```

#### 8. 분기별 "2세 영혼 점검" 자가 진단

```
허브교회 분기 점검에 2세 항목 추가:
□ 우리 교회 2세 출석률은 안정적인가?
□ EM 부서 자율성을 보장하는가?
□ 2세에게 한국 정체성 + 미국 정체성 둘 다 가르치는가?
□ 가나안 교인이 다시 돌아왔을 때 환영했는가?
```

## 🛡️ Hard Rules — 절대 금지

```
❌ 2세에게 한국어 강요
❌ 1세 가치관(권위·집단주의) 강요
❌ K-pop 등 문화 접점 우상화 (수단이지 목적 ❌)
❌ EM = 1세 통제 받는 부속 부서 묘사 (자율성 ✅)
❌ "교회 안 나오면 신앙 없다" 프레임 (4기둥 ③ 헤브론 제단 위반)
❌ 2세를 "잃어버린 양"으로 일방적 묘사 (그들의 자유와 선택 존중)
```

## 💡 영혼 점검 (2세 콘텐츠 작성 전 5질문)

```
□ Q0 — 2세 본인이 이 콘텐츠를 보고 호감을 느낄까? (강요 ❌)

□ Q5 — 그리스도의 환대를 한국 우월주의 없이 자연스럽게 보여주는가?
□ Q7 — 2세에게 미국적 정체성을 부정적으로 말하지 않는가?
□ Q9 — 2세도 송영(Doxology)에 함께 참여하도록 초대하는가?
```
