# HebronGuide 파트너십 프레임워크
**버전 1.0 | 2026년 | Hebron Platform LLC**

---

## 비전

> *"내가 나그네 되었을 때 너희가 영접하였다"* — 마태복음 25:35

HebronGuide는 한인 이민자가 새 도시에서 뿌리내리도록 돕는 디지털 정착 플랫폼입니다. 교회·교단·기관과의 파트너십은 이 사명을 위한 전략적 연합이며, 재정 거래가 아닌 섬김의 연대입니다.

## 공통 파트너십 원칙

### 1. 미션 우선 (Mission First)
파트너십의 목적은 수익이 아니라 한인 이민자 섬김입니다. 재정 조건은 사역을 가능하게 하는 수단이며 목적이 아닙니다.

### 2. 교회 노출 순서 — 돈이 아닌 신실함
HebronGuide 내 교회 노출 순서는 파트너십 재정 규모가 아닌 신학적 검증 Tier로만 결정됩니다.

```
Tier 1: IHM 인증 가정교회 (크기 무관, 최우선)
Tier 2: KSBC·NWBC 소속 SBC 교회
Tier 3: 기타 정통 교단
Tier 4: 독립·무소속 (검토 후)
```

### 3. 상호 섬김 (Mutual Service)
HebronGuide가 파트너에게 일방적으로 의존하거나, 파트너가 HebronGuide를 도구로만 사용하지 않습니다. 양방향 섬김이 기준입니다.

### 4. 투명한 재정
모든 파트너십 재정은 Hebron Platform LLC 계정으로 처리되며, 연간 리포트를 통해 파트너에게 공개됩니다.

## 재정 구조 개요

### 교단 파트너십 (IHM·KSBC·NWBC)

| 교단 규모 | 연간 파트너십 비용 | 교회당 환산 |
|---|---|---|
| 소형 (10-30교회) | $600/년 | $20-60/교회 |
| 중형 (30-100교회) | $1,500/년 | $15-50/교회 |
| 대형 (100교회+) | $3,000/년 | $10-30/교회 |

### 가정교회 개별 등재 (교단 파트너십 외)
- 등록비: $50 (1회, 평생)
- 연회비: $20/년
- 추천 크레딧: 추천 1교회 등록 시 연회비 $20 차감

### World Relief
- 재정 방향: World Relief → HebronGuide (서비스 제공 대가)
- 방식: 서비스 계약 또는 보조금(Grant)
- 규모: $5,000~20,000/년 (협의)
