# HebronGuide 파트너십 One-Page Brief
**미팅 전 배포용 | 2026**

---

## HebronGuide란?

한인 이민자가 새 도시에 도착한 첫 날부터 뿌리내리기까지 — 정착 정보, 한인 커뮤니티, 신실한 교회를 디지털로 연결하는 플랫폼입니다.

> *"내가 나그네 되었을 때 너희가 영접하였다"* — 마태복음 25:35

**현황:** hebronguide.com · 44개 도시 · 한·영·스페인어 3개 언어

---

## 우리가 섬기는 사람

매년 수만 명의 한인이 미국에 새로 도착합니다. 이들은 공통적으로 필요합니다:
- 정착 정보 (주택·은행·운전면허·의료·SSN)
- 한국어로 소통할 수 있는 커뮤니티
- 신뢰할 수 있는 교회 연결

---

## 파트너십이 만드는 것

```
파트너(교단·기관)          HebronGuide              한인 이민자
교회 네트워크       →   디지털 플랫폼       →   정착 + 교회 연결
재정 지원          ←   노출·데이터·연결    ←   신뢰·성장
```

---

## 파트너별 재정 구조 요약

| 파트너 | 연간 기여 | 혜택 |
|---|---|---|
| IHM | $1,200/년 | 소속 교회 전체 무료 등재 + Tier 1 우선 노출 |

| NWBC | $1,500/년 | 소속 교회 전체 무료 등재 + 서북미 우선 노출 |
| World Relief | $10,000/년 | 한국어 디지털 정착 서비스 파트너 |

## 교회 노출 원칙 (변하지 않는 것)

**돈이 아닌 신실함으로 순위가 결정됩니다.**

재정 지원이 많다고 더 잘 보이지 않습니다. 제자 만드는 교회, IHM 인증 가정교회가 대형 교회보다 먼저 노출됩니다.
