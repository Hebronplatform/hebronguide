## 🔑 3가지 구조적 핵심 발견

### 1️⃣ 캐나다 vs. 미국 — 결정적 차이
```
캐나다(Toronto·Vancouver):
  ✅ 공적 의료보험(OHIP·MSP) → 무료 의료
  ✅ 연방 다문화주의 → 강제추방 공포 없음
  ✅ 이민자 정착 서비스 체계적 (KCWA 등)
  ❌ 의료 대기 시간 길고 주거비 급등

미국:
  ⚠️ 의료비·이민 단속 리스크 항상 존재
  ⚠️ 주(州)별 이민 정책 편차 극심
  ✅ 취업 기회·경제 규모는 압도적
```

### 2️⃣ Sanctuary 정책 = 한인의 실질적 안전
```
2025년 트럼프 행정부 강경 이민 단속 환경에서:

✅ 안전: CA·WA·MA·NY (Sanctuary 주)
⚠️ 위험: TX·GA·FL (반이민 주법 강화)

Dallas, Houston, Atlanta 거주 한인:
→ 동일한 서류 미비 상황에서도 체감 공포 수준 다름
→ 교회, 한인 커뮤니티 내 이민 상담 수요 폭증
```

### 3️⃣ 커뮤니티 규모의 비선형 효과
```
LA 코리아타운 (250,000명+):
  → 의료·법률·언어·문화 문제를 커뮤니티 내에서 자체 해결
  → 한국어 전문의·변호사·세무사 충분
  → 사실상 작은 한국

Kansas City (8,000~10,000명):
  → 모든 문제가 개인에게 귀착
  → 한국어 의사 없음 → 중병 시 혼자 해결
  → 이민 위기 시 도와줄 법률 자원 없음
```

## 🏥 미진한 영역 — HebronGuide가 목소리를 내야 할 것

### 1. 의료 언어 장벽 (가장 긴급)
```
Nashville, Kansas City, Miami, Guadalajara, Monterrey:
한국어 가능 의사·통역 없음 → 중증 시 치료 지연·포기

요구 사항:
→ 각 도시 한인회 + 교회 협력으로 의료 통역 봉사 네트워크 구성
→ 대형 병원에 한국어 통역 서비스 공식 요청(Title VI 권리)
→ HebronGuide에 "한국어 가능 의사" 실시간 DB 구축
```

### 2. 이민 법률 지원 공백
```
Texas 3개 도시(Dallas, Houston), Georgia(Atlanta), Florida(Miami):
  반이민 주법 + 이민자 법률 지원 단체 부족 = 이중 위기

요구 사항:
→ 교회 연합으로 이민 법률 상담 정기 제공
→ KCLA(Korean Community Law Center) 등과 파트너십
→ HebronGuide "이민 법률 긴급 연락처" 도시별 목록화
```

### 3. 증오범죄 신고 후 대응 부재
```
Seattle, LA, New York:
아시아계 증오범죄 신고해도 실질적 경찰 조치 없음

요구 사항:
→ 각 도시 시의회에 아시아계 증오범죄 전담팀 구성 청원
→ Stop AAPI Hate 연계 신고·기록 시스템 HebronGuide 소개
→ 교민 안전 교육 (위기 대응, 신고 방법)
```

### 4. 소규모 도시 공동체 고립
```
Nashville, Kansas City, Miami:
한인 커뮤니티 규모 작아 모든 문제가 개인에게 귀착

요구 사항:
→ HebronGuide가 소규모 도시 한인 네트워크 연결 플랫폼 역할
→ 대도시 한인 단체와 소도시 한인 1:1 멘토링 프로그램
→ 온라인 한인 커뮤니티 가교 역할
```
