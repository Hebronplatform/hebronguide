## 🏛️ 왜 만들었는가 — 25년 목회 회고에서

**가장 좋은 전도는, 잃어버린 영혼들이 이미 믿는 우리 성도들의 모습을 보고 호감을 갖고 자연스럽게 찾아오게 하는 것이다.**

- **보여주는 전도** — 직접 전도가 아니라, 성도의 삶이 빛나는 전도 (마 5:16)
- **준비된 교회** — 영혼 구원이 준비된 교회로만 자연스럽게 인도
- **헤브론의 제단** — 아브람이 비전을 받고(창 13:14) 땅을 다닌(13:17) 후 헤브론에 거주하며 제단 쌓듯, 우리도 일상에서 하나님과 연합한 삶이 다리가 됨
- **연대하는 생태계** — 한두 교회 각개 전투 ❌, 모든 복음적 교회 연대 ✅

## 🛡️ 운영 원칙 — 신뢰의 4단계 게이트

```
신청 → 문서 검증 → 신원 조회 → 연결
검증 안 된 제공자는 매칭하지 않습니다.
```

**핵심 원칙 (Chick-fil-A 7원칙 + 환대 신학)**

- 📖 직접 전도 ❌ / 검증된 정보만 ✅ / 데이터 판매 ❌
- 🌿 **일요일 안식** (매주 일요일 운영팀 안식·상업 활동 ❌)

## 🤝 목회자 동행 초대 — 함께 만드는 Faith Ecosystem

**한 교회의 각개 전투가 아닙니다. 모든 복음적 교회가 연대하여 다민족 디아스포라를 함께 환대로 몰아갑니다.**

목회자가 동행해 주실 수 있는 자리:

```
✦ 그 도시의 신뢰 앵커 — 검증된 한인 교회의 보증
✦ 가정교회 Tier 1 인증 — IHM 인증 교회 우선 연결  
✦ 새가족 환대 동행자 — 도착부터 정착까지 함께
✦ 사역자·전문가 추천 — 검증된 한인 자격자 매칭
✦ 분기별 영혼 점검 — 함께 송영 자세 자가 진단
```

**목회자가 받으시는 것:**
- 새 이민자가 도착하기 전 미리 연결 → 환영 준비 가능
- 17개 도시 한인 이민자 정착 현황 데이터 공유
- 전국·전 세계 목회자·선교사 네트워크 참여
- 무료 — 목회 비용 부담 ❌, 수익 공유는 합의 시에만

**저희는 절대 하지 않는 것:**
- ❌ 목회자를 마케팅 채널로 도구화
- ❌ 교회·플랫폼 자금 혼용
- ❌ 한국 우월주의 / 민족주의
