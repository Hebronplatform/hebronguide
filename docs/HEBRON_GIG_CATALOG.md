# 헤브론 기그 (Hebron Gig) — 이웃 서비스 마켓플레이스
## 전체 서비스 카탈로그 v1.0

> *"네 이웃을 네 자신 같이 사랑하라"* — 마태복음 22:39  
> *"황금율을 모든 것에 적용한다"* — 마7:12

**핵심 철학:**  
한인이 한인만 섬기지 않는다.  
미국인·히스패닉·한인 — 이웃이 이웃을 섬긴다.  
재능과 기술이 있는 사람이 돈 벌고, 필요한 사람이 편하게 연결된다.  
HebronGuide는 연결창구로서 **15% 커미션**만 받는다.

## 🧭 비즈니스 모델

```
서비스 제공자 (Provider)  →  HebronGuide 앱 등록
         ↓
서비스 요청자 (Client)    →  앱에서 검색·예약
         ↓
서비스 완료             →  자동 결제
         ↓
HebronGuide 수수료: 15%  →  제공자 수령: 85%
```

모든 서비스 설명·채팅·리뷰 → 브라우저 번역 연동

## 🌍 다민족 연결 전략 — "이웃이 이웃을 섬긴다"

### 왜 다민족인가?

```
한인 커뮤니티만 섬기면:
  시장 = 한인 인구 (전체의 1-2%)

다민족으로 확장하면:
  시장 = 한인 + 히스패닉 + 미국인
       = 같은 동네 모든 이웃
       = 전체 인구의 100%
```

### 3개 언어 자동 통역 시스템

```
서비스 제공자 (한국어로 작성)
    ↓ 자동 번역
영어 클라이언트 → 영어로 표시
히스패닉 클라이언트 → 스페인어로 표시

채팅·리뷰·서비스 설명 모두 3개 언어 자동 적용
```

### 다민족 연결의 실제 예시

| 제공자 | 서비스 | 클라이언트 |
|---|---|---|
| 한인 아주머니 | Hebron HomeCook 한식 요리 | 미국인 가정에 한식 배달 |
| 히스패닉 정원사 | Hebron YardCrew 잔디 서비스 | 한인 가정 잔디 관리 |
| 미국인 대학생 | Hebron TutorAce 영어 튜터 | 한인·히스패닉 자녀 영어 과외 |
| 한인 IT 전문가 | Hebron TechFix 컴퓨터 수리 | 동네 어르신들 IT 도움 |
| 히스패닉 청소부 | Hebron Clean 청소 서비스 | 한인·미국인 가정 청소 |
| 한인 베이커 | Hebron BakeShare 한국 빵 | 미국·히스패닉 이웃에 판매 |

### 신뢰 레이어 (Trust Layer) — 헤브론 패스

```
기존 TaskRabbit: 낯선 사람 → 신뢰 부족
Hebron Gig:     검증된 제공자 + 허브교회 커뮤니티 → 신뢰 기반
```

**검증 없이 연결 안 함 (Hard Rule)** — 모든 제공자는 다음 4단계 게이트를 통과해야 등재·매칭됩니다.

```
Gate 1 · 신청        → 온라인 신청서 + 자기소개
Gate 2 · 문서 검증   → 정부 발급 신분증·SSN/ITIN·카테고리별 라이선스
Gate 3 · 신원 조회   → Checkr/Sterling 등 합법 백그라운드 체크
Gate 4 · 연결 승인   → 운영팀 최종 승인 + 환대 서약
```

추가로 통과 후 받을 수 있는 신뢰 등급(Trust Tier 1-3): 교회 출석 → 사역 경력 → 선교사 인증.
리뷰는 3개 언어로 자동 통합 표시.

## 💰 수익 구조

### 15% 커미션 기반

```
서비스 예시: Hebron Clean $100
  → 제공자 수령: $85 (85%)
  → HebronGuide 수수료: $15 (15%)

월 100건 연결 시:
  → 평균 서비스 $80
  → HebronGuide 월 수익: $80 × 15% × 100 = $1,200

도시별 500건/월 달성 시:
  → $80 × 15% × 500 = $6,000/월/도시
  → 5개 도시 × $6,000 = $30,000/월
```

### 추천 수수료 (Referral Fee) — 카테고리 9 정착·주거 전용

```
부동산 매매 거래 — 평균 시애틀 $600,000
  → 에이전트 측 커미션 3% = $18,000
  → HebronGuide referral 15% = $2,700/건

월 1건 성사 시 → $2,700/월
연 10건 성사 시 → $27,000/년/도시

렌트 거래 — 평균 월세 $2,200
  → 에이전트 측 1개월 커미션 = $2,200
  → HebronGuide referral 15% = $330/건
```

> 부동산은 거래 횟수가 적어도 1건 단가가 높아 수익 구조가 일반 긱과 완전히 다름.
> 1년 10건 성사면 일반 긱 1년치 매출에 맞먹음.
