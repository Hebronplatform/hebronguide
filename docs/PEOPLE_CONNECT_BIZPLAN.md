# HebronGuide — People Connect 비즈니스 플랜

> "나그네를 영접하였다" — 마태복음 25:35
> 헤브론은 플랫폼이 아니라 사람이다. 목사와 선교사의 신뢰 네트워크 위에 세워진 한인 디아스포라 커뮤니티.

---

## 비전 요약 (Vision Summary)

HebronGuide는 북미 17개 도시 한인 이민자들이 낯선 땅에서 "영접받는" 경험을 디지털로 구현한다. 6개 서비스는 모두 하나의 질문에서 출발한다: **"이 사람을 믿을 수 있는가?"** 그 답은 교회가 준다.

**플랫폼 철학:** 거래가 아닌 관계 → 알고리즘이 아닌 신뢰 → 앱이 아닌 공동체

---

## 신뢰 레이어: 헤브론 패스 (Trust Layer: Hebron Pass)

모든 6개 서비스를 관통하는 핵심 인프라.

| 인증 단계 | 내용 | 신뢰 효과 |
|---|---|---|
| **Level 1** | 교회 출석 확인 (담임/부목사 서명) | 기본 신원 보증 |
| **Level 2** | 사역 경력 / 봉사 이력 | 공동체 기여도 검증 |
| **Level 3** | 선교사 파송 기관 연동 (KWMA 등) | 글로벌 사역자 인증 |

헤브론 패스는 "교회 멤버십을 신뢰 여권(Trust Passport)으로" 변환하는 디지털 시스템이다. 앱 내 모든 프로필에 인증 배지 표시.

## 서비스 1: 헤브론 라이드 (Hebron Ride)

### 시장 규모
- Uber 글로벌 매출 $35B (2023), 북미 라이드쉐어 시장 $40B+
- 한인 이민자 공항 픽업 수요: 17개 도시 연 추정 120,000회 이상
- 한국어 가능 운전자 Uber/Lyft 내 비율 < 2% → 언어 미충족 수요 존재

### 벤치마크 분석 (Best Practice)
**BlaBlaCar** — 100M users, 26개국, $1.6B valuation
- 핵심 성공 요소: Community trust model (신원 인증 + 평점 + 장거리 카풀)
- 수수료 15% + 예약 수수료 고정
- 교훈: "아는 사람처럼 느껴지는 낯선 사람"이 서비스 철학

### 헤브론 고유성 (Hebron Differentiation)
- **교회 인증 운전자**: Hebron Pass Level 1 이상 필수
- **한국어 전용**: 공항 픽업, 정착 초기 이동 특화
- **목사 추천 시스템**: 담임목사 추천 운전자 우선 배치
- **공항 영접 패키지**: 픽업 + 정착 정보 안내 번들

### 수익 모델
- 수수료: 15% (BlaBlaCar 동일 수준)
- 공항 영접 패키지: $25 고정 추가 요금
- 월 구독 (운전자): $29/월 (우선 배정 + 보험 보조)
- **Year 1 목표**: 운전자 200명, 월 2,000회 → 월 매출 $45,000

### 실행 로드맵 (3단계)
1. **Phase 1 (0-6개월)**: LA·뉴욕·시카고 파일럿, 교회 파트너십 10곳, 운전자 50명 모집
2. **Phase 2 (6-18개월)**: 17개 도시 확장, 앱 공항 예약 기능 출시, 운전자 500명
3. **Phase 3 (18-36개월)**: 정기 통근 카풀 서비스 추가, 법인 계약 (한인 기업 출장)

## 서비스 2: 헤브론 스테이 (Hebron Stay)

### 시장 규모
- 북미 단기 임대 시장 $100B+, Airbnb $9.9B 매출 (2023)
- 한인 이민자 초기 정착 기간: 평균 1-3개월 임시 거처 필요
- 17개 도시 연간 한인 신규 이민자 추정 15,000-20,000명

### 벤치마크 분석 (Best Practice)
**Airbnb** — 7M 리스팅, Superhost 시스템 (신뢰 차별화)
**Homestay.com** — 문화 몰입 + 현지인 호스트 연결
- 핵심 교훈: 슈퍼호스트 배지 = 프리미엄 가격 정당화 + 예약률 40% 향상

### 헤브론 고유성 (Hebron Differentiation)
- **방 + 사람 + 정보**: 숙박이 아닌 정착 패키지 (교회 연결, 학교 정보, 병원 안내 포함)
- **교회 인증 호스트**: 담임목사 추천 가정만 리스팅
- **1-3개월 중기 특화**: Airbnb(단기) / 일반 임대(장기) 사이의 공백 공략
- **정착 멘토링**: 호스트가 선배 이민자로서 생활 가이드 제공

### 수익 모델
- 호스트 수수료: 12% (Airbnb 3% 대비 높지만 인증 프리미엄)
- 게스트 서비스 수수료: 8%
- 정착 패키지 번들: $150 (1회성, 정보 자료 + 교회 연결)
- **Year 1 목표**: 호스트 300가정, 월 500건 예약 (평균 $1,200/월) → 월 매출 $120,000

### 실행 로드맵 (3단계)
1. **Phase 1**: LA·뉴욕·애틀랜타 교회 호스트 100가정 모집, 베타 리스팅 오픈
2. **Phase 2**: 17개 도시 확장, 정착 패키지 표준화, 한인 이민 변호사 파트너십
3. **Phase 3**: 어학연수·단기 선교팀 숙박으로 타깃 확장

## 서비스 3: 헤브론 튜터 (Hebron Tutor)

### 시장 규모
- 미국 과외 시장 $12B (2023), 연 7% 성장
- 한인 교육열: 가처분 소득 대비 교육비 지출 비율 전체 이민자 중 최상위
- SAT·AP 과외 수요 집중 + 한국어 수업 (2세 대상) 니즈 급증

### 벤치마크 분석 (Best Practice)
**Wyzant** — 2.5M 학생, $80M 매출, 25% 수수료, 신원 검증 튜터
**VIPKid** — 온라인 1:1, 글로벌 플랫폼, 시간당 $14-22 교사 수익
- 핵심 교훈: 검증된 튜터 프로필 + 온라인 전환으로 지역 제약 극복

### 헤브론 고유성 (Hebron Differentiation)
- **이중 인증**: 교회 인증 + 학력/경력 인증 (교사 자격증, 대학원 이상)
- **한국어 과목 특화**: 2세 한국어 교육, 한국어 SAT Subject Test
- **온라인 17개 도시 커버**: 특정 도시에 없는 과목도 타 도시 튜터 연결
- **기독교 세계관 교육**: 요청 시 성경적 관점 통합 가능

### 수익 모델
- 플랫폼 수수료: 20% (Wyzant 25%보다 낮게, 교회 네트워크 활용)
- 튜터 인증 패키지: $49 (최초 1회)
- 그룹 수업 특화: 5명 소그룹 $30/인 → 튜터 $120/시간 (1:1 $60 대비 2배)
- **Year 1 목표**: 튜터 500명, 월 3,000시간 → 월 매출 $90,000

### 실행 로드맵 (3단계)
1. **Phase 1**: 교회 내 교사·대학원생 튜터 모집 (LA·뉴욕 집중), 100명 목표
2. **Phase 2**: 온라인 플랫폼 고도화, 17개 도시 전체 커버, 그룹 수업 기능 출시
3. **Phase 3**: 한국 본국 연결 (해외교포 자녀 대상 한국 학교 연계)

## 서비스 4: 헤브론 커넥트 (Hebron Connect — Community)

### 시장 규모
- Meetup: 50M users, 이익 관심사 그룹 플랫폼
- Bumble BFF: 우정 매칭, MAU 수천만
- Nextdoor: 위치 기반 이웃 연결, $700M+ 매출
- 한인 디아스포라 고독 문제: 이민 1세 고립감 지수 전체 이민자 최상위권 (UCLA 연구)

### 벤치마크 분석 (Best Practice)
**Nextdoor** — 위치 인증으로 신뢰 구축, 광고 수익 모델
**Bumble BFF** — 스와이프 기반 우정 매칭, 비낭만적 관계에 집중
- 핵심 교훈: 관심사·위치·목적이 명확할수록 매칭 품질 향상

### 헤브론 고유성 (Hebron Differentiation)
- **다목적 연결**: 친구·멘토·기도 파트너·비즈니스 파트너를 하나의 플랫폼에서
- **교회 인증 프로필**: 신원 보증으로 여성 사용자 안전 우선
- **17개 도시 크로스 매칭**: 출장·이사·방문 시 타 도시 한인 연결
- **소그룹 큐레이션**: 5-8명 소그룹 자동 구성 (교회 구역 모델 디지털화)

### 수익 모델
- Freemium: 기본 무료, 프리미엄 $9.99/월 (크로스 도시 매칭, 우선 노출)
- 광고: 한인 비즈니스 스폰서십 $500-2,000/월
- 소그룹 이벤트 수수료: 10%
- **Year 1 목표**: MAU 5,000명, 프리미엄 전환 15% → 월 매출 $25,000

### 실행 로드맵 (3단계)
1. **Phase 1**: LA·뉴욕·달라스 교회 청년부 타깃 베타, 가입 1,000명
2. **Phase 2**: 전 도시 확장, 소그룹 기능 출시, 교회 대시보드 제공
3. **Phase 3**: 한인 비즈니스 네트워킹 이벤트 온·오프라인 연계

## 서비스 5: 헤브론 매칭 (Hebron Matching — Marriage)

### 시장 규모
- 미국 온라인 데이팅 시장 $3.1B (2023)
- Hinge 매출 $400M, "designed to be deleted" 철학
- 한국 결혼정보회사: 선우·듀오 가입비 $1,000-3,000 (불투명한 가격 구조)
- **검증된 공백**: 북미에 한인 기독교인 대상 신앙 기반 결혼 플랫폼 없음

### 벤치마크 분석 (Best Practice)
**Hinge** — 진지한 관계 지향, 상세 프로필, "designed to be deleted"
**선우·듀오** — 고가 유료 모델, 대면 상담, 매칭 전문가 개입
**DUO America** — 해외 한인 대상이나 디지털화 미흡
- 핵심 교훈: 진지함의 신호(진입 장벽) + 투명한 가격이 신뢰 형성

### 헤브론 고유성 (Hebron Differentiation)
- **목사 추천서**: 담임목사 서명 필수 → 진지함의 최고 신호
- **언약 관계 (Covenant Relationship) 프레이밍**: 기독교 세계관 명시
- **신앙 + 가치관 정렬**: 교단, 신앙 연수, 사역 경험 매칭 변수
- **투명한 가격**: 한국 결혼정보회사 불투명 구조 대비 명확한 고정 요금
- **담임목사 알림**: 매칭 성립 시 양측 담임목사에게 통보 (선택 옵션)

### 수익 모델
- 가입비: $299 (6개월 멤버십, 선우·듀오 $1,000-3,000 대비 경쟁력)
- 프리미엄 멤버십: $499 (전담 매칭 코디네이터 배정)
- 성공 커플 헌금 프로그램: 결혼 성사 시 자발적 감사 헌금 → 선교 기금
- **Year 1 목표**: 가입자 500명 → 매출 $150,000

### 실행 로드맵 (3단계)
1. **Phase 1**: 3개 교단 파트너십, 목사 추천 시스템 구축, 가입자 200명 파일럿
2. **Phase 2**: 17개 도시 전체, 매칭 코디네이터 10명 (은퇴 목사·사모 활용)
3. **Phase 3**: 캐나다·영국·호주 한인 교회 네트워크 확장

## 서비스 6: 헤브론 네트워크 (Hebron Network — Pastors & Missionaries)

### 시장 규모
- KWMA (한국세계선교협의회): 한국인 선교사 22,000명, 170개국
- 북미 한인 교회: 약 4,500개, 담임목사 4,500명+
- Acts 29 Network: 700+ 교회, 목사 상호 책임 모델
- Lausanne Movement: 200+ 국가 복음주의 리더십 네트워크

### 벤치마크 분석 (Best Practice)
**Acts 29 Network** — 교회 개척 중심, 동료 책임(peer accountability), 연회비 모델
**Lausanne Movement** — 글로벌 컨퍼런스 중심, 200개국 리더 연결
**KWMA** — 22,000 한국 선교사 조직, 파송 기관 중심 (디지털 약함)
- 핵심 교훈: 컨퍼런스만으로는 일상적 연결 불가 → 디지털 플랫폼이 필수

### 헤브론 고유성 (Hebron Differentiation)
- **디아스포라 특화**: 북미 목사 + 해외 선교사 통합 플랫폼 (KWMA는 파송 기관 중심)
- **디지털 퍼스트**: 컨퍼런스가 아닌 일상적 디지털 교제·협력 채널
- **양방향 연결**: 목사(북미) ↔ 선교사(글로벌) 상호 지원 구조
- **자원 나눔**: 설교 자료·선교 보고서·기도 제목 공유 플랫폼
- **헤브론 패스 최고 레벨**: Level 3 인증자 = 최고 신뢰 보증인

### 수익 모델
- 교회 멤버십: $99/월 (교회 단위 구독)
- 선교사 개인 멤버십: $29/월
- 파송 기관 파트너십: $1,000/월 (기관 소속 선교사 프로필 관리)
- 연간 온라인 컨퍼런스: 스폰서십 $5,000-20,000
- **Year 1 목표**: 교회 200곳 + 선교사 500명 → 월 매출 $34,500

### 실행 로드맵 (3단계)
1. **Phase 1**: 17개 도시 한인 교회 200곳 파트너십, 선교사 네트워크 300명 초대
2. **Phase 2**: KWMA·KAICAM 공식 파트너십, 온라인 컨퍼런스 1회 개최
3. **Phase 3**: Lausanne Movement 연계, 영어권 디아스포라 선교사 확장

## 핵심 성공 요소 (Critical Success Factors)

1. **신뢰가 제품이다**: Hebron Pass 없이는 차별화 없음. 교회 파트너십이 모든 것의 기반.
2. **목사가 첫 번째 세일즈팀**: 담임목사의 추천이 가장 강력한 마케팅. 목사 온보딩에 집중.
3. **도시보다 교회**: 도시 수보다 파트너 교회 수가 성장 지표. 교회 1곳 = 잠재 사용자 300명.
4. **수익이 사명을 지속시킨다**: 영리와 선교적 사명의 균형. 수익의 10%는 선교 기금 헌납.
5. **언어와 문화**: 한국어 UX, 한국 문화 코드 (체면·관계·교육열) 반영이 필수.
6. **느리게, 그러나 깊게**: 빠른 확장보다 신뢰 인프라 구축 우선. 가짜 리뷰·사기 건 0 목표.
