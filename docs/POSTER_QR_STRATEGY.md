# HebronGuide 홍보 포스터·QR 전략
**작성일: 2026-05-08 | 폴 김 목사 비전**  
**핵심: 모든 사람의 셀폰에 HebronGuide가 들어가게 — QR 코드로 즉시 추가**

---

## 🎯 전략의 본질

```
포스터 본 사람 → QR 코드 스캔 → 셀폰에 HebronGuide
                ↓
       1초 만에 사용자가 됨 (앱스토어 ❌, 회원가입 ❌)
                ↓
   PWA 자동 설치 안내 → 홈 화면 아이콘 추가
                ↓
       영구 사용자 + 친구·가족 추천 도구
```

**황금률 자문**: *"내가 처음 외국 도시에서 한식당 찾을 때, 어떻게 정보를 얻고 싶을까?"*  
→ QR 한 번 스캔으로 모든 게 셀폰에 들어오는 것

---

## 🌐 3개 언어 전략

### 핵심 결정: 영어 + 한국어 + 스페인어

```
영어    — 미국·캐나다·글로벌 공통어
한국어  — 한인 디아스포라 1·1.5세대
스페인어 — 멕시코 3개 호스트 + 미국 라티노 4천만 인구
```

|---|---|---|---|
| **포스터** | ✅ | ✅ | ✅ |
| **홈 화면 환영** | ✅ | ✅ | ✅ |
| **월드컵 배너** | ✅ | ✅ | ✅ |

## 📱 QR 코드 전략

### QR 코드 종류

| 용도 | URL | 노출 위치 |
|---|---|---|
| **메인 (모든 도시)** | `hebronguide.com` | 글로벌 포스터 |
| **시애틀** | `hebronguide.com/seattle/` | 시애틀 포스터 |
| **달라스** | `hebronguide.com/dallas/` | 달라스 포스터 |
| **(8 호스트)** | `hebronguide.com/{city}/` | 도시별 포스터 |
| **월드컵 전용** | `hebronguide.com/worldcup/` (예정) | 경기장 주변 |

### QR 코드 추적 (Phase 2)

## 🌐 PWA 설치 유도 (앱스토어 없이)

### "셀폰에 추가하기" 단계

iOS (Safari):
```
1. 공유 버튼 (네모+위쪽 화살표)
2. "홈 화면에 추가"
3. 추가 ✅
```

Android (Chrome):
```
1. 메뉴 (⋮)
2. "홈 화면에 추가" 또는 "앱 설치"
3. 설치 ✅
```

### 앱 내 설치 안내 강화 (코드 작업 — 이미 부분 적용)

```typescript
// useInstallPrompt 훅 활용
- 첫 방문 후 30초 → "셀폰에 추가" 배너 표시
- 두 번째 방문 → 설치 권장 강조
- 설치 거부 시 → 7일 후 재시도
```

## ⚖️ 윤리 (마10:16 적용)

### 🐍 지혜
- 월드컵 = 절호의 기회 — 활용
- 3개 언어 = 시장 확장 = 지혜로운 결정

- 광고 기만 ❌ — "FIFA 공식" 등 거짓 표시 금지
