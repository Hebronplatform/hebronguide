## ⚠️ 이 전략의 진정한 목적 (CORE_VISION 통합)


## 🎯 핵심 비전

### 김구 선생의 비전 → HebronGuide 사명

> **"내가 원하는 우리나라는 세계에서 가장 아름다운 나라가 되기를 원한다.  
> 가장 부강한 나라가 되기를 원하는 것은 아니다.  
> 오직 한없이 가지고 싶은 것은 높은 문화의 힘이다."**  
> — 김구, 「나의 소원」 (1947)

**HebronGuide가 디지털 시대에 이 비전을 실현하는 방법:**

```
재외 한인 700만 → Korean American으로 자긍심을 가지고 살도록
                ↓
     각 도시에서 한국 문화의 힘을 보여주는 플랫폼
                ↓
     김구 선생의 "문화의 힘"이 디지털로 구현됨
```

## 🌊 HebronGuide 디아스포라 전략 4단계

### Phase 1: 문화 정체성 노출 (2026 Q2-Q3)

**앱 내부 구현:**
- ✅ Korean American 비전 섹션 (이미 적용)
- ⏳ **한인 문화 캘린더** — 광복절·한글날·추석·설날·코리안 페스티벌
- ⏳ **Korean American 영웅** 콘텐츠 — 도시별 한인 1세대 정착 스토리
- ⏳ **김구·안중근·유관순 인용** — 도시별 정착 가이드 도입부
- ⏳ **아리랑 외 한국 명곡** — 향수병 위로 BGM 옵션

**외부 확산:**
- 광복절(8/15) 시즌 캠페인
- 한글날(10/9) 한국학교 홍보
- 설날·추석 한인 가족 모임 정보

---

### Phase 2: 세대 브릿지 (2026 Q3-Q4)

**1세대 이민자 ↔ 2세 한국계 연결**

| 1세대 | 2세 | HebronGuide 역할 |
|---|---|---|
| 한식·교회 정보 필요 | K-pop·K-drama 친숙 | 두 세대 모두 사용 |
| 한국어 콘텐츠 | 영어 콘텐츠 | 한·영 병기 |
| 정착 정보 | 정체성 콘텐츠 | 통합 플랫폼 |
| 카카오톡 사용 | 인스타·틱톡 | 다채널 공유 |

**구체 적용:**
- 부모(1세) → 자녀(2세)에게 앱 추천 가능한 콘텐츠
- 자녀(2세) → 부모(1세)에게 한국 정체성 자랑하는 도구

---

### Phase 3: 자발적 확산 (ARMY 모델, 2027)

**파트너 교회·한인회를 ARMY化**

```
시애틀지구촌교회 → "HebronGuide Ambassador 1호 교회"
파트너 도시 교회 → 각 도시의 첫 번째 추천자
검증된 한인 업소 → "HebronGuide Verified" 배지 (자랑)
```

**도구:**
- 도시별 추천 링크 (`hebronguide.com/dallas?ref=GMC`)
- 추천 보상 (다음 도시 무료 설치 등)
- 추천 리더보드 (커뮤니티 자긍심)

---

### Phase 4: 한국 본토와 연결 (2028+)

**역방향 영향력 — 미주 한인이 한국에 영감 주기**

- 한국에서 미국 이민·유학 준비하는 분들에게 HebronGuide 노출
- 재외동포재단·한국문화원 협력
- 미주 한인 성공 스토리를 한국 본토에 전파
- "Korean American Wave" — 한류의 미주 버전
