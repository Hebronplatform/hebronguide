## 1️⃣ Ad Sales AI (광고 영업·디자인) — ⭐ 1순위

### 직무 (Job Description)
- 광고주 신청서 → 자동 광고 생성
- 한·영·스 3개 언어 카피 작성
- QR 코드 자동 발급 (추적 가능)
- 광고 위치 추천
- 도시·탭·시기 매칭

### 업무 흐름

```
광고주 신청서 (Notion Form 또는 hebronguide.com/광고)
    ↓
[입력 데이터]
- 업소명·주소·전화·웹사이트
- 광고 종류 (할인·신메뉴·이벤트)
- 예산 ($50/월·$100/월·$200/월)
- 도시·기간
    ↓
[Ad Sales AI 자동 처리 — 5분]
1. Claude API: 카피 3개 언어 생성
2. DALL-E / Midjourney API: 이미지 시안 3개
3. QR 코드 생성 (?ref=ad_001_seattle)
4. 광고 위치 추천 (어느 탭·어느 도시)
5. 가격 견적 자동 계산
    ↓
[폴 김 목사 검수 — 5분]
- 카피·이미지 OK/수정
- 마10:16 윤리 점검
- 가격 최종 승인
    ↓
[자동 게시]
- 앱 광고 슬롯 (지정 도시·탭)
- 카톡 단톡방 발송
- 인스타그램 스토리
- 이메일 (광고주에게 보고서)
```

### 사용 도구

| 도구 | 용도 | 비용 |
|---|---|---|
| **Claude API** | 카피 생성·번역 | ~$0.10/광고 |
| **DALL-E API / Midjourney** | 이미지 생성 | ~$0.50/광고 |
| **QR Code Generator API** | 추적 QR | 무료 |
| **Vercel Edge Function** | 서버리스 처리 | 무료 (제한 내) |
| **Supabase** | 광고 DB | 무료/저렴 |
| **Resend API** | 이메일 발송 | 무료 (월 3,000) |

**광고 1건 처리 비용: ~$1 → 광고주에게 $50/월 받으면 마진 $49**

### 광고 윤리 (마10:16 자동 점검)

AI가 자동으로 다음 체크:
```
□ 검증된 업소인가? (DB 조회)
□ "FIFA 공식" 등 거짓 문구 없는가?
□ 광고가 광고로 명시되는가?
□ 사용자 데이터 수집·판매 약속 없는가?
□ 한식·한인 사역 가치와 일치하는가?
```
하나라도 ❌ → 자동 거부 + 폴 김 목사에게 알림

## 2️⃣ Marketing AI (마케팅 매니저)

### 직무
- 매주 SNS 콘텐츠 자동 생성 (인스타·페북·카톡)
- 도시별·시즌별 캠페인 기획
- 한·영·스 번역
- 한인 미디어 보도자료 작성
- 광복절·추석·월드컵 등 시즌 캠페인 자동 트리거

### 업무 흐름
```
매주 월요일 09:00 자동 실행:
1. 도시별 추천 콘텐츠 5개 생성
   - 시애틀: "이번 주 검증된 신규 식당"
   - 달라스: "텍사스 한인 행사 안내"
2. 인스타그램 캡션 (3개 언어)
3. 카톡 단톡방용 짧은 메시지
4. 폴 김 목사 슬랙 알림 → 검수
5. 승인 시 자동 게시
```

### 시즌 캠페인 자동화

```
6월: 월드컵 환영 캠페인 자동 시작
8/15: 광복절 캠페인 자동 시작
9-10월: 추석 캠페인
10/9: 한글날 캠페인
설날·추석: 가족 모임 정보 강조
```

## 3️⃣ Content Curator AI (콘텐츠 큐레이터)

### 직무
- 새 도시 데이터 자동 수집·검증
- Yelp·Google·kSeattle·WowSeattle 자동 스크랩
- 검증된 데이터만 DB에 등재
- 폐업 업소 감지 (월 1회)

### 업무 흐름
```
폴 김 목사: "휴스턴 데이터 추가해 줘"
    ↓
Content Curator AI:
1. Yelp API: 휴스턴 한식당 상위 50개
2. Google Maps: 한국어 키워드 검색
3. kSeattle 스크랩 (광고 = 운영 확인)
4. 각 업소 정보 교차 검증
5. 검증 결과:
   - ✅ 검증됨 (3개 소스 일치): 자동 등재
   - 🟡 불일치: 사람 검수 요청
   - ❌ 폐업 가능: 미등재
    ↓
폴 김 목사: 5분 검수 후 승인
    ↓
자동 게시 + verified_at 기록
```

## 4️⃣ Customer Support AI (고객 응대)

### 직무
- 사용자 문의 자동 응답 (한·영·스)
- FAQ 자동 처리
- 광고 신청 안내
- 새 도시 추가 요청 처리
- 사용자 피드백 분류·전달

### 업무 흐름
```
사용자: "보스턴에 한식당 더 추천해 주세요"
    ↓
Customer Support AI:
1. 보스턴 식당 DB 조회
2. 사용자 위치 기반 거리순 정렬
3. 친절한 답변 작성 (한국어)
4. 추가 정보 제공 (지도 링크, 영업시간)
    ↓
즉시 응답 (10초 이내)
    ↓
복잡한 질문은 폴 김 목사에게 에스컬레이션
```

### 황금률 자동 적용

## 🛡️ AI 안전장치 (위험 관리)

### 사람 검수 필수 영역 (AI 자동화 ❌)
1. **새 파트너 교회 결정** — 영적 분별이 필요
2. **법인·금융 결정** — LLC·세금·계약
3. **Hard Rules 경계 사례** — 애매한 윤리 판단
4. **위기 대응** — 폐업·소송·언론 이슈

### AI 자율성 단계
```
Level 1 (지금): AI 제안 → 사람 100% 검수 → 게시
Level 2 (3개월): AI 자동 실행 → 사람 무작위 샘플 점검
Level 3 (1년): AI 완전 자율 → 사람 KPI만 모니터
```

월드컵 시기는 **Level 1** 유지 (품질 우선).

### 감사 추적 (Audit Trail)
모든 AI 결정·생성 콘텐츠는 다음 기록:
- 입력 데이터
- AI 모델 버전 (Claude Sonnet 4.6 등)
- 사람 승인 여부
- 게시 일시·위치
- 사용자 반응 (피드백·신고)
