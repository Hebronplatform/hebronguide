# HebronGuide 핵심 가치 선언문
**Core Values — 모든 도시에 적용되는 불변 원칙**  
제정: 2026-05-05 | 운영 주체: Global Mission Church of Greater Seattle

---

## 선언

> **"지식 없는 백성은 망하느니라"** (호세아 4:6)
>
> HebronGuide는 이민자·이주자·방문자가  
> 무지(無知)로 인해 손해 보지 않도록  
> 사명적으로 정보를 섬기는 플랫폼이다.

---

## 핵심 가치 3축

### ① 미국 사회 시스템 활용
> "미국이 제공하는 자원을 최대한 활용하도록 안내한다"

한인 이민자 대다수가 모르거나 활용 못 하는 미국 공공자원을 발굴하여 소개한다.

**포함 영역:**
- 공공 의료 (King County Public Health, Free Clinic, Apple Health)
- 무료 법률 지원 (Northwest Justice Project, KCBA Pro Bono)
- 세금 지원 (VITA — 연 1-4월 무료 세금 신고)
- 식품 지원 (Food Lifeline, Northwest Harvest, 211 서비스)
- 주거 지원 (King County Housing Authority)
- 정신건강 (Crisis Connections, NAMI)
- 취업 지원 (WorkSource WA — 무료 이력서·면접 코칭)
- 교육 (공립학교 무료 ESL, 도서관 무료 프로그램)
- 교통 (ORCA 카드, 대중교통 할인)

**원칙:** 영어가 장벽이어서, 정보가 없어서, 문화가 달라서 — 이 자원들을 못 쓰는 일이 없도록.

---

### ② 한인 커뮤니티 연결
> "같은 뿌리를 가진 사람들과 연결하여 소속감을 준다"

한인이 운영하는 검증된 업소·기관·커뮤니티를 큐레이션하여 신뢰할 수 있는 정보를 제공한다.

**포함 영역:**
- 한인 정착 지원 기관 (KCSC 등 각 도시 한인 서비스 센터)
- 주한국 총영사관·대사관
- 한인 운영 의료·치과·법률·부동산
- 한인 식당·카페·마트 (검증된 곳만)
- 한인 교회 커뮤니티 (가정교회 우선)
- 한인 온라인 커뮤니티 (카카오 오픈채팅, 네이버 카페)

**원칙:** 광고가 아닌 검증. "✅ 검증됨" 표시는 실제 확인된 곳만.

---

### ③ 타민족·주류 상업 발굴
> "한인 사회 밖에서 더 저렴하고 더 좋은 것을 찾아준다"

한인 이민자들이 한인 네트워크에만 의존하는 경향을 넘어, 미국 주류·타민족 커뮤니티가 이미 잘 활용하는 좋은 자원과 상업시설을 소개한다.

**포함 영역:**

| 분야 | 예시 | 이유 |
|------|------|------|
| 저가 마트 | Costco, WinCo, Grocery Outlet | H-Mart보다 30-50% 저렴한 품목들 |
| 아시안 마트 | 99 Ranch (중국), Uwajimaya (일본) | 신선식품·두부·해산물 더 저렴 |
| 타민족 맛집 | 베트남 Pho, 멕시칸 타코, 인도 뷔페 | $10-15에 든든, 가성비 탁월 |
| 절약 앱 | GasBuddy, Ibotta, Costco Gas | 월 $50-100 절약 가능 |
| 무료 자원 | 도서관, SAM 무료 목요일, Discover Pass | 이민자에게 알려지지 않은 혜택 |
| 중고 마켓 | Goodwill, Facebook Marketplace, Buy Nothing | 가구·의류·가전 저렴하게 |

**원칙:** "한인 것만이 좋은 것이 아니다. 더 좋은 것이 있다면 알려줘야 한다."

## 소비자·생활인 최대 유익 원칙

```
모든 정보를 추가할 때 묻는 질문:

① 이 정보가 이민자에게 실제로 돈을 아끼게 해주는가?
② 이 정보가 이민자에게 시간을 아끼게 해주는가?
③ 이 정보가 이민자의 삶의 질을 높여주는가?
④ 이 정보를 모르면 손해를 보는가?

하나라도 YES → 반드시 포함
모두 NO → 제외
```

**3대 소비자 유익 목표:**
- 💰 **절약** — 같은 물건·서비스를 더 싸게
- ⏱️ **시간 절약** — 정보 찾는 시간 단축
- 🌱 **삶의 질** — 더 풍요롭고 즐거운 이민 생활

## 사명적 배경

### 성경적 근거

> **"내가 나그네 되었을 때 너희가 영접하였다"** (마태복음 25:35)

이민자 = 현대의 나그네.  
HebronGuide = 그들을 영접하는 디지털 환대(Hospitality) 사역.

> **"지식 없는 백성은 망하느니라"** (호세아 4:6)

정보 격차 = 이민자가 당하는 불평등.  
HebronGuide = 그 격차를 메우는 정보 사역.

### 히든 미션 (드러내지 않되 스며들게)

```
Surface:  실용적 생활 정보
Hidden:   사람과 사람의 연결
Deepest:  사람과 하나님의 연결

"도시를 알고, 사람을 찾다"
  → 도시 정보로 신뢰 획득
  → 커뮤니티(교회)로 자연스러운 연결
  → 하나님께로 인도 (히든 미션 완성)
```

## 도시별 적용 원칙 (모든 도시 공통)

새 도시를 추가할 때 반드시 포함해야 할 3축 정보:

### 필수 1 — 해당 도시 미국 공공자원
- [ ] 한국 총영사관 또는 영사 서비스 (해당 도시 또는 가장 가까운 곳)
- [ ] 해당 주 DMV (운전면허)
- [ ] 가장 가까운 SSA 오피스
- [ ] USCIS 지역 오피스
- [ ] 무료 법률 지원 (해당 도시 Legal Aid)
- [ ] VITA 세금 신고 장소 (연 1-4월)
- [ ] 211 서비스 (전국 공통: 전화 211)
- [ ] 해당 도시 Public Health 기관

### 필수 2 — 한인 커뮤니티
- [ ] 한인 커뮤니티 센터 또는 생활상담소
- [ ] H-Mart 또는 한인 마트 위치
- [ ] 검증된 한인 식당 Top 5
- [ ] 가정교회 파트너 교회 (담임목사 인증)
- [ ] 한인 의료 (한국어 가능 의사/치과)
- [ ] 한인 온라인 커뮤니티 (카카오 오픈채팅 해당 도시 방)

### 필수 3 — 타민족·주류 절약 정보
- [ ] Costco 위치 (해당 도시)
- [ ] WinCo 또는 Grocery Outlet 위치
- [ ] 99 Ranch 또는 아시안 마트 (해당 도시)
- [ ] 저렴한 타민족 맛집 Top 3 (Pho, 타코, 인도 등)
- [ ] 해당 도시 무료 문화 이벤트 (박물관 무료일 등)
- [ ] 가장 저렴한 주유소 (Costco Gas, GasBuddy)

## Claude Code 적용 지침

이 파일을 읽은 Claude Code는 HebronGuide 작업 시:

1. **새 콘텐츠 추가 시** → 3축(미국·한인·타민족) 균형 확인
2. **새 도시 추가 시** → "도시별 적용 원칙 3축 체크리스트" 완수
3. **UI 개선 시** → "소비자·생활인 최대 유익" 기준으로 판단
4. **수익 기능 추가 시** → REVENUE_STRATEGY.md + OPERATING_PRINCIPLES.md 참조
5. **교회 관련 기능 추가 시** → CLAUDE.md 히든 미션 원칙 참조
