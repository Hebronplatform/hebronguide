# Part 0. +헤브론이란 무엇인가

## 0.1 정의

```
+헤브론 = 목사와 성도들이 함께 하나님의 선교를 이루는 글로벌 환대 생태계

구성:
  ├── 9개 Hebron 앱 (HebronGuide·HebronAltar·HebronShepherd 등)
  ├── 파트너 교회 네트워크 (준비된 교회들의 연합)
  ├── 성도 앰배서더 (보여주는 삶)
  ├── 채플린 네트워크 (교도소·이민국·병원·군)
  └── Hebron Gig Services (이웃을 섬기는 자원)

HebronGuide = 이 생태계의 첫 관문 (Step 1 of 9)
```

## 0.2 근원: 환대는 하나님의 마음

환대는 인간의 전략이 아닙니다. **하나님의 마음 그 자체**입니다.

```
하나님의 마음 = 환대 (롬 5:8 — 죄인 되었을 때 그리스도가 먼저 오심)
              ↓
아브라함이 그 마음을 받아 낯선 자에게 흘려보냈다 (창 18장)
              ↓
"나그네를 영접하였다" — 그것이 그리스도를 영접한 것 (마 25:35)
              ↓
우리가 그 마음을 받아 이민자·비신자·낯선 자에게 흘려보낸다
```

**환대는 전략이 아니라 하나님의 마음을 전하는 것입니다.**
받은 마음이기 때문에 줄 수 있습니다. 이것이 HebronGuide의 DNA입니다.

## 0.3 주체: 목사와 성도들 (잊혀진 사명)

```
올바른 주어:  목사와 성도들이 → 환대로 섬긴다 → 하나님 나라가 세워진다
잘못된 주어:  HebronGuide가  → 이민자를 돕는다 (이것은 수단일 뿐)
```

**목사와 성도들이 이 사명을 잊고 있다:**
- 교회 내부 프로그램에 묻혀
- 교회 간 성장 경쟁에 매여
- 바깥을 향한 눈을 잃었다

**HebronGuide는 이 잊혀진 사명을 다시 깨우는 도구입니다.**

## 0.4 대상: 생각보다 넓다

```
1차 대상: 이민자·이주자·관광객·방문자 (새로 온 사람)
2차 대상: 이미 그 도시에 살고 있는 비신자 (오래된 이웃)
공통점:   아직 하나님의 환대를 경험하지 못한 사람
```

HebronGuide는 "새로 온 사람"만이 아니라
**그 도시 안의 아직 닿지 않은 모든 사람**을 향해 있습니다.

## 0.5 자생적 사이클 (The Self-Generating Cycle)

이것이 전체 시스템의 심장입니다:

```
보여주는 성도의 삶 (각 도시·각 교회)
          ↓
  호감을 가진 영혼이 자연스럽게 다가옴
          ↓
HebronGuide — 첫 관문 (정착 정보·환대·커뮤니티)
          ↓
  준비된 교회 (파트너 Tier 1)
          ↓
  그리스도 안에서 뿌리내림
          ↓
  그 사람도 다시 "보여주는 성도"가 됨
          ↑ (사이클 반복)
```

이것은 억지가 아닙니다. 전략이 아닙니다. **삶의 자연스러움**입니다.

## 0.6 제4공간 (The Fourth Space)

```
제1공간: 가정 (집)
제2공간: 직장 (일)
제3공간: 교회·카페·커뮤니티센터 (모임)
제4공간: 온라인 + 오프라인이 동시에 살아있는 중립 공간
```

**HebronGuide는 제4공간입니다:**
- 특정 교회 소속이 아님 → 비신자도 부담 없이 들어옴
- 디지털로 연결되고 오프라인으로 만남이 일어남
- 교회 간 경쟁이 아닌 연합의 공간
- 방향은 분명히 교회로, 그리스도께로

## 0.7 교회 간 연합 (왜 경쟁이 아닌가)

```
지금: 교회 A vs 교회 B vs 교회 C (각개 전투)
      각 교회의 한정된 자원·신뢰·도달 범위
      흩어진 영혼들을 놓침

+헤브론: 교회 A + 교회 B + 교회 C (연합)
          한 도시 전체를 함께 품음
          어디서 만나든 같은 신뢰·같은 환대
          하나님의 선교를 함께 이룸
```

# Part 1. 양면 전략 (The Two-Layer Strategy)

## 1.1 왜 양면인가

비신자들이 느끼는 두 가지 경우:

```
조작감을 느낄 때:
  "이게 사실 교회 전도 작전이었구나" (숨겼다가 드러날 때)

신뢰를 갖게 될 때:
  "이 사람들은 믿는 사람들인데 나한테 아무것도 요구하지 않네"
  "진짜로 나를 도와주는 거구나"
```

**결론: 숨기면 배신감, 투명하면 신뢰감**

속임수가 아닌 **층위(Layer)** 전략입니다.

## 1.2 문 A — 공개 제품 (모두를 위한)

**hebronguide.com**

```
메시지:  "이 도시에서 잘 살아가기 위한 안내"
정체성:  "지역 커뮤니티 자원봉사자들이 만든 서비스"
언어:    종교 언어 최소화. 십자가·성경 구절 없음
교회:    "커뮤니티 앵커"로 존재 (종교 기관이 아닌 지역 신뢰 기관으로)
비신자:  부담 없이 사용 가능
핵심:    탁월한 서비스 = 증거
```

**비신자가 경험하는 흐름:**
```
정착 정보 → 맛집 → 커뮤니티 연결 → 교회 (하나의 커뮤니티 옵션으로) → 만남
```

## 1.3 문 B — 사역 제품 (목사·성도를 위한)

**+헤브론 / pastor.hebronguide.com**

```
메시지:  "하나님의 선교를 위한 제4공간"
정체성:  명확한 신학·사역 언어
내용:    교회 연합 플랫폼·목사 대시보드·훈련·네트워크
대상:    파트너 목사·성도·채플린·앰배서더
```

## 1.4 성경적 근거 — 요한복음 4장 모델

예수님이 사마리아 여인에게 처음 한 말:
> *"물을 좀 주시겠어요?"*

"저는 메시아입니다, 회개하세요"가 아니었습니다.

```
예수님의 접근:          HebronGuide 적용:
물 (실제 필요)      →   정착 정보 (실제 필요)
관계와 신뢰 형성    →   커뮤니티·환대
자연스러운 만남     →   교회·그리스도와의 만남
```

## 1.5 역사적 검증 모델

| 조직 | 정체성 | 비신자 경험 | 결과 |
|------|--------|-----------|------|
| YMCA | 명백히 기독교 | 탁월한 시설·커뮤니티 | 전 세계 비신자 이용 |
| 구세군 | 명백히 기독교 | 누구에게나 무조건 도움 | 비신자가 가장 신뢰하는 기관 |
| 칙필레 | 명백히 기독교 | 최고의 서비스 | 비신자가 더 좋아함 |

**공통점: 숨기지 않았다. 강요하지 않았다. 서비스가 진짜였다.**

## 1.6 환대의 투명성 선언

> "HebronGuide는 이웃을 진심으로 섬기는 것이
> 가장 강력한 선(善)이라고 믿는 커뮤니티가 만든 서비스입니다."

종교 언어 없이도 완전히 정직합니다.
이 문장이 신뢰를 만듭니다.

## 2.2 헤브론 제단 4가지 메시지 (창 13장)

```
① 보여주는 전도 (Demonstrative Witness)
   잃어버린 영혼들이 성도들의 삶을 보고
   호감을 갖고 자연스럽게 찾아오게 하는 것이 가장 좋은 전도
   (마 5:16 "너희 빛이 사람 앞에 비치게 하여")

② 준비된 교회 (Prepared Church)
   영혼구원을 위해 준비된 교회로만 연결한다
   준비 없는 교회로 데려가지 않는다

③ 헤브론의 제단 (Altar at Hebron)
   아브람이 비전(14절) → 실행(17절) → 제단(18절) 순서
   일상에서 하나님과 연합 = 자연스러운 다리

④ 연대하는 생태계 (Networked Ecosystem)
   한두 교회가 각개 전투하지 않는다
   모든 복음적·제자 만드는 교회가 연대하여
   함께 환대로 이 도시를 품는다
```

## 2.3 4가지 선교 신학 통합

```
① 구원론적 선교 (9Marks)
   HebronGuide 자체는 복음 선포 ❌
   복음적·제자 만드는 교회와 연결 ✅
   우리는 다리(bridge), 강단(pulpit)이 아님

② 참여 선교 (Christopher Wright)
   이민자 환대 = 하나님의 거대한 이야기에 참여
   정착 정보 큐레이션 = 창조 세계의 청지기 사역

③ 상황 중심 선교 (John Franke)
   시애틀 표준 + 각 도시 고유성
   각 도시의 성령의 역동적 생명에 따라 적응

④ 성례전적 선교 (Peter Leithart)
   식당 안내 = 단순 정보 ❌, 환대의 성례 ✅
   일상 정보 = 그리스도의 음성이 담긴 환영
```

## 2.4 복음이 흐르는 경로

```
환대 → 관계 → 신뢰 → 대화 → 복음 → 공동체 → 제자 → 재헌신
  ↑
HebronGuide가 이 흐름의 첫 관문을 디지털로 지원
```

## 2.5 세 기둥 (What)

| 기둥 | 의미 |
|------|------|
| 문화 상품 제작 | 콘텐츠 자체가 자랑할 상품 |
| 글로벌 커뮤니티 | 한국 입구 → 모두에게 열림 |
| 그리스도의 빛 | 강요 없이 자연스럽게 빛남 |

# Part 4. 홈페이지 전략

## 4.1 현재 문제

```
지금: 도시 타일 71개가 첫 화면
  → "이게 뭐지?" (이민자도, 목사도, 성도도 모름)

목표: 각 방문자가 "이게 나를 위한 것이구나" 즉시 알게
```

## 4.2 새 홈페이지 구조 (세 입구)

```
Section 1: 히어로
  "처음 미국 땅을 밟는 그 날부터
   이미 오래 살았지만 아직 연결되지 않은 그 날까지"
  (이민자도, 비신자도 포함된 문장)

Section 2: 세 길 (Three Paths)
  ┌──────────────┬──────────────┬──────────────┐
  │   새로 왔어요  │ 목사·사역자   │  이미 살고 있어요│
  │  정착 정보     │  +헤브론 사역  │  이웃 찾기     │
  └──────────────┴──────────────┴──────────────┘

Section 3: 도시 선택 (71개 그리드)

Section 4: "이 서비스는 이웃을 진심으로 섬기는 것이
            가장 강력한 선이라 믿는 커뮤니티가 만들었습니다"
            (투명하지만 종교 강요 없음)

Section 5: 파트너 교회·채플린 띠

Section 6: 앱 설치 (PWA)
```

## 4.3 앱(PWA) 전략

```
현재: 도시 페이지가 이미 앱처럼 작동
필요: "홈 화면에 추가하세요" 안내 추가

앱스토어 앱: 나중에 고려 (지금은 PWA로 충분)
각 도시 페이지: 폰에 설치 가능한 독립 앱처럼 경험
```

## 5.1 도시 선정 기준
