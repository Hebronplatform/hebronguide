## ⭐ 매 작업 전 의사결정 6질문

```
[중심 — 황금률]
0. "내가 사용자라면 이걸 받고 싶을까?"  ← 가장 중요

[지혜]
1. 시장·문화·인간 본성을 정확히 이해한 결정인가?
2. 효과적으로 목적을 달성하는가?

[순결]
3. 사람을 수단으로 쓰지 않는가?
4. 정직하고 투명한가?

[통합]
5. 그리스도의 환대를 자연스럽게 보여주는가?

→ 6가지 모두 YES일 때만 진행
   질문 0이 NO면 다른 5개가 YES여도 ❌
```

## 📐 8가지 영역별 실행 규칙

### 영역 1. 콘텐츠 작성

**황금률 자문**: "내가 처음 이민 왔을 때 어떤 정보를 받고 싶었을까?"

| 규칙 | 실행 |
|---|---|
| ✅ 검증 우선 | 미검증 정보는 "정보 확인 중" 표시 |
| ✅ 출처 명시 | 모든 데이터에 마지막 확인 날짜 |
| ✅ 한·영 병기 | 1.5세·2세도 사용 가능하게 |
| ✅ 도시별 고유성 | 시애틀 fallback ❌, 도시 특성 보존 |
| ❌ 추측 금지 | 모르는 정보는 게시 ❌ |
| ❌ 광고성 도배 ❌ | 한 도시당 한식당 5개만 노출 (Top5) |
| ❌ 민족주의 ❌ | "한국 최고" 메시지 금지 |

**광고 정책 (Yelp 모델):**
- 광고는 "이것은 광고입니다" 명시
- 광고 vs 큐레이션 명확히 구분

**상세**: `docs/DINING_GUIDE.md`

---

### 영역 2. 데이터 검증 (Yelp + 한인 + 교회 3중)

**황금률 자문**: "내가 식당 찾을 때 어떤 정보를 신뢰할까?"

```
1. AI가 공식 웹사이트 검색 → 1차 수집
2. 전화번호·주소 개별 대조 확인
3. ✅ 검증 완료 후 입력
4. 날짜 기록 → "마지막 확인: YYYY-MM-DD"
5. annual_update.py → 연 1회 자동 점검
```

**검증 표시 체계:**
| 표시 | 의미 |
|---|---|
| `✅ 검증됨` | 공식 사이트·직접 확인 완료 |
| `🔗 웹사이트` | 사용자가 직접 최신 정보 확인 가능 |
| `★ Yelp 4.2` | Yelp 검증 별점 |
| (표시 없음) | 미검증 → 공개 보류 |

**소스 우선순위:**
1. kSeattle·WowSeattle 업소록 (광고 = 운영 확인)
2. 공식 웹사이트
3. Google Maps + Yelp
4. 한인 미디어 (한국일보·기독일보)

---

### 영역 3. 도시 확장 (시애틀 표준 + 도시 고유성)

**황금률 자문**: "내가 새 도시 사용자라면 시애틀 데이터만 보면 어떨까?"

**시애틀 표준 (모든 도시 동일):**
- 9탭 구조·16 Quick Menu·더보기 시트
- ScreenHeader 디자인·Korean American 비전
- 검증 체계·카테고리 필터

**도시 고유성 (반드시 보존):**
- TOP5 맛집·정착·탐방 (도시별 검증)
- 거주지·면허·세금·이민·건강보험
- 태그라인 차별화

**새 도시 추가 체크리스트:**
```
□ 시애틀 템플릿 그대로 복사 (구조 동일)
□ TOP5_RESTAURANTS_[CITY]: 도시 한식당 5선 + Yelp 별점
□ TOP5_SETTLE_[CITY]: 한인회·H-Mart·DMV·의료·총영사관
□ TOP5_EXPLORE_[CITY]: 명소 5선 + 한인타운
□ areasKo/En: 거주지 5곳 (렌트·학군·교통)
□ adminKo/En: SSN/SIN·면허·의료·이민 (지역 특화)
□ financeKo/En: 세금 특이사항 (소득세 유무)
□ CitySlug + CITY_CONFIGS 등록
□ Top5Banner 분기에 연결
```

**위반 금지:**
- ❌ 모든 도시에 시애틀 식당 표시 (Park's BBQ in LA로 표시 X)
- ❌ 모든 도시에 같은 면허 절차 (TX 90일 / CA 10일 / MA 60일)
- ❌ 모든 도시에 "주 소득세 없음" (CA·NY는 있음)

### 영역 4. 운영 리듬 (Chick-fil-A 7원칙)

**황금률 자문**: "사역자도 안식이 필요하다 — 내가 안식이 필요하면 다른 사람도 그렇다"

| Chick-fil-A 원칙 | HebronGuide 적용 |
|---|---|
| **1. 일요일 안식** | 일요일 배포·상업 콘텐츠 ❌ |
| **2. 한 가지 탁월** | 한인 정착만. 범용 앱 기능 추가 ❌ |
| **3. My Pleasure** | 모든 UI 문구는 따뜻하고 섬기는 어조 |
| **4. 간접 증거** | 성경구절·노골적 전도 문구 ❌ |
| **5. 선발 기준** | 아무 업소나 등재 ❌. 검증 기준 유지 |
| **6. 장기 관점** | 단기 수익 < 장기 신뢰 |
| **7. 지역 투자** | 기본 기능 영구 무료 |

**일요일 원칙:**
```
✝️ 운영팀 안식 — 일요일 업무·배포 없음
✝️ 상업 SNS 금지 — 광고·홍보 ❌
✝️ 커뮤니티 — 예배·묵상·감사만
```

**상세**: `docs/OPERATING_PRINCIPLES.md`

### 영역 5. 목표 관리 (Alignify 시스템)

**황금률 자문**: "내가 팀원이라면 어떤 목표 시스템을 원할까?"

**3개 연간 목표 (2026):**
1. 도시: 3 → 8개 (Dec 31)
2. MAU: 220 → 1,000 (Dec 31)
3. 월수익: $200 → $500 (Dec 31)

**모든 목표 공식:**
```
From X → To Y → By When → And Why
담당자: 1명 (위원회 ❌)
```

**리듬:**
- 매주: 60초 점검 (🟢🟡🔴)
- 매월: 30분 패턴 분석
- 분기: 2-3시간 리셋
- 연간: 반나절 설정

**상세**: `docs/GOAL_SYSTEM.md`

### 영역 6. 수익·법인 (LLC 우선, 비영리는 추후)

**황금률 자문**: "내가 후원자·파트너라면 어떤 운영 구조를 신뢰할까?"

**구조적 분리 원칙:**
```
HebronGuide LLC ≠ 시애틀지구촌교회
재정 완전 분리 (계좌·인력·자원)
영적 파트너십만 유지
```

**수익 활동 3원칙:**
1. 섬김이 먼저, 수익은 결과
2. 투명한 관계 ("HebronGuide 파트너입니다")
3. 담임목사 인지·동의 하에

**교회 사역자 영업 권한:**
- 담임목사 서면 동의 필수 (1년 유효, 연간 갱신)
- 수수료는 교회 계좌로 지급 (개인 ❌)

**상세**: `docs/REVENUE_STRATEGY.md`, `docs/LLC_SETUP_WORKFLOW.md`

### 영역 7. 한인 교회 등재 (4-Tier)

**황금률 자문**: "내가 신앙 공동체 찾을 때 어떤 검증을 원할까?"

**Tier 1 ⭐ 가정교회 (최우선):**
- 국제가정교회 사역원(IHM) 인증
- 가정교회 3축: 예배 + 목장 + 삶공부
- 표시: ⭐ "가정교회"

**Tier 2 ✅ SBC·정통 보수:**
- SBC·KSBC·KAPC·PCA 등
- 표시: ✅ "정통 교단"

**Tier 3 ✅ 기타 정통:**
- UMC·PCUSA·EFCA·CMA·AG
- 표시: ✅ "검증됨"

**Tier 4 🔍 독립·무소속:**
- 사역팀 신학 검토 후
- 표시: 🔍 "사역팀 확인"

**🚫 등재 X — 이단·사이비:**
- 신천지·통일교·JMS·하나님의교회·여호와의증인·몰몬교

## 🚨 절대 금지 사항 (Hard Rules)

다음은 어떤 상황에서도 **절대 위반 금지**:

```
❌ 직접 전도 메시지 (앱 내)
❌ 성경 구절 도배
❌ 검증되지 않은 정보 게시
❌ 사용자 데이터 판매
❌ 광고를 큐레이션처럼 위장
❌ "한국 최고" 민족주의 메시지
❌ 일요일 상업 활동
❌ 교회 자금과 LLC 자금 혼용
❌ 이단 단체 등재
❌ AI 추측 데이터 검증 없이 게시
```

---

## ✅ 절대 지킬 것 (Hard Yes)

```
✅ 매 작업 전 6질문 통과
✅ 모든 정보에 검증 날짜
✅ 광고는 광고로 명시
✅ 한·영 병기 (가능한 한)
✅ 시애틀 표준 + 도시 고유성
✅ 일요일 안식
✅ 매주 목표 60초 점검
✅ 인사이트 즉시 문서화 + 적용
✅ 교회·LLC 재정 완전 분리
✅ 사용자 프라이버시 최우선
```
