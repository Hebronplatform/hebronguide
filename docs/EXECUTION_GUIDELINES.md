```
[중심 — 황금률]
0. "내가 사용자라면 이걸 받고 싶을까?"  ← 가장 중요

[지혜]
1. 시장·문화·인간 본성을 정확히 이해한 결정인가?
2. 효과적으로 목적을 달성하는가?

[순결]
3. 사람을 수단으로 쓰지 않는가?
4. 정직하고 투명한가?

[통합]
5. 그리스도의 환대를 자연스럽게 보여주는가?

[선교 신학 — 분기별 점검]
6. 우리가 종교 기업으로 변질되고 있는가?
7. 식민주의·민족주의에 빠지지 않았는가?
8. 세상과 분리되거나 동화되지 않았는가?
9. **모든 활동이 결국 송영(Doxology)을 향하는가?** ← 가장 깊은 질문

→ 9가지 모두 YES일 때만 진행
   질문 0과 9가 NO면 다른 모든 게 YES여도 ❌
```

📖 **선교 신학 상세**: [`MISSIONAL_THEOLOGY.md`](MISSIONAL_THEOLOGY.md) — 4가지 견해 통합 + 송영 자세

## 📐 8가지 영역별 실행 규칙

### 영역 1. 콘텐츠 작성

**황금률 자문**: "내가 처음 이민 왔을 때 어떤 정보를 받고 싶었을까?"

| 규칙 | 실행 |
|---|---|
| ✅ 검증 우선 | 미검증 정보는 "정보 확인 중" 표시 |
| ✅ 출처 명시 | 모든 데이터에 마지막 확인 날짜 |
| ✅ 한·영 병기 | 1.5세·2세도 사용 가능하게 |
| ✅ 도시별 고유성 | 시애틀 fallback ❌, 도시 특성 보존 |
| ❌ 추측 금지 | 모르는 정보는 게시 ❌ |
| ❌ 광고성 도배 ❌ | 한 도시당 한식당 5개만 노출 (Top5) |
| ❌ 민족주의 ❌ | "한국 최고" 메시지 금지 |

**광고 정책 (Yelp 모델):**
- 광고는 "이것은 광고입니다" 명시
- 광고 vs 큐레이션 명확히 구분

**상세**: `docs/DINING_GUIDE.md`

### 영역 2. 데이터 검증 (Yelp + 한인 + 교회 3중)

**황금률 자문**: "내가 식당 찾을 때 어떤 정보를 신뢰할까?"

```
1. AI가 공식 웹사이트 검색 → 1차 수집
2. 전화번호·주소 개별 대조 확인
3. ✅ 검증 완료 후 입력
4. 날짜 기록 → "마지막 확인: YYYY-MM-DD"
5. annual_update.py → 연 1회 자동 점검
```

**검증 표시 체계:**
| 표시 | 의미 |
|---|---|
| `✅ 검증됨` | 공식 사이트·직접 확인 완료 |
| `🔗 웹사이트` | 사용자가 직접 최신 정보 확인 가능 |
| `★ Yelp 4.2` | Yelp 검증 별점 |
| (표시 없음) | 미검증 → 공개 보류 |

**소스 우선순위:**
1. kSeattle·WowSeattle 업소록 (광고 = 운영 확인)
2. 공식 웹사이트
3. Google Maps + Yelp
4. 한인 미디어 (한국일보·기독일보)

### 영역 3. 도시 확장 (BTS 흐름 + 복음적 교회 + 시애틀 표준)

📖 **상세 전략**: [`EXPANSION_STRATEGY.md`](EXPANSION_STRATEGY.md)

**황금률 자문**: "내가 새 도시 사용자라면 시애틀 데이터만 보면 어떨까?"

#### 진출 전 5질문 (6질문 통과 후 추가)

```
1. 이 도시에 BTS 또는 K-Wave 영향력이 있는가?  (문화 토양)
2. 한인 인구 1만+ 인가?                       (디아스포라 규모)
3. 복음적·제자 만드는 교회가 있는가?           (영적 인프라)
4. 그 교회와 신뢰 관계 가능한가?               (관계)
5. 도시 콘텐츠 검증할 현지 협력자가 있는가?     (실행)

→ 5가지 모두 YES일 때만 진출
   3-4번 NO면 절대 진출 ❌
```

#### 시애틀 표준 (모든 도시 동일)
9탭·16 Quick Menu·더보기·ScreenHeader·Korean American 비전·검증·카테고리 필터

#### 도시 고유성 (반드시 보존)
TOP5 맛집·정착·탐방 / 거주지·면허·세금·이민·건강보험 / 태그라인

#### 새 도시 추가 체크리스트
```
[진출 자격]
□ BTS·K-Wave 도시인가?
□ 한인 인구 1만+ 확인
□ 복음적·제자 만드는 교회 발견 (Tier 1 가정교회 우선)
□ 파트너 목사님 직접 만남 완료

[코드 작업]
□ TOP5_RESTAURANTS_[CITY] (Yelp 별점 포함)
□ TOP5_SETTLE_[CITY] (한인회·H-Mart·DMV·의료·총영사관)
□ TOP5_EXPLORE_[CITY] (명소 + 한인타운)
□ areasKo/En (거주지 5곳)
□ adminKo/En (SSN/SIN·면허·의료·이민)
□ financeKo/En (세금 특이사항)
□ CitySlug + CITY_CONFIGS 등록
□ Top5Banner 분기 연결
```

#### 위반 금지
- ❌ K-Wave 흐름만 보고 영적 인프라 없는 도시 진출
- ❌ 명목적 교회와 파트너십
- ❌ 교회를 사용자 확보 채널로만 활용
- ❌ 모든 도시에 시애틀 식당 표시
- ❌ 모든 도시에 같은 면허 절차 / 동일 세금 정보

### 영역 7. 한인 교회 등재 (4-Tier)

**황금률 자문**: "내가 신앙 공동체 찾을 때 어떤 검증을 원할까?"

> **2026-05-18 폴 김 목사 신학적 정정 (절대 원칙)**  
> Tier 1의 기준은 **IHM 인증 여부가 아니다.**  
> 기준은: **"영혼구원하여 그리스도의 제자를 만드는 사명에 충실한가?"**  
>  
> "가정교회 장려 운동"은 조직 명칭이 아니라 **신약교회 회복의 정신**이다.  
> IHM 회원교회이든 아니든, 소그룹 이름이 목장·셀·다락방·구역 무엇이든 —  
> 이 사명 DNA를 가진 교회가 Tier 1이다.

---

#### ⭐ HebronGuide가 장려하는 교회의 DNA

```
"그 가정교회는 교회 안에 들어온 VIP만 아니라,
 공항에서부터 모든 성도이 새가족팀이 되어
 환대하고 환영하자는 취지로 만든 거야."
— 폴 김 목사 (2026-05-18)
```

**HebronGuide의 교회 비전 = 이 DNA의 디지털 확장:**
- 물리적 교회가 하던 것: 공항에서부터 성도들이 새로 온 한인을 환대
- HebronGuide가 하는 것: 비행기에서 내리는 순간부터 디지털 환대

**Tier 1 교회 판별 기준 (3가지 모두 YES):**
1. 영혼구원과 제자 만드는 사명에 분명한가?
2. 소그룹을 통한 목자-양 관계가 있는가? (이름 불문: 목장·셀·다락방·구역·소그룹)
3. 새로 온 사람을 환대하는 문화가 있는가?

---

**Tier 1 ⭐ 신약교회 회복 DNA (최우선):**
- 영혼구원 → 그리스도의 제자 만드는 사명에 충실
- 소그룹 공동체를 통한 목자-양 관계 (이름 불문)
- 새가족 환대 문화 — 이상적으로는 "공항부터"
- IHM 회원 포함 + 같은 DNA를 가진 모든 교회
- 표시: ⭐ "신약교회 DNA"

**Tier 2 ✅ SBC·정통 보수:**
- SBC·KSBC·KAPC·PCA 등
- 건전한 교리 + 복음적 사역
- 표시: ✅ "정통 교단"

**Tier 3 ✅ 기타 정통:**
- UMC·PCUSA·EFCA·CMA·AG
- 표시: ✅ "검증됨"

**Tier 4 🔍 독립·무소속:**
- 사역팀 신학 검토 후
- 표시: 🔍 "사역팀 확인"

**🚫 등재 X — 이단·사이비:**
- 신천지·통일교·JMS·하나님의교회·여호와의증인·몰몬교

## 🚨 절대 금지 사항 (Hard Rules)

다음은 어떤 상황에서도 **절대 위반 금지**:

```
❌ 직접 전도 메시지 (앱 내)
❌ 성경 구절 도배
❌ 검증되지 않은 정보 게시
❌ 사용자 데이터 판매
❌ 광고를 큐레이션처럼 위장
❌ "한국 최고" 민족주의 메시지
❌ 일요일 상업 활동
❌ 교회 자금과 LLC 자금 혼용
❌ 이단 단체 등재
❌ AI 추측 데이터 검증 없이 게시
```
