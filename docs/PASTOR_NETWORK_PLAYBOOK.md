## 1단계 — 도시 대표 목사님 선정

### 선정 기준 (6가지)
```
① 복음적 — 오직 성경·오직 믿음·오직 은혜
② 순수한 동기 — 교회 성장 수치 아닌 영혼이 목적
③ 제자 만드는 열정 — 삶공부·목장 등 체계적 사역
④ 헌신 — 계절적이 아닌 지속적 사역
⑤ 이민자 환영 — 새 교인의 필요를 아는 교회
⑥ 협력 마인드 — 다른 교회를 경쟁이 아닌 동역으로
```

### 도시별 첫 번째 목사님 찾는 방법
```
경로 1: 폴 김 목사님 개인 네트워크 (신학교 동기·선후배)
경로 2: SBC 한인 목회자 네트워크
경로 3: 가정교회 IHM 네트워크
경로 4: 해외 도시 → KWMA 선교사 연락망
경로 5: 지인 소개 (현재 교인 중 타 도시 아는 목사님)
```

```

  - 다음 분기 미팅 날짜 확정
  - HebronGuide 업데이트 예정 사항
  - 연락 채널 확인 (카카오 단톡방 개설)
```

## 자주 묻는 질문 (FAQ)

**Q: 허브교회가 되면 돈이 드나요?**
```
A: 전혀 없습니다. 완전 무료입니다.
   오히려 새가족을 먼저 연결받는 혜택이 있습니다.
```

**Q: 우리 교단이 아닌 교회와도 협력하나요?**
```
A: 네. 교단보다 복음과 제자 삼는 헌신이 기준입니다.
   SBC·합동·통합·고신·침례·순복음 모두 환영합니다.
   단, 이단·사이비는 절대 불가합니다.
```

**Q: 우리 교회가 작아도 되나요?**
```
A: 오히려 작은 교회가 새가족을 더 잘 돌봅니다.
   교인 수보다 목회자의 마음이 중요합니다.
```

**Q: 이미 다른 네트워크에 있어도 되나요?**
```
A: 물론입니다. HebronGuide는 배타적이지 않습니다.
   여러 네트워크가 함께 이민자를 섬길수록 좋습니다.
```
