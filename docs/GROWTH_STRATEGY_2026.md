## 🗓️ 12개월 실행 로드맵

### Phase 1: 씨앗 (1-2개월) — 3개 도시 깊게 파고들기

> **원칙**: 67개 도시에 얕게 퍼지지 말고, LA·뉴욕·시애틀 3개 도시를 완벽하게 소유한다.

**Week 1-2: 기반 구축**
- [ ] 헤브론 파트너 교회 신청 랜딩페이지 완성
- [ ] 카카오톡 공유 버튼 앱에 추가 (SDK 연동)
- [ ] 네이버 서치어드바이저 등록 + 사이트맵 제출
- [ ] 네이버 블로그 개설 (blog.naver.com/hebronguide)

**Week 3-4: 교회 첫 접촉**
- [ ] 각 도시 5명 목사님께 직접 연락 (총 15명)
- [ ] 주보 인서트 디자인 완성 (한국어, QR코드 포함)
- [ ] "헤브론 파트너 교회" 신청서 + 혜택 정리

**Week 5-8: 씨앗 심기**
- [ ] 도시별 카카오톡 오픈채팅 채널 개설 (시애틀/LA/뉴욕)
- [ ] 첫 파트너 교회 온보딩 (각 도시 3-5개 교회)
- [ ] 한국인 마트 (H-Mart 등) 커뮤니티 보드 QR 카드 설치
- [ ] 나 시애틀/LA/뉴욕 산다 Facebook 그룹에 소개 게시물

**Month 2 목표: 400-600 MAU**

## 📱 즉시 구현할 코드 기능들

### 1. 카카오톡 공유 SDK 버튼
앱 내 "친구에게 공유하기" 버튼 → 카카오톡 리치 카드 공유  
(도시 이름, 설명, 링크, 아이콘 포함)

### 2. PWA 설치 가이드 모달
모바일 첫 방문자에게 한국어 설치 안내  
"홈화면에 추가하면 앱처럼 쓸 수 있어요 📱"  
iOS/Android 각각 맞춤 가이드 (3단계 스크린샷)

### 3. 바이럴 공유 섹션
홈 화면에 "이 앱을 도움이 필요한 분께 공유해 주세요" 카드  
KakaoTalk 버튼 + 링크 복사 버튼

### 4. 신규 도착자 웰컴 페이지
/[city]/?new=true 또는 /[city]/welcome  
"방금 도착하셨나요?" → 즉시 쓸 수 있는 TOP 5 필수 정보

## 💡 핵심 인사이트

> **한인 디아스포라는 분산되어 있지만 연결되어 있다.**  
> MissyUSA(720만 방문), HeyKorean(47만), WorkingUS(34만)이 있지만  
> **67개 도시를 하나의 앱으로 묶어 정착·교회·커뮤니티를 통합한 플랫폼은 HebronGuide가 유일하다.**

경쟁자들의 약점:
- MissyUSA — 커뮤니티 게시판 (구조화된 정착 가이드 없음)
- HeyKorean — 구인구직 중심 (교회·정착 없음)  
- RadioKorea — 업체 디렉터리 (LA 중심, 신앙 레이어 없음)
- 나 산다 Facebook — 토론 그룹 (검증된 정보 없음)

**HebronGuide의 해자**: 67개 도시 × 검증된 정보 × 교회 신뢰 레이어 × 무료 PWA
