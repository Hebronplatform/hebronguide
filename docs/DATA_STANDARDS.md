# HebronGuide 데이터 표준 (Data Standards)
> 핵심 원칙 1: **정확도 최우선** — 추정·가정 데이터는 반드시 출처와 함께 명시
> 핵심 원칙 2: **최신 통계 우선** — 항상 가장 최근에 발표된 공식 통계를 사용
> 작성: 2026-05-28 | 최신 통계 업데이트: 2026-05-28 (2025 재외동포현황 반영)

---

## ⚠️ 핵심 원칙 (절대 원칙)

> **"검증되지 않은 데이터는 넣지 않는다. 틀린 정보가 없는 정보보다 나쁘다."**

> **"항상 가장 최신 통계를 사용한다. 구버전 수치는 즉시 교체한다."**

### 최신 통계 우선 규칙
- 새로운 공식 통계 발표 → **즉시** 이 문서와 관련 파일 업데이트
- 재외동포현황: 홀수년 발표 (2023→2025→2027...)
- 현재 최신본: **2025 재외동포현황** (2024년 12월 31일 기준, 2026년 3월 발표)
- 다음 업데이트 예정: **2027년** (2026년 12월 31일 기준)

모든 데이터 항목은 다음 3단계 중 하나로 표시:
- ✅ **검증됨** — 공식 출처 확인
- ⚠️ **추정** — 간접 자료 기반, 출처 명시 필수
- ❌ **미확인** — 표시하지 않거나 "정보 없음" 표기

## 2. 교회 데이터 표준

### 2.1 교회 등록 필수 필드

| 필드 | 설명 | 검증 방법 |
|------|------|-----------|
| name | 교회명 (한국어) | 웹사이트 또는 KCMUSA 확인 |
| name_en | 교회명 (영어) | 웹사이트 확인 |
| city_slug | HebronGuide 도시 코드 | 주소 기반 매핑 |
| denomination | 교단 | 웹사이트 또는 직접 확인 |
| pastor | 담임목사 | 웹사이트 확인 |
| phone | 전화 | 국가 코드 포함 |
| email | 이메일 | 유효성 확인 |
| website | 웹사이트 | 접속 가능 여부 확인 |
| source | 등록 출처 | kcmusa·cksbca·직접신청 등 |

### 2.2 교회 신학 검증 기준 (파트너 신청 시)

**자동 승인 기준 (모두 충족 시):**
- [ ] 삼위일체 신앙 명시 또는 추정 가능
- [ ] 성경의 권위 인정
- [ ] KCMUSA·KAPC·KSBC·침례회·감리회 등 공인 교단 소속 또는 독립교회
- [ ] 이단 목록 미포함 (한국기독교이단상담소·현대종교 기준)
- [ ] 활성 웹사이트 또는 연락 가능

**추가 검토 필요 (자동 검색 후 관리자 확인):**
- 신천지·JMS·하나님의교회·통일교·안상홍·구원파 등 이단 키워드
- 설립 2년 미만 신생 교회
- 담임목사 신학교 이력 불명확

**자동 검색 도구:**
1. 한국기독교이단상담소 (https://hana.pe.kr)
2. 현대종교 이단 목록
3. KCMUSA 교회 디렉토리 검색
4. 웹사이트 신앙고백 확인

## 3. 도시 콘텐츠 데이터 표준

### 3.1 도시별 인구 표시 기준

```
표시 형식: "X만+" (예: "33만+")
근거: 외교부 재외동포현황 또는 현지 센서스
정확도 표시: 출처 연도 명시 (예: "2023 외교부 기준")
업데이트 주기: 외교부 통계 갱신 시마다 (2년)
```

### 3.2 정착 정보 데이터 기준

| 항목 | 검증 주기 | 출처 우선순위 |
|------|-----------|--------------|
| 비자·체류 정보 | 연 1회 | 해당국 이민국 공식 사이트 |
| 건강보험 | 연 1회 | 보험 공식 사이트 |
| 은행 정보 | 2년 | 각 은행 공식 사이트 |
| 학교·교육 | 연 1회 | 학교구 공식 사이트 |
| 생활비 | 연 1회 | Numbeo, 현지 한인 검증 |

### 3.3 맛집·업소 데이터 기준

**포함 기준:**
- 구글 별점 4.0 이상 (단, 300건 미만은 ⚠️ 표시)
- 현재 영업 중 확인
- 주류 위주 업소 제외 (기독교 신앙 원칙)

**제외 기준:**
- 주점·바·나이트클럽 (주류가 주목적인 업소)
- 폐업·이전 확인된 업소
- 도박·성인 관련 업소
