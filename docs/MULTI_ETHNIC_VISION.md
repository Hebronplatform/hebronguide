# HebronGuide 다민족 확장 비전
## 글로벌 환대 커뮤니티 — 각 민족을 위한 플랫폼

> **"각 민족 단위로 만들어 주면 되잖아. 월드 릴리프와 이야기 해서..."**  
> — 폴 김 목사 (2026-05-18)

---

## 🌍 비전 선포

HebronGuide는 한인 디아스포라를 위한 플랫폼으로 시작했지만,  
궁극적인 비전은 **모든 민족을 위한 글로벌 환대 플랫폼**입니다.

```
현재 (Phase 1)         →    미래 (Phase 2+)
───────────────────         ──────────────────────────
한인 디아스포라 가이드        각 민족 단위 가이드

hebronguide.com/seattle      hebronguide.com/seattle (Korean)
                             hebronguide.com/seattle/zh (Chinese)
                             hebronguide.com/seattle/vi (Vietnamese)
                             hebronguide.com/seattle/es (Spanish)
                             hebronguide.com/seattle/ar (Arabic)
```

---

## 📖 신학적 근거

> **"모든 민족을 제자로 삼으라"** — 마태복음 28:19

한인 이민자를 섬기는 것이 시작이지만,  
하나님의 나라는 모든 민족을 향합니다.

한인 디아스포라가 정착하여 뿌리내린 후,  
**그 도시에서 다른 민족들을 섬기는 선교 주역**이 되는 것 —  
이것이 HebronGuide 글로벌 환대 커뮤니티의 완성입니다.

---

## 🤝 월드 릴리프 (World Relief) 파트너십 방향

### World Relief란?
- 1944년 설립, 22개국 운영
- 연간 880만명 난민·이민자 지원
- 미국 내 사무소: 시애틀·시카고·볼티모어·내쉬빌·콜롬비아·트라이앵글

### 시너지 포인트
| HebronGuide | World Relief |
|---|---|
| 디지털 정착 플랫폼 | 현장 정착 지원 인프라 |
| 한인 교회 네트워크 | 다민족 교회 파트너십 |
| 67개 도시 콘텐츠 | 22개국 현지 서비스 |
| 기술·앱 플랫폼 | 인력·관계 네트워크 |

### 제안할 협력 모델
1. **World Relief 클라이언트 → HebronGuide 연결**  
   World Relief가 돕는 난민/이민자에게 도시별 HebronGuide 제공

2. **다민족 가이드 공동 제작**  
   World Relief의 민족별 커뮤니티 전문성 + HebronGuide의 플랫폼

3. **교회 파트너십 공유**  
   HebronGuide 파트너 교회 ↔ World Relief 협력 교회 연결

## 🗺️ 다민족 확장 로드맵

### Phase 1 (현재): 한인 디아스포라
- 67개 도시, 730만 재외 한인 대상
- 한국어 + 영어

### Phase 2 (2027-2028): 아시아 디아스포라 확장
우선순위 민족 (미국 내 인구 + 이민 규모 기준):
1. **중국계** — 미국 내 530만명, 가장 큰 아시아계
2. **인도계** — 460만명, IT 이민자 급증
3. **필리핀계** — 460만명, 의료·군인 가족 다수
4. **베트남계** — 230만명, 강한 커뮤니티 의식

### Phase 3 (2029-2030): 글로벌 전 민족
- 스페인어권 (미국 내 6,200만명)
- 아랍어권 (중동·북아프리카 이민자)
- 아프리카계 (사하라 이남 이민자)

## 💡 "글로벌 환대 커뮤니티" — 태그라인 의미

**기존:** `#1 한인 정착 가이드` → `#1 글로벌 한인 가이드`  
**현재:** `글로벌 환대 커뮤니티`

왜 "환대(Hospitality)"인가:
- 마태복음 25:35 — "내가 나그네 되었을 때 너희가 영접하였다"
- Hebron의 히브리어 의미 = "연합·교제·동반자"
- "한인"에서 "모든 민족"으로 가는 다리

왜 "커뮤니티(Community)"인가:
- 정보 제공(가이드)을 넘어 공동체를 이룸
- 신약교회 회복의 핵심 = 공동체
- World Relief와 연결될 때, 공동체가 플랫폼
