# HebronGuide 운영 원칙
**Operating Principles — Chick-fil-A 모델 적용**  
작성: 2026-05-05

---

## 왜 Chick-fil-A인가

> Chick-fil-A는 치킨 회사가 아니다.  
> **크리스천 가치로 운영되는 서비스 기업**이다.

설립자 S. Truett Cathy의 선언:
> *"We didn't set out to be a Christian company.  
> We set out to do right — and that reflected our Christian values."*

우리는 "한인 정보 앱"을 만들려는 게 아니다.  
**옳은 일을 하려 한다 — 그것이 크리스천 가치의 표현이다.**

## Chick-fil-A 7대 원칙 → HebronGuide 적용

---

### 원칙 1. 안식일을 지킨다 (Closed on Sunday)

**Chick-fil-A:**
```
연간 $2 billion 매출 포기
매주 일요일 전 매장 휴무
"쉬는 날이 있어야 더 잘 섬길 수 있다"
결과: 주 6일 매출이 경쟁사 주 7일보다 높음
```

**HebronGuide 적용:**
```
일요일 원칙:
  ✝️ 운영팀 안식 — 일요일 업무·배포 없음
  ✝️ 상업적 SNS 금지 — 광고·홍보 콘텐츠 없음
  ✝️ 커뮤니티 중심 — 올린다면 예배·묵상·감사 내용만
  ✝️ 큐레이터 안식 — 일요일 게시 의무 없음

역설:
  일요일을 비워둠으로써 오히려
  HebronGuide의 정체성이 더 강해진다
  "이 앱은 다르다"는 인식
```

### 원칙 2. 하나만 탁월하게 (One Thing, Done Excellently)

**Chick-fil-A:**
```
메뉴: 치킨 샌드위치 하나에 집중
     피자·버거·스테이크 없음
결과: 그 하나로 맥도날드 단위 매장 매출 2배
```

**HebronGuide 적용:**
```
우리가 하는 한 가지:
  "한인 이민자·이주자·방문자의 정착과 연결"

이것 하나를 세계 최고로 한다

하지 않는 것들:
  ✗ 일반 여행 앱 (Airbnb·TripAdvisor와 경쟁 안 함)
  ✗ 일반 배달 앱 (배달의민족과 경쟁 안 함)
  ✗ 일반 SNS (인스타·틱톡과 경쟁 안 함)
  ✗ 한국 내 서비스 (재외 한인 700만에 집중)

"이민자를 위한 것이라면" — Yes
"일반인을 위한 것이라면" — No
```

### 원칙 3. "My Pleasure" 문화 — 진심 서비스

**Chick-fil-A:**
```
직원 교육 1순위: "My pleasure"
"Thank you" → "My pleasure" (You're welcome 아님)
"기꺼이 섬깁니다"의 자세

결과: 고객 서비스 만족도 10년 연속 1위
```

**HebronGuide 적용:**
```
모든 사용자 접점에서의 태도:

앱 사용:
  정보를 찾는 것 → "찾아주셔서 감사합니다"의 자세로 설계
  오류 메시지 → 비난 없이 따뜻하게
  
SNS 댓글:
  불만 → 즉시 인정, 개선 약속
  감사 → "기꺼이 섬깁니다" (My pleasure)

큐레이터 훈련:
  "정보를 파는 것"이 아니라
  "이웃을 섬기는 것"의 마음으로

언어 원칙:
  ✗ "저희 서비스를 이용해주세요"
  ✅ "도움이 되었으면 좋겠습니다"
  ✗ "프리미엄 회원이 되세요"
  ✅ "더 잘 섬길 수 있도록 준비했습니다"
```

### 원칙 4. 전도하지 않는다 — 삶으로 보여준다

**Chick-fil-A:**
```
컵에 성경 구절 없음
음식 포장에 십자가 없음
계산대에서 전도 없음

대신:
  정직한 재료
  공정한 직원 대우
  지역사회 투자
  일관된 품질

결과: 비기독교인도 "Chick-fil-A는 뭔가 다르다" 느낌
```

**HebronGuide 적용:**
```
앱에서 하지 않는 것:
  ✗ 성경 구절 화면 배너
  ✗ "예수님을 믿으세요" 팝업
  ✗ 교회 광고처럼 보이는 교회 탭
  ✗ 노골적 전도 메시지

앱에서 하는 것:
  ✅ 정직한 정보 (틀리면 즉시 수정)
  ✅ 검증된 데이터만 (환각 정보 없음)
  ✅ 이민자 입장에서 설계된 UX
  ✅ 교회 탭은 "정보"로, 강요 없이

결과:
  비기독교인도 "이 앱은 뭔가 다르다" 느낌
  → 신뢰 → 자연스러운 교회 탭 클릭
  → 히든 미션 완성
```

### 원칙 5. 사람을 선발한다 — 아무나 받지 않는다

**Chick-fil-A:**
```
가맹점 선발 경쟁률: 60,000:1 (하버드보다 낮은 합격률)
선발 기준: 사업 능력 + 인성 + 가치관 일치
초기 비용: $10,000 (업계 최저)
이유: "돈이 많은 사람"이 아니라
     "올바른 사람"에게 맡긴다
```

**HebronGuide 적용:**
```
큐레이터 선발 원칙:
  ✗ SNS 팔로워 많은 사람
  ✗ 마케팅 경험자
  ✅ 해당 도시를 진심으로 사랑하는 사람
  ✅ 이민자의 어려움을 직접 경험한 사람
  ✅ 담임목사가 인격을 보증하는 사람
  ✅ 돈보다 섬김의 동기가 큰 사람

파트너 교회 선발 원칙:
  ✗ 큰 교회
  ✗ 유명 목사
  ✅ HebronGuide 정신을 이해하는 교회
  ✅ 이민자 사역에 진심인 교회
  ✅ 재정 투명성이 있는 교회

비즈니스 등재 원칙:
  ✗ 광고비 낸다고 다 등재
  ✅ 검증 기준 통과한 곳만 등재
  ✅ "✅ 검증됨" 배지는 진짜 검증된 곳만
```

### 원칙 6. 상장하지 않는다 — 장기 관점

**Chick-fil-A:**
```
비상장 가족 기업 유지 (Cathy 가문 100% 소유)
이유: 주주 압력으로 가치관 타협하지 않기 위해
분기 실적보다 100년 유산을 본다
결과: 미국 내 단위 매장 매출 1위
```

**HebronGuide 적용:**
```
단기 수익보다 장기 신뢰:
  ✗ 빠른 성장을 위해 미검증 정보 게시
  ✗ 수익을 위해 광고주 압력에 굴복
  ✗ 사용자 데이터 판매
  ✗ VC 투자 받아 가치관 희석

  ✅ 느려도 검증된 정보만
  ✅ 광고주가 아닌 사용자에게 충성
  ✅ 개인정보 보호 최우선
  ✅ 필요하면 독립 비영리 유지

목표:
  "2년 후 매각" ❌
  "20년 후에도 이민자가 찾는 플랫폼" ✅
```

### 원칙 7. 지역 사회에 투자한다

**Chick-fil-A:**
```
연간 장학금 $19 million (직원 자녀)
지역 푸드뱅크 기부
재난 발생 시 무료 식사 제공
이유: "우리가 뿌리 내린 곳을 돌본다"
```

**HebronGuide 적용:**
```
수익의 일부를 이민자 커뮤니티에:

  ① 무료 정보 (기본 기능 영구 무료)
     → 돈 없는 이민자도 똑같이 접근

  ② 이민자 법률 무료 연결
     → 수익 일부로 무료 법률 상담 지원

  ③ 새 도시 런치 = 그 도시 한인 커뮤니티에 선물
     → 수익 이전에 정보 먼저 제공

  ④ 연간 감사 보고서 공개
     → "우리 수익이 어디 쓰였나" 투명 공시
```

## 운영 문화 — 일상에서 원칙이 보여야 한다

```
월요일: 한 주 시작 기도 (팀)
화-토: 집중 운영 + 섬김
일요일: 안식 (팀 전체)

모든 콘텐츠 게시 전 질문:
  "이것이 이민자에게 진짜 도움이 되는가?"
  "이것이 하나님 앞에 부끄럽지 않은가?"
  YES → 게시
  NO  → 보류

모든 비즈니스 결정 전 질문:
  "이것이 사용자를 위한 것인가,
   수익을 위한 것인가?"
  사용자 우선 → 진행
  수익 우선   → 재검토
```
