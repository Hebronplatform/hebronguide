## 📝 문서 작성·갱신 규칙

### 새 문서 작성 시

```
1. 문서 시작에 출처·작성일 명시
2. 비전(`CORE_VISION.md`)과의 연결 명시
3. EXECUTION_GUIDELINES.md 어느 영역인지 매핑
4. 변경 이력 섹션 포함
5. 이 README.md 인덱스에 추가
```

### 기존 문서 갱신 시

```
1. 변경 이력 섹션에 날짜·요약 추가
2. CORE_VISION 또는 EXECUTION_GUIDELINES 영향 확인
3. CLAUDE.md 동기화 필요 여부 확인
```
