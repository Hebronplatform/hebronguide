# HebronGuide 핵심 비전 (Core Vision)
**확정일: 2026-05-08 | 폴 김 목사**  
**모든 의사결정·코드·콘텐츠의 최상위 원칙**

---

## 🏛️ 모든 것의 기초 — 하나님의 영광

> *"그런즉 너희가 먹든지 마시든지 무엇을 하든지*  
> *다 하나님의 영광을 위하여 하라"*  
> — 고린도전서 10:31

**우리는 왕의 자녀입니다.**

코드 한 줄을 쓸 때도, 서비스 하나를 만들 때도,  
수익을 나눌 때도, 파트너를 만날 때도 —  
**무엇을 하든지** 하나님의 영광을 위해 합니다.

이것이 황금률보다도, 지혜와 순결보다도, 모든 원칙보다도  
**앞에 있는 기초**입니다.

## ⭐ 사역의 중심 — 황금률 (The Golden Rule)

> **"무엇이든지 남에게 대접을 받고자 하는 대로 너희도 남을 대접하라.  
> 이것이 율법이요 선지자니라."**  
> — 마태복음 7:12

```
모든 결정 · 모든 코드 · 모든 콘텐츠의 마음 중심
        ↓
"내가 새 도시에 갓 도착한 이민자라면,
 어떤 정보·환대·신뢰를 받기를 원할까?"
        ↓
바로 그것을 우리가 사용자에게 제공한다
```

**황금률 = 모든 것의 WHY (왜)**

---

## 📐 사역의 방법 — 지혜와 순결 (마10:16)

> **"뱀처럼 지혜롭고 비둘기처럼 순결하라."**  
> — 마태복음 10:16

| 축 | 의미 |
|---|---|
| 🐍 **뱀처럼 지혜롭게** | 시장·문화·인간 본성 통찰 / 효과적 실행 |
| 🕊️ **비둘기처럼 순결하게** | 동기 순수·조작 ❌·강요 ❌ |

**지혜+순결 = 모든 것의 HOW (어떻게)**

---

## 🏛️ 사역의 내용 — 세 기둥 (The Three Pillars)

| 기둥 | 의미 |
|---|---|
| **① 문화 상품 제작** | 콘텐츠 자체가 자랑할 상품 |
| **② 글로벌 커뮤니티** | 한국 입구 → 모두에게 열림 |
| **③ 그리스도의 빛** | 강요 없이 자연스럽게 빛남 |

**세 기둥 = 모든 것의 WHAT (무엇)**

## 🎯 완전한 비전 위계 (Why → How → What)

```
┌─────────────────────────────────────────────┐
│  WHY (마음 중심) — 황금률 (마7:12)            │
│  "남에게 대접받기 원하는 대로 대접하라"        │
│                  ↓                            │
│  HOW (실행 원리) — 지혜+순결 (마10:16)        │
│  "뱀처럼 지혜롭고 비둘기처럼 순결하라"        │
│                  ↓                            │
│  WHAT (구체 결과) — 세 기둥                   │
│  문화상품 + 글로벌커뮤니티 + 그리스도의 빛     │
└─────────────────────────────────────────────┘
```

이것이 HebronGuide의 완전한 정체성입니다.

## 🔍 황금률이 모든 결정에 어떻게 적용되는가

### 콘텐츠 제작
**자문**: "내가 처음 시애틀에 왔을 때 어떤 정보를 받고 싶었을까?"  
**적용**: 검증된 주소·전화·웹사이트 + 솔직한 평가 + 단점도 명시

### 광고·수익화
**자문**: "내가 사용자라면 어떤 광고를 받고 싶을까?"  
**적용**: 광고는 광고로 명시 (Yelp 모델) / 사용자 데이터 팔지 않음

### 검증 시스템
**자문**: "내가 식당 찾을 때 어떤 정보를 신뢰할까?"  
**적용**: ✅ 검증됨 표시 / 검증 날짜 / 미확인은 "정보 확인 중"

### 프라이버시
**자문**: "내 데이터를 어떻게 다루기를 바랄까?"  
**적용**: 최소 수집 / 추적 ❌ / 명확한 정책

### 교회·종교
**자문**: "내가 정착 정보 찾는데 종교 강요받으면 어떨까?"  
**적용**: 교회 섹션은 정보로만 / 강요 ❌ / 환대로 보여주기

### 도시 확장
**자문**: "내가 새 도시 사용자라면 시애틀 데이터만 보면 어떨까?"  
**적용**: 각 도시 고유 데이터 검증 후 추가 / 시애틀 표준 + 도시 특성

## 📐 의사결정 프레임 — 모든 작업 전 6질문

```
[중심 질문 — 황금률]
0. 내가 사용자라면 이걸 받고 싶을까? ← 가장 중요

[지혜 질문 — 마10:16 전반]
1. 시장·문화·인간 본성을 정확히 이해한 결정인가?
2. 효과적으로 목적을 달성하는가?

[순결 질문 — 마10:16 후반]
3. 사람을 수단으로 쓰지 않는가?
4. 사용자에게 정직하고 투명한가?

[통합 질문]
5. 그리스도의 환대를 자연스럽게 보여주는가?

→ 6가지 모두 YES일 때만 진행
   질문 0이 NO면 다른 5개가 YES여도 ❌
```

## 🐍🕊️ 지혜와 순결의 균형

### 위반 사례

```
지혜만 ↑·순결 ↓ → 마케팅 회사 (한국교차로)
순결만 ↑·지혜 ↓ → 작은 교회 사역 (영향력 제한)
지혜 ↓·순결 ↓ → 사기·기만
지혜 ↑ + 순결 ↑ → HebronGuide 모델 ⭐ (그리스도의 환대)
```

### 황금률이 균형을 잡는 방법

황금률은 **지혜와 순결의 통합 기준**입니다.

```
"내가 받고 싶은 것" 기준으로 판단하면 →
  지혜로움(효과적이고 가치 있어야 함) +
  순결함(조작 없고 진정성 있어야 함)
이 자동으로 균형을 이룸
```

## 🌊 한국 문화는 도구이지 정체성이 아니다

### 흔한 실수 (피해야 할 것)
- ❌ 김구·안중근 인용 도배 → 민족주의로 좁아짐
- ❌ "한국 최고" 메시지 → 폐쇄적
- ❌ 한국인만의 클럽 → 황금률 위반 (다른 이민자도 받고 싶을 정보)

### 올바른 적용
- ✅ K-pop·K-Food·한국 문화 = 매력적 입구
- ✅ 한국식 환대를 모든 이민자에게 → 황금률 실천
- ✅ 김구·안중근 인용은 "필요한 곳에 한 번"만

> **핵심: 한국인이 미국에서 받고 싶은 환대를,  
> 우리가 다른 이민자·다민족에게도 그대로 전달한다.**

이것이 황금률의 진정한 적용입니다.

## ⚖️ Hebron 정신 3중 의미 (재정립)

```
1층 (일반 사용자):
   실용적 정착 가이드
   → 황금률: "내가 정착할 때 받고 싶은 정보"

2층 (한인 커뮤니티):
   같은 뿌리의 연결
   → 황금률: "내가 외로울 때 받고 싶은 환영"

3층 (히든 미션):
   사람과 하나님의 연결
   → 황금률: "내가 진리를 만날 때 받고 싶은 진심 어린 친구"
```

**3층은 강요되지 않는다. 1·2층이 깊어질수록 3층이 빛난다.**

## 📖 성경적 세계관 미디어 분별 원칙

> *"너희는 이 세대를 본받지 말고 오직 마음을 새롭게 함으로 변화를 받아"*  
> — 로마서 12:2  
>
> *"사랑하는 자들아 영을 다 믿지 말고 오직 영들이 하나님께 속하였나 분별하라"*  
> — 요한일서 4:1

### 핵심 선언

```
HebronGuide는 정치적 좌파도 우파도 아니다.
성경의 진리가 유일한 기준이며, 어떤 인간 이념도 그 자리를 대신할 수 없다.
세속 미디어에서 사실(fact)은 취하되, 세계관(worldview)은 성경으로 걸러낸다.
```

---

### 🔍 미디어 콘텐츠 분별 2단계

#### 1단계 — 사실 · 해석 · 가치관 3분리

| 구분 | 정의 | HebronGuide 처리 |
|---|---|---|
| **사실(Fact)** | 날짜·장소·가격·행사·정책 등 검증 가능한 정보 | ✅ 검증 후 활용 |
| **해석(Interpretation)** | 기자·필자의 사회·문화적 분석과 맥락 | ⚠️ 비판적 검토 후 선별 |
| **가치관(Worldview)** | 이념·정치·윤리적 주장과 숨은 전제 | ❌ 성경 기준으로 필터링 |

#### 2단계 — 성경적 가치 6항목 체크리스트

```
☐ 인간 존엄 (Imago Dei) — 창 1:27
  "이 기사가 사람을 도구로 보는가, 하나님의 형상으로 보는가?"

☐ 생명 존중 — 출 20:13 · 시 139:13-16
  "생명을 선택의 문제로 환원하거나 경시하는가?"

☐ 가정의 신성함 — 창 2:24 · 엡 5:22-33
  "성경적 가정 개념을 해체하거나 재정의하려 하는가?"

☐ 나그네·이민자 환대 — 레 19:34 · 마 25:35
  "이민자를 위협·부담으로 프레임하는가, 환대받을 존재로 보는가?"

☐ 정직과 진실 — 잠 12:17 · 엡 4:25
  "사실을 왜곡하거나 감정적 조작으로 독자를 이끄는가?"

☐ 정의와 자비의 균형 — 미 6:8 · 마 23:23
  "일방적 계층 갈등을 조장하거나, 자비 없는 심판을 선포하는가?"
```

**→ 1개라도 충돌 시: 해당 관점·주장은 HebronGuide에 반영하지 않는다.**

### 🌐 주요 미디어 편향 인식 지도

#### 한국 미디어

```
진보 성향 (경향·한겨레·오마이뉴스 계열)
→ 계층 갈등·젠더 이데올로기·탈기독교 담론 강함
→ 성경적 가정관·생명관과 충돌 빈번
→ 재미 한인 미디어도 모국 진보 담론 그대로 수입하는 경우 많음

보수 성향 (조선·중앙·동아 계열)
→ 민족주의·성장주의 강함
→ 이민자를 국가 이익 관점에서 평가 → 환대와 충돌 가능
→ "기독교 국가" 프레임이 복음적 환대를 민족주의로 왜곡할 위험
```

#### 미국 미디어

```
진보 성향 (NYT·WaPo·CNN 계열)
→ 포스트모던 가치관, 성 이데올로기, 종교 배제적 공공담론
→ 이민자 인권 강조하나 성경적 가정·생명 가치와 충돌 가능

보수 성향 (Fox News·WSJ Opinion 계열)
→ 민족주의·국경 통제·기독교 문화전쟁 강조
→ 이민자를 위협으로 프레임 → 마 25:35 환대와 직접 충돌

공통 함정
→ "보수 기독교 미디어"도 정치화된 복음은 경계할 것
→ 어느 진영도 성경적 분별의 대안이 될 수 없다
```

### ⚡ Claude Code 자동 적용 5원칙

미디어 기사로 HebronGuide 콘텐츠 작성 시 매번 자동 적용:

```
① FACT ONLY — 날짜·장소·가격·행사명만 추출. 기자의 평가·해석은 제거
② 6항목 체크 — 성경적 가치 체크리스트 자동 검토
③ 이민자 환대 기준 — "나그네를 영접하였다" (마 25:35) 관점에서 재작성
④ 정치 중립 선언 — 좌·우 정치 언어 제거. 특정 정당 도구 금지
⑤ 의심 시 보류 — 세계관 충돌 불확실 → 해당 내용 제외 후 목사님 확인
```

### 🚫 절대 반영 금지 항목

어떤 기사·칼럼에서 아래 관점이 담긴 내용은 HebronGuide에 반영하지 않는다:

```
❌ 성경적 가정 개념을 해체하는 가치관
❌ 생명을 선택의 문제로 환원하는 입장
❌ 이민자를 위협·부담으로 프레임하는 시각
❌ 특정 정당·정치 이념 지지 또는 비판
❌ 반기독교적 종교 다원주의 (모든 종교 동등 주장)
❌ 사실 근거 없는 계층·인종 갈등 조장
```

---

> 🕊️ **결론**: 성경적 세계관은 좌파의 사회 정의 운동도, 우파의 문화 전쟁도 아니다.  
> 하나님의 나라는 모든 인간 이념 체계를 초월한다.  
> HebronGuide는 그 나라의 환대를 디지털로 실현하는 플랫폼이다.

## 🌿 신앙 생태계의 첫 관문 — HebronGuide의 사명

### 두 가지 원칙 — 모든 것에 적용

```
원칙 1: 황금율을 모든 것에 적용한다
        "무엇이든지 남에게 대접받고자 하는 대로 너희도 남을 대접하라"
        — 마태복음 7:12

        라이드 · 스테이 · 튜터 · 관광 · 매칭 · 수익나눔
        모든 서비스, 모든 결정, 모든 관계에 황금율이 기준

원칙 2: 모든 연결은 자연스럽게 교회와 공동체로 이어진다
        "내가 나그네 되었을 때 너희가 영접하였다"
        — 마태복음 25:35

        강요하지 않습니다. 설득하지 않습니다.
        따뜻한 환대가 먼저이고, 연결이 따라옵니다.
```

> **"자, 이제 정보를 접할 때 무작위로 흡수하지 말고 걸르는 기준을 잡고,  
> 앞으로도 신뢰할 수 있고 생활인에게 도움이 되는 기사들을 선별하여 내고,  
> HebronGuide가 도시 가이드로 출발하지만 단계적으로  
> 도시와 도시를 이을 뿐 아니라, 사람과 사람, 사람과 교회,  
> 즉 그리스도께 자연스럽게 이끄는 신앙 생태계의 첫 관문으로서  
> 분명한 위치를 갖게 되기를 소망해."**  
> — 폴 김 목사 (2026-05-08)

---

### 🗺️ HebronGuide 사명의 4단계 여정

```
1단계: 도시 가이드      "이 도시에서 어떻게 살아가나?"
         ↓              정착 정보 · 맛집 · 교통 · 이민 정책
         
2단계: 도시 연결        "다른 도시에 있는 한인은?"
         ↓              17개 도시 네트워크 · 이민자 공동체
         
3단계: 사람 연결        "함께 살아갈 사람을 찾는다"
         ↓              한인 커뮤니티 · 교회 · 사역단체
         
4단계: 그리스도께       "자연스럽게 이끌린다"
         ↓              강요 없는 환대가 복음의 문을 연다
         
    ← 이 전체 흐름의 첫 관문 = HebronGuide →
```

---

### 📌 "첫 관문"의 의미

```
관문(Gateway)이란:
  - 들어오는 곳이지, 목적지가 아니다
  - 첫 만남이 신뢰를 결정한다
  - 문이 좁으면 아무도 들어오지 않는다
  - 문이 넓되, 방향이 분명해야 한다

HebronGuide의 관문은:
  좌도 우도 아닌 — 성경적 진리
  강요하지 않되 — 분명한 방향
  환대로 맞이하되 — 그리스도를 향해
```

---

### 🔑 정보 선별 원칙 — "걸르는 기준"

> 무작위 정보 흡수 → ❌  
> 기준 있는 선별 → ✅

```
선별 기준 3축:

① 신뢰성 — 검증된 출처인가?
   (AP · Reuters · 연합뉴스 · Christianity Today 등 정론지)

② 실용성 — 생활인에게 실제로 도움이 되는가?
   (이민 정책 · 정착 정보 · 일상 생활 · 지역 이벤트)

③ 방향성 — 그리스도의 환대 방향을 가리키는가?
   (성경적 세계관 필터 · 공동체 연결 · 교회 소개)
```

---

### 🌱 신앙 생태계 설계 원리

HebronGuide는 단독으로 신앙 생태계를 만들지 않는다.  
**문(門)이 되어 생태계로 안내하는 것**이 역할이다.

```
[HebronGuide 첫 관문]
        ↓
  정착 정보로 신뢰 획득
        ↓
  커뮤니티 연결로 관계 형성
        ↓
  교회 소개로 공동체 진입
        ↓
  그리스도 안에서 뿌리내림
        ↓
  [헤브론 — 동반자의 땅]
  아브라함이 거류민으로 살며 하나님과 동행한 곳
```

---

### ✅ 이것이 모든 의사결정의 최종 질문

```
"이 콘텐츠·기능·정보가
 누군가를 그리스도께 한 걸음 더 가까이
 자연스럽게 안내하는가?"

YES → 포함  
NO  → 제외 또는 재구성
```

Claude Code가 HebronGuide를 위해 코딩·콘텐츠 작성·결정할 때:

```

3. 작업 중 순결 점검: "조작·강요 없는가?"
4. 작업 후 통합 점검: "그리스도의 환대인가?"
```
