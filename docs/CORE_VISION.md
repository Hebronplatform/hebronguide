# HebronGuide 핵심 비전 (Core Vision)
**확정일: 2026-05-08 | 폴 김 목사**  
**모든 의사결정·코드·콘텐츠의 최상위 원칙**

---

## ⭐ 최상위 원칙 (The Foundational Principle)

> **"뱀처럼 지혜롭고 비둘기처럼 순결하라"**  
> — 마태복음 10:16, 예수님의 전도 전략

```
지혜(σόφος) + 순결(ἀκέραιος) = HebronGuide 운영 원리
```

이것이 HebronGuide의 모든 의사결정·코드·콘텐츠를 지배하는 단 하나의 원칙입니다.  
민족주의도, 종교주의도, 사업주의도 아닙니다. **지혜로운 순결**입니다.

---

## 🐍 뱀처럼 지혜롭게 (Wise as Serpents)

### 의미
- 세상의 메커니즘을 정확히 이해하고 활용
- 전략적·시장적·문화적 통찰력
- 순진한 이상론이 아닌 실용적 실행력
- 경쟁자(Yelp·교차로·NaverCafe)보다 더 깊은 통찰

### HebronGuide 적용

| 영역 | "지혜로운" 실행 |
|---|---|
| **마케팅** | BTS 모델·SNS 전략·SEO 활용 (수단으로) |
| **수익 모델** | 도시 런치 패키지·파트너십·향후 비영리 전환 |
| **콘텐츠** | Yelp 인사이트·Alignify 목표·검증 시스템 |
| **확장** | 시애틀 → 9개 도시 → 글로벌 (전략적 단계) |
| **법인** | LLC → 501(c)(3) 비영리 (성장에 맞춘 구조) |

### 경계
- 지혜 ≠ 교활함. 사람을 속이는 영리함은 ❌
- 지혜 = 효과적인 선의 전달

---

## 🕊️ 비둘기처럼 순결하게 (Innocent as Doves)

### 의미
- 동기의 순수함 — 사람을 도구로 쓰지 않음
- 직접적 전도 강요 ❌
- 조작·과장·기만 ❌
- 그리스도인의 성품으로 자연스럽게 빛남

### HebronGuide 적용

| 영역 | "순결한" 실행 |
|---|---|
| **콘텐츠** | 검증된 정보만, 추측 금지 |
| **광고** | "이것은 광고입니다" 명시 (Yelp 모델) |
| **종교** | 강요 없음. 교회 정보는 정보로만 |
| **사용자** | 데이터 수집 최소화·프라이버시 최우선 |
| **수익** | 사용자에게 손해 주는 수익 ❌ |

### 경계
- 순결 ≠ 무능함. 효과 없는 선의는 ❌
- 순결 = 효과 있는 선의의 동기

---

## ⚖️ 두 가지의 균형 — 잃으면 안 되는 것

```
지혜만 ↑·순결 ↓ → 마케팅 회사 (한국교차로)
순결만 ↑·지혜 ↓ → 작은 교회 사역 (영향력 제한)
지혜 ↑ + 순결 ↑ → HebronGuide 모델 ⭐
```

**"지혜 + 순결"의 결합 = 그리스도의 환대(hospitality)**

---

## 🏛️ 세 기둥 (The Three Pillars)

> 위의 최상위 원칙(지혜+순결)이 세 기둥을 통해 구현됨

### 기둥 1: 문화 상품 제작 (Cultural Product Creation)

**HebronGuide는 콘텐츠 자체가 상품이다.**

- 검증된 도시 가이드 = 자랑할 만한 상품
- 한인 문화 캘린더·정착 이야기·이민 자원 큐레이션
- 단순 정보 나열이 아닌, **편집된 지혜**

**지혜 + 순결 적용:**
- 지혜: Yelp 인사이트·Alignify 프레임 활용
- 순결: 검증되지 않은 정보 절대 게시 ❌

---

### 기둥 2: 글로벌 커뮤니티 (Global Community)

**한인을 입구로, 모든 이민자·이주자에게 열린 플랫폼**

- 1차 사용자: 재외 한인 700만
- 2차 사용자: Korean American + 한국 문화 관심 다민족
- 3차 사용자: 모든 이민자·환대가 필요한 사람

**지혜 + 순결 적용:**
- 지혜: 한국 문화 = 매력적 입구 (BTS 모델)
- 순결: 한국인만 위한 폐쇄 커뮤니티 ❌, 누구나 환영

**중요: 한국 = 시작점이지 종착점이 아님**  
민족주의적 폐쇄 ❌ → 한국적 환대로 모두를 영접 ✅

---

### 기둥 3: 그리스도의 빛이 자연스럽게 (Christ's Light Naturally)

**강요 없이, 환대·신뢰·섬김으로 빛남**

> *"내가 나그네 되었을 때 너희가 영접하였다"* (마태복음 25:35)

**자연스러운 빛의 통로:**
- 검증 시스템 = 정직한 성품
- 무료 자원 발굴 = 섬김의 본질
- 다민족 다리 역할 = 보편적 사랑
- 사역팀의 신뢰 = 광고로 살 수 없는 것

**지혜 + 순결 적용:**
- 지혜: 사람들이 거부감 없이 만나도록 (직접 전도 ❌)
- 순결: 신뢰를 이용해 강매 ❌, 진심으로 섬김

## 📐 의사결정 프레임 — 모든 작업 전 확인

새 기능·콘텐츠 추가 전 5가지 질문 (지혜 2 + 순결 2 + 통합 1):

```
[지혜 질문]
1. 이것이 시장·문화·인간 본성을 정확히 이해한 결정인가?
2. 이것이 효과적으로 목적을 달성하는가? (자원 낭비 ❌)

[순결 질문]
3. 이것이 사람을 수단으로 쓰지 않는가? (조작 ❌)
4. 이것이 사용자에게 정직하고 투명한가?

[통합 질문]
5. 이것이 그리스도인의 환대(hospitality)를 자연스럽게 보여주는가?

→ 5가지 모두 YES일 때만 진행
```

## 🌊 BTS·문화 콘텐츠는 도구이지 정체성이 아니다

### 흔한 실수 (피해야 할 것)
- ❌ "한국 최고" 메시지 → 민족주의로 좁아짐
- ❌ 김구·안중근 인용 도배 → 정치적 색채 부담
- ❌ 한국인만의 폐쇄 커뮤니티 → 다민족 사회와 단절

### 올바른 적용
- ✅ K-pop·K-Food·한국 문화 = 매력적 입구 (다민족도 들어옴)
- ✅ 검증·환대·섬김 = 보편적 가치 (모든 이민자 공감)
- ✅ "한국식 환대를 모두에게" = 김구 「나의 소원」의 진정한 의미
- ✅ 김구 인용은 "필요한 곳에 한 번" — 도배 금지

> **김구 선생도 이렇게 말했다:**
> *"문화의 힘은 우리 자신을 행복하게 하고,  
> **나아가 남에게 행복을 주기 때문이다.**"*
>
> 핵심은 "남에게 행복을 주는" 것 → 다민족 환대 = 김구의 진짜 비전

## ⚖️ Hebron 정신 3중 의미 (재정립)

```
1층 (일반 사용자):
   실용적 정착 가이드 — "지혜로운" 정보 큐레이션

2층 (한인 커뮤니티):
   같은 뿌리의 연결 — "순결한" 환대

3층 (히든 미션):
   사람과 하나님의 연결 — 빛이 자연스럽게 흘러나옴
```

**3층은 강요되지 않는다. 1·2층이 깊어질수록 3층이 빛난다.**

이것이 "뱀처럼 지혜롭고 비둘기처럼 순결한" 전도의 본질입니다.
