# HebronGuide 핵심 비전 (Core Vision)
**확정일: 2026-05-08 | 폴 김 목사**  
**모든 의사결정·코드·콘텐츠의 최상위 원칙**

---

## 🎯 진정한 정체성 — HebronGuide는 무엇인가?

### ❌ HebronGuide는 단순히 이것이 아니다
- 단순한 지역 정보 공유 앱이 아니다
- Yelp의 한국어 버전이 아니다
- 교차로의 디지털 버전이 아니다
- 단순한 정착 가이드가 아니다

### ✅ HebronGuide는 이것이다

```
┌──────────────────────────────────────────────┐
│  ① 문화 상품을 만들고 공유하는              │
│     글로벌 한인 커뮤니티 플랫폼                │
│                                              │
│  ② 700만 재외 한인의 디아스포라 자긍심을      │
│     디지털로 결집시키는 인프라                 │
│                                              │
│  ③ 그 가운데 교회와 기독교 이미지를          │
│     자연스럽게 밝게 만드는 지혜의 도구          │
└──────────────────────────────────────────────┘
```

---

## 🏛️ 세 기둥 (Three Pillars)

### 기둥 1: 문화 상품 제작 (Cultural Product Creation)

**단순 정보 큐레이션 ≠ HebronGuide**  
HebronGuide는 **콘텐츠 자체가 상품**입니다.

**문화 상품의 예시:**
- 각 도시별 한인 정착 스토리북
- Korean American 영웅 시리즈 (1세대 정착자 인터뷰)
- 한인 문화 캘린더 (광복절·한글날·추석)
- 김구·안중근 인용 시리즈
- 도시별 한인 음식 지도
- 다민족 환대 가이드 (Korean American → 다른 이민자에게)

**원칙:**
- 모든 콘텐츠는 "이걸 누가 자랑스럽게 공유할까?"를 통과해야 함
- 정보의 질은 Yelp + 한인 커뮤니티 검증 + 교회 신뢰로 3중 검증
- 콘텐츠 = 상품 = 신뢰 자산

---

### 기둥 2: 글로벌 한인 커뮤니티 (Global Korean Community)

**지역 앱이 아닌 글로벌 플랫폼**

**확산 전략:**
```
시애틀 (모델) → 미주 9개 도시 (Phase 1)
            ↓
       토론토·밴쿠버 (Phase 2)
            ↓
   런던·시드니·프랑크푸르트·오클랜드 (Phase 3)
            ↓
   재외 한인 700만 디지털 인프라 (Phase 4)
```

**커뮤니티 특징:**
- 1세대 + 2세대 + 3세대 모두 사용
- 한국어 + 영어 병기
- 디아스포라 자긍심 공유
- BTS·K-pop·K-Food 시대의 자긍심 활용

---

### 기둥 3: 교회·기독교 이미지를 밝게 (Brightening Church Image)

**핵심 원칙: 강요하지 않되, 자연스럽게 빛난다**

> *"내가 나그네 되었을 때 너희가 영접하였다"* (마태복음 25:35)

**HebronGuide가 교회 이미지를 밝게 하는 방식:**

| ❌ 하지 말 것 | ✅ 해야 할 것 |
|---|---|
| 직접적 전도 메시지 | 환대·신뢰·섬김으로 보여주기 |
| 성경구절 도배 | 김구·안중근 등 보편적 가치 인용 |
| "교회 오세요" 강요 | 교회 정보를 자연스럽게 노출 |
| 종교적 색채 강조 | 사역 정신을 콘텐츠 품질에 녹임 |

**구체적 예시:**

1. **검증 시스템** = 신뢰 = 그리스도인의 성품
2. **무료 자원 제공** = 섬김 = 사역의 핵심
3. **이민자 환대** = 마태복음 25장의 디지털 구현
4. **교회 섹션 항상 존재** = 강요하지 않되 준비된 자에게 열려있음
5. **다민족 다리 역할** = "이웃을 사랑하라" 실천

**기독교 이미지를 밝게 하는 차별화 요소:**
- 한국 교차로보다 더 정확한 정보 → "교회가 만들면 다르다"
- Yelp보다 더 따뜻한 톤 → "환대의 문화"
- 한인 그룹채팅보다 더 신뢰 → "검증의 약속"

## 🌟 김구 선생 「나의 소원」과의 연결

> **"내가 원하는 우리나라는 세계에서 가장 아름다운 나라가 되기를 원한다.  
> 가장 부강한 나라가 되기를 원하는 것은 아니다.  
> 오직 한없이 가지고 싶은 것은 높은 문화의 힘이다.  
> 문화의 힘은 우리 자신을 행복하게 하고, 나아가 남에게 행복을 주기 때문이다."**

**HebronGuide의 사명 = 디지털 시대의 「나의 소원」 실현**

```
재외 한인이 미국·캐나다에서                  → "우리 자신을 행복하게"
Korean American으로 자긍심 가지고
                ↓
다민족·주류 사회에 한국식 환대를 보여주며   → "남에게 행복을 주기"
                ↓
그리스도의 빛이 자연스럽게 흘러나오는 도구    → "문화의 힘"
```

## 🎬 BTS·아리랑 모델의 깊은 적용

**BTS가 한 일 = 문화로 세상을 감동시킴**  
**HebronGuide가 할 일 = 문화로 디아스포라를 결집시키고, 그 가운데 그리스도의 빛이 흘러나오게 함**

| BTS 요소 | HebronGuide 적용 | 깊은 의미 |
|---|---|---|
| 아리랑 → 세계 | 김구 인용·한인 캘린더 → 디아스포라 | 정체성 자긍심 |
| K-pop 댄스 | 도시별 콘텐츠·스토리 | 문화 상품 |
| ARMY 자발 확산 | 파트너 교회·한인회 추천 | 커뮤니티 힘 |
| 한국어 자랑 | 한·영 병기 콘텐츠 | 언어·문화 보존 |
| 진정성 | 검증·환대·섬김 | 그리스도인의 성품 |

## 📐 의사결정 프레임 — 모든 작업 전 확인

새 기능·콘텐츠 추가 전 다음 3가지 질문을 통과해야 함:

```
질문 1: 이것이 자랑스러운 문화 상품이 되는가?
        (단순 정보 나열이면 ❌)

질문 2: 이것이 글로벌 한인 커뮤니티에 기여하는가?
        (시애틀만의 것이면 ❌, 다른 도시 확장 가능해야 ✅)

질문 3: 이것이 교회·그리스도의 빛을 밝게 하는가?
        (강요는 ❌, 자연스러운 환대·신뢰는 ✅)
```

3가지 모두 YES → 진행  
하나라도 NO → 재검토

## ⚖️ 「Hebron 정신」 3중 의미 (CLAUDE.md 기존 섹션과 통합)

```
1층 (일반인이 보는 것):
   실용적 도시 정착 가이드
   = "기둥 1: 문화 상품"

2층 (한인 커뮤니티가 느끼는 것):
   같은 뿌리를 가진 사람들의 연결
   = "기둥 2: 글로벌 커뮤니티"

3층 (히든 미션):
   사람과 하나님의 연결
   = "기둥 3: 교회 이미지를 밝게"
```

**3층은 강요되지 않는다. 그러나 1층과 2층이 깊어질수록 3층은 자연스럽게 빛난다.**
