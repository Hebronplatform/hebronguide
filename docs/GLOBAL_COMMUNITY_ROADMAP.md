# 글로벌 한인 크리스천 커뮤니티 로드맵
> HebronGuide의 진짜 정체성 — 흩어진 디아스포라를 환대로 모아 그리스도께 잇는 신앙 생태계
> 작성: 2026-06-25 | 기반: 폴 김 목사 비전 + 막10:45 섬김 + 환대 신학

---

## 0. 한 문장 정의
> **HebronGuide는 "한인 정착 앱"이 아니라, 흩어진 730만 디아스포라를 환대로 모아 교회·그리스도께 잇는 글로벌 신앙 공동체다.**

커뮤니티는 **영혼(해자)**이지 **수익계**가 아니다.
→ **커뮤니티로 신뢰를 쌓고, 그 신뢰 위에서 필요를 섬기며 수익이 따라온다.**

---

## 1. 핵심 긴장 — 커뮤니티는 어떻게 돈이 되는가
커뮤니티 자체는 돈이 안 된다. 그래서 대부분 광고·데이터로 빠진다(우리 Hard Rule 위반).
HebronGuide의 답: **커뮤니티(모으기) + 필요를 섬기는 상품·서비스(지속) + 제휴(보조).**

```
모으기(영혼)  →  신뢰  →  필요를 섬김(수익)  →  사명 지속  →  다시 모으기
```

---

## 2. 무엇을 모으는가 — 커뮤니티 4축 구조
단일 게시판이 아니라, **들어오는 문이 여럿**인 다층 구조.

| 축 | 소그룹 예시 | 들어오는 동기 |
|---|---|---|
| **도시별** | 시애틀·LA·뉴욕 사는 사람들 | "나 같은 사람이 사는 곳" |
| **생애별** | 새로 온 가정 · 유학생 · 주재원 · 은퇴 | "지금 내 처지" |
| **세대별** | 1세 · 2세(EM) · 자녀 양육 | 소속·언어 ([[project_second_generation_strategy]]) |
| **관심사별** | 정착·육아·창업·신앙성장·월드컵 | 취미·또래 (Midas에서 배운 입구) |

> 문은 취미·필요로 열고, 안에서 환대·신뢰·교회 연결이 흐르게. (라벨 금지 [[feedback_no_explicit_labeling]])

---

## 3. 사람의 여정 — 게스트에서 환대자로
HebronGuide 신앙 생태계 8단계([[project_hebron_faith_os_soul]])의 입구를 커뮤니티가 담당.

```
낯선 게스트 → 환대 경험 → 커뮤니티 멤버 → 교회 연결 → 제자 → 환대자(다시 섬김)
```
한 사람이 받은 환대를 다시 흘려보내는 **자생적 사이클**이 진짜 목표.

### 역할
- **환대자(Verified Host)** — 검증된 성도·가게. 커뮤니티의 심장
- **목자(Pastor)** — 파트너 교회. 신뢰 앵커
- **성도 / 게스트(비신자 포함)** — 환대의 대상이자 주체

---

## 4. 어떻게 모으는가 (Gather) — 우선순위
1. **교회 네트워크 = 경쟁 불가 유통망** — 파트너 교회 주보·카톡방·새가족 환영 ([[project_church_partnership]])
2. **있는 곳에서 씨 뿌리기** — 페북 그룹·카톡 오픈채팅·유학생 카페·한인회 (Midas 벤치마킹)
3. **한 도시부터 압도** — 시애틀 밀도 완성 → 복제 (Amazon 좁게 깊이)
4. **후크 먼저** — "도착 첫 주 가이드"·"운전면허"·"세금" 즉시 쓸모로 유입
5. **콘텐츠 엔진** — 정착 반복 질문에 숏폼(릴스·쇼츠·유튜브) + SEO
6. **때를 탐** — 2026 북미 월드컵 (`EVENT_PLAYBOOK.md`)

## 5. 어떻게 지속하는가 (Sustain) — 3층 수익
```
1층 영혼:  글로벌 한인 크리스천 커뮤니티        ← 신뢰·해자 (영구 무료)
2층 수익:  디지털 상품 + 검증 서비스 + 교회 SaaS  ← 진짜 마진
3층 보조:  큐레이션 제휴(Amazon)               ← 거들기
```

| 수익원 | 내용 | 비고 |
|---|---|---|
| 디지털 상품 | 정착 키트, 2세 신앙+한국어 (지난 리서치) | 마진 최고 |
| 검증 서비스 중개 | 운전면허·세무·이민·부동산 수수료 | 헤브론 패스 |
| 교회 SaaS | 새가족·정착 도우미 구독 + 가정교회 $50등록/$20연회비 | B2B 안정 |
| 큐레이션 제휴 | "환대 추천템" Amazon Associates | 보조, 고지 필수 |

> **복음은 유료화하지 않는다.** 유료 = 편의·도구·서비스이지, 소속·환대·복음이 아니다.

## 6. 단계별 로드맵

### Phase 1 — 뿌리 (0~6개월)
- [ ] 시애틀 커뮤니티 밀도 완성 (도시·관심사 소그룹 3~5개 활성)
- [ ] 파트너 교회 10곳 등록 → 새가족 유입 경로
- [ ] 첫 디지털 상품 출시 ("미국 첫 1년 정착 키트")
- [ ] 콘텐츠 엔진 시작 (주 2~3 숏폼)
- **지표:** 활성 소그룹 수 · 교회 연결 수 · 첫 매출

### Phase 2 — 확장 (6~18개월)
- [ ] 다도시 복제 (8개 도시)
- [ ] 2세·EM 커뮤니티 + 신앙+한국어 상품
- [ ] 교회용 SaaS 베타
- [ ] 검증 서비스 중개 가동
- **지표:** 도시 수 · MAU · 유료 전환 · 재방문/소속률

### Phase 3 — 열방 (18개월~)
- [ ] 글로벌 디아스포라(유럽·오세아니아·동남아) 연결
- [ ] 다민족 브릿지로 확장 ([[project... MULTI_ETHNIC_VISION]])
- **지표:** 국가 수 · 자생적 환대 사이클 회전율

## 7. KPI — 진짜 측정 기준
다운로드·방문자 수가 아니라 **사이클이 도는가**:
- 환대받은 사람이 **환대자가 되는** 전환율
- 게스트 → **교회 연결** 수
- 파트너 교회 수 · 활성 소그룹 수
- 재방문·소속감 (단발 방문 ❌)
- 수익이 **사명을 지속**시키는가

---

## 8. 가드레일 (절대 지킴)
- 기본(커뮤니티·환대·복음) **영구 무료** — 복음 유료화 금지
- 데이터 판매 금지 · 광고 위장 금지
- 일요일 안식 (상업 활동 ❌)
- 교회 ↔ LLC 자금 분리
- **섬김이 먼저, 수익은 결과** — 분기 점검: "송영 없는 종교 기업으로 변질되는가?"
- 직접 전도 ❌ — 환대가 먼저, 신뢰가 따라옴
