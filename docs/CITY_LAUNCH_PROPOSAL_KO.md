# HebronGuide 도시 런치 제안서
**목사님·교회 지도자 전용**

---

## HebronGuide란?

> 한인 이민자·이주자·방문자가 새 도시에 도착한 첫날부터  
> 필요한 모든 정보를 한 앱에서 찾을 수 있도록 돕는  
> **검증된 한인 생활 정착 가이드 플랫폼**

현재 운영 중: **hebronguide.com/seattle** (시범 도시)

---

## 우리 도시에 HebronGuide가 필요한 이유

```
새 이민자가 도착했을 때 가장 먼저 찾는 것:
  ① 어디서 한국 음식을 먹나?
  ② SSN·운전면허 어디서 받나?
  ③ 한인교회는 어디 있나?
  ④ 긴급할 때 누구에게 전화하나?
  ⑤ 렌트는 얼마나 해야 하나?

지금까지: 카카오톡 그룹채팅에서 물어봄
앞으로:   HebronGuide 앱 하나로 해결
```

---

## 앱이 제공하는 것

| 기능 | 내용 |
|------|------|
| 🛬 정착 가이드 | 단계별 정착 체크리스트 (SSN·면허·은행·학교) |
| 🍽️ 검증 맛집 | 현지 한인 Top 5 검증 맛집 |
| 🗺️ 관광·명소 | 가족·방문객을 위한 현지 명소 |
| ⛪ 교회 연결 | 가정교회 중심 한인교회 정보 |
| 🆘 긴급연락 | 911·총영사관·병원·가스·경찰 |
| 🌐 현장통역 | 한국어↔영어 실시간 음성 통역 |
| 📤 공유 | 카카오·인스타·왓츠앱·틱톡 공유 |

**모두 무료** — 앱 설치 후 즉시 사용

---

## 교회에게 돌아오는 것

```
① 새 이민자가 앱을 열면 → 교회 탭 자연스럽게 발견
② 담임목사님 추천 → 즉각적 신뢰 형성
③ 교회 정보 무료 등재 (주소·예배시간·연락처)
④ 새가족이 앱 통해 교회 발견 → 자연스러운 전도
⑤ 교회가 지역 한인 커뮤니티의 허브로 자리매김
```

## 런치 패키지 — $100 / 년

| 항목 | 내용 | 금액 |
|------|------|------|
| 앱 기본 구축 | 우리 도시 맞춤 HebronGuide | $50 |

| 연간 유지관리 | 데이터 업데이트·기술 지원 | $20 |

> 시애틀 샌프란시스코 지역 교회들이 이미 시작했습니다.

## 교회의 역할 (선택)

**기본 참여 (무료):**
- 교회 정보 등재
- 교인들에게 앱 소개

**파트너 참여 ($100/년):**
- 우리 도시 HebronGuide 런치 후원
- 교회가 "공식 파트너 교회"로 표시
- 지역 한인 새 이민자 커뮤니티 연결 허브

**큐레이터 참여 (선택):**
- 교회 청년·집사 중 SNS 담당자 지정
- 주 1회 지역 한인 정보 업데이트
- 담임목사 서명으로 1년 임기
