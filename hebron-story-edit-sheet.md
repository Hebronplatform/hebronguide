# 환대의 여정 — 실사 영상 편집 시트 (CapCut용)

> 목표: 실사 영상. 클립을 순서대로 놓고 자막(.srt)을 얹으면 완성.
> 색 흐름: 차가운 톤(외로움) → 따뜻한 톤(연결). 장면 6(미소)부터 따뜻해지게.

## 길이 선택 — 자막 파일만 고르면 됨
| 버전 | 자막 파일 | 특징 |
|---|---|---|
| **1분** | `hebron-story-1min.srt` | 빠른 템포·SNS·발표 훅에 적합 |
| **2분** | `hebron-story-2min.srt` | 여유 있는 호흡·감정 충분히 |
| (기본 1:20) | `hebron-story.srt` | 중간 길이 |

→ 1분으로 갈 땐 각 클립을 5~6초, 2분은 8~10초로 두면 자막 타이밍과 맞습니다.

## 준비물
- 무료 스톡 영상: **Pexels.com**, **Pixabay.com** (둘 다 무료·상업 사용 OK·회원가입 불필요)
- 자막 파일: `hebron-story.srt` (CapCut에 드래그하면 타이밍 자동)
- 내레이션: `hebron-story-narration.txt` → ElevenLabs / 클로바더빙으로 음성 생성
- 음악: YouTube 오디오 보관함 / Pixabay — "warm piano hopeful"

## CapCut 작업 순서 (10분)

1. CapCut 열기 → 새 프로젝트
2. 위 검색어로 Pexels/Pixabay에서 클립 11개 다운로드 → 타임라인에 순서대로 배치
3. **자막**: `hebron-story.srt` 파일을 타임라인에 드래그 → 자막·타이밍 자동 입력 (폰트·색만 다듬기)
4. **내레이션**: 생성한 음성 파일을 오디오 트랙에 올림
5. **음악**: 배경음악 추가 → 볼륨 20~30%로 낮춰 내레이션이 들리게
6. **색 보정**(선택): 1~5번 클립은 약간 차갑게(블루+), 6번부터 따뜻하게(웜+)
7. 내보내기 → 1080p MP4
