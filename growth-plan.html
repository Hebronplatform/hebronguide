핵심 · 필수
커뮤니티 매니저
추천: 각 도시 파트너 교회 사모님 또는 청년 사역자
  • 카카오 채널 콘텐츠 주 1회 발행
  • 오픈채팅방 운영·답변
  • 신규 이민자 첫 연락 대응
  • 파트너 교회·업체 관계 유지
처음에는 폴 김 목사님이 겸임 가능. Phase 2부터 도시별 1명 선발 권장. 사례비: 도시당 월 $100–200 (광고 수익 연동)
콘텐츠 · 중요
스토리텔러
추천: 글 잘 쓰는 청년 성도 또는 2세 자녀
  • 환대 받은 사람 인터뷰·증언 카드 제작
  • 도시별 정착 가이드 PDF 작성
  • "처음 이민한 날" 영상 스크립트
  • 카카오·인스타 콘텐츠 디자인
월 8시간 파트타임으로 시작 가능. 신학교 인턴십 연계 추천. 2세 청년이 하면 한·영 동시 제작 가능.
네트워크 · 전략
파트너십 디렉터
추천: 폴 김 목사님 (직접) 또는 신뢰하는 장로님
  • 파트너 교회 목사님 Zoom 월례 미팅 주관
  • 신규 도시 교회 파트너십 개척
  • 한국 본토 교회 연결 (Phase 3)
  • IHM·KSBC 등 교단 네트워크 활용
가장 중요한 역할. 신뢰 관계가 곧 성장 엔진. 목사님의 기존 네트워크가 최대 자산.
기술 · 운영
플랫폼 관리자
추천: Claude Code + 폴 김 목사님 (현재 구조 유지)
  • 사이트 업데이트·버그 수정
  • 새 파트너 교회·업소 등재
  • Supabase 데이터 관리
  • 광고 시스템 구축 (Phase 3)
현재 Claude Code가 이 역할을 담당. 장기적으로 개발자 1명 파트타임 채용 권장 (Phase 2 후반).
도시별 · 현장
도시 앰배서더
추천: 각 도시 파트너 교회 청년 1명
  • 공항 영접 자원봉사 연결
  • 신규 이민자 첫 주 정착 동행
  • 현지 한인 업소·병원 직접 확인
  • 도시별 오픈채팅방 운영
Phase 2부터 시애틀·달라스·토론토 3개 도시 우선. 자원봉사로 시작, Phase 3에서 사례비 연동 가능.
재무 · 투명성
운영 재무 담당
