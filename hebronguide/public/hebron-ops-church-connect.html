🚨 405 오류 원인 분석
원인: www.ijiguchon.org 서버가 외부 링크 클릭 요청을 거부함
→ HTTP 405 "Method Not Allowed" — 서버측 설정 문제

우리 코드 문제 아님. 교회 홈페이지 서버가 www 리다이렉트를
잘못 처리하거나, CDN이 특정 요청 방식을 차단하는 경우.
✅ 이미 적용된 수정 (2026-05-13)
https://www.ijiguchon.org → https://ijiguchon.org
www 제거 후 빌드 · 배포 완료.

추가로 확인할 것:
1. ijiguchon.org 도 안 되면 → 홈페이지 버튼 임시 제거
2. 교회에 정확한 홈페이지 URL 확인 요청
3. 홈페이지 대신 카카오채널 URL로 대체 가능
