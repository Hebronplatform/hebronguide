📊 Google Sheets 자동 수집 설정

커뮤니티가 올린 정보를 구글 시트에 자동 저장 → 언제든 Excel 다운로드

한 번만 설정하면 이후 누가 올릴 때마다 자동으로 시트에 쌓입니다.
설정 시간: 약 10분 · 비용: 무료
1

Google 시트 새로 만들기

sheets.new 접속 → 새 시트 생성
시트 이름을 HebronGuide 커뮤니티로 저장합니다.

1행에 아래 열 제목을 입력하세요:

A날짜
B카테고리
C도시
D이름/상호명
E연락처
F소개
G웹사이트
2

Apps Script 열기

구글 시트 상단 메뉴 → 확장 프로그램 → Apps Script
새 창에서 기존 코드를 모두 지우고 아래 코드를 붙여넣기 합니다:

function doPost(e) {
  try {
    var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    var data  = JSON.parse(e.postData.contents);

    sheet.appendRow([
      data.date     || new Date().toISOString(),
      data.category || "",
      data.city     || "",
      data.name     || "",
      data.contact  || "",
      data.desc     || "",
      data.website  || "",
    ]);

    return ContentService
      .createTextOutput(JSON.stringify({ result: "ok" }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ error: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
3

웹 앱으로 배포

Apps Script 화면 → 오른쪽 상단 배포 → 새 배포

  • 유형: 웹 앱
  • 다음 사용자로 실행: 나
  • 액세스 권한: 모든 사용자 (익명 포함)

배포 버튼 클릭 → 권한 허용 → 웹 앱 URL 복사

URL 형식: https://script.google.com/macros/s/XXXXXXXXXX/exec

HebronGuide 코드에 URL 붙여넣기

복사한 URL을 아래에 붙여넣으면 정확히 어디를 수정해야 하는지 알 수 있습니다:

// 찾을 줄:
const SHEETS_URL = "https://script.google.com/macros/s/REPLACE_WITH_YOUR_SCRIPT_ID/exec";

// 교체할 줄:
const SHEETS_URL = "";

Excel로 다운로드

구글 시트 → 파일 → 다운로드 → Microsoft Excel (.xlsx)
언제든 최신 데이터를 Excel로 받을 수 있습니다.
